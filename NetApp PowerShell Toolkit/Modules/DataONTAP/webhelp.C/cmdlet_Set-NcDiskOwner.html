Cmdlet: Set-NcDiskOwner


NAME
    Set-NcDiskOwner
    
SYNOPSIS
    Changes ownership on a disk.
    
SYNTAX
    Set-NcDiskOwner [-Name] <String> [[-Owner] <String>] [-Pool <UInt32>] 
    [-Checksum <String>] [-Force] [-IsRootPartition] [-IsDataPartition] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Set-NcDiskOwner [-Name] <String> [[-CopyOwnerFrom] <String>] [-Pool 
    <UInt32>] [-Checksum <String>] [-Force] [-IsRootPartition] 
    [-IsDataPartition] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Set-NcDiskOwner [-Name] <String> -OwnerId <UInt32> [-Pool <UInt32>] 
    [-Checksum <String>] [-Force] [-IsRootPartition] [-IsDataPartition] 
    [-IsData1Partition] [-IsData2Partition] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcDiskOwner [[-Owner] <String>] [-Pool <UInt32>] [-Checksum <String>] 
    [-Force] -DiskCount <Int32> [-DiskType <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcDiskOwner [[-Owner] <String>] -All [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcDiskOwner [[-Owner] <String>] -Auto [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcDiskOwner [[-CopyOwnerFrom] <String>] -DiskCount <Int32> [-DiskType 
    <String>] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Set-NcDiskOwner -OwnerId <UInt32> [-Pool <UInt32>] [-Checksum <String>] 
    [-Force] -DiskCount <Int32> [-DiskType <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Changes ownership on a disk.  The normal usage is when there is no current 
    owner or to assign a disk to a pool.
    

PARAMETERS
    -Name <String>
        Name of disk to assign.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Owner <String>
        Assign disk to specific owner (node).  Either Owner or OwnerId must be 
        specified.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Pool <UInt32>
        Assign disk to specific pool, e.g. '0' or '1'.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Checksum <String>
        Assign checksum type to disk.  Option may only be specified on a RAID 
        array LUNs.  Possible values: 'block' or 'zoned'.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force
        Specify if assigning disks which are already owned a filer.  However, 
        if that filer is up and has put a reservation on the disk, even the 
        Force option won't work.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsRootPartition
        Specify to set owner of only the root partition of this disk
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsDataPartition
        Specify to set owner of only the data partition of this disk
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -CopyOwnerFrom <String>
        The disk from which ownership information is to be copied for this disk.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -OwnerId <UInt32>
        Assign disk to specific owner ID (NVRAM ID).  Either Owner or OwnerId 
        must be specified.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsData1Partition
        Specify to execute AssignOwner only on data1 partition of disk.
        
        This parameter is available with Data ONTAP 8.4 and up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsData2Partition
        Specify to execute AssignOwner only on data2 partition of disk.
        
        This parameter is available with Data ONTAP 8.4 and up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DiskCount <Int32>
        Assign specified count of disks.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DiskType <String>
        Assign specified type of disk or set of disks.
        
        Type of disk: ATA, BSAS, EATA, FCAL, FSAS, LUN, MSATA, SAS, SATA, SCSI, 
        SSD, BSSD, XATA, XSAS, or unknown.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -All
        Assign all unowned visible disks to the specified node.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Auto
        Auto-assign unowned disks which are on loops where only one filer owns 
        the disks and the pool information is the same.  No other input 
        parameters are allowed when specifying this option.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see

INPUTS

NOTES
    
    
        Category: disk
        API: disk-sanown-assign
        Family: cluster
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NcDisk 0a.27 | Set-NcDiskOwner -Owner fas3070cluster01-01
    
    
    Assign disk '0a.27' to node 'fas3070cluster01-01'.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Set-NcDiskOwner 0a.27 -OwnerId 118054408
    
    
    Set owner of disk '0a.27' to the node with NVRAM ID '118054408'.
    
    
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Get-NcNode fas3070cluster01-01 | Get-NcDiskOwner -OwnershipType 
    unowned | Set-NcDiskOwner -Owner fas3070cluster01-01
    
    
    Assume ownership of all unowned disks attached to node 
    'fas3070cluster01-01'.
