Cmdlet: Remove-NcCredential


NAME
    Remove-NcCredential
    
SYNOPSIS
    Remove saved login credentials for a Data ONTAP controller.
    
SYNTAX
    Remove-NcCredential [-SystemScope] [<CommonParameters>]
    
    Remove-NcCredential -Controller <NcController[]> [-SystemScope] 
    [<CommonParameters>]
    
    Remove-NcCredential [-Name] <String> [-SystemScope] [<CommonParameters>]
    
    Remove-NcCredential [-Credential] <NcCredential> [<CommonParameters>]
    
    
DESCRIPTION
    Remove saved login credentials for a Data ONTAP controller.
    

PARAMETERS
    -SystemScope
        Remove credentials for the specified controller that were saved using 
        system scope.  If not specified, remove credentials that were saved 
        with user scope.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        One or more NcController objects, as returned by Connect-NcController.  
        If neither this parameter nor -Name is specified,
         this cmdlet will use the value in $global:CurrentNcController (if set).
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Name <String>
        The name or address of a storage controller.  If a host name is 
        specified, it must be resolvable to an address.  If neither this 
        parameter nor -Controller is specified, this cmdlet will use the 
        value(s) in $global:CurrentNcController (if set).
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Credential <NcCredential>
        An NcCredential object, as returned by Get-NcCredential.  If specified, 
        that record is removed from the credentials cache (including the 
        correct value of system scope).
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
