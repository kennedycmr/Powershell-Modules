Cmdlet: Clear-NcDiskOwner


NAME
    Clear-NcDiskOwner
    
SYNOPSIS
    Removes ownership information on a disk.
    
SYNTAX
    Clear-NcDiskOwner -Owner <String> [-All] [-Force] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Clear-NcDiskOwner [-Name] <String> -Owner <String> [-OwnerId <Int64>] 
    [-Force] [-IsRootPartition] [-IsDataPartition] [-IsData1Partition] 
    [-IsData2Partition] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Clear-NcDiskOwner [-OwnerId <Int64>] [-All] [-Force] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Removes ownership information on a disk.
    

PARAMETERS
    -Owner <String>
        Remove ownership of all disks owned by Owner (i.e. a node).  Either 
        Owner or OwnerId is mandatory.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -All
        If specified, remove ownership information from all disks.  This option 
        is supported in maintenance mode only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force
        If specified, override the RAID checks, such as whether the disk is 
        part of an aggregate.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the disk to unassign.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -OwnerId <Int64>
        Remove ownership of all disks owned by this OwnerId (NVRAM ID).  This 
        option is supported in maintenance mode only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -IsRootPartition
        Specify to set owner of only the root partition of this disk
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsDataPartition
        Specify to set owner of only the data partition of this disk
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsData1Partition
        Specify to execute AssignOwner only on data1 partition of disk.
        
        This parameter is available with Data ONTAP 8.4 and up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IsData2Partition
        Specify to execute AssignOwner only on data2 partition of disk.
        
        This parameter is available with Data ONTAP 8.4 and up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
