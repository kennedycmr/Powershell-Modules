Cmdlet: Set-NcStorageArrayPort


NAME
    Set-NcStorageArrayPort
    
SYNOPSIS
    Update an array port with new or changed information.
    
SYNTAX
    Set-NcStorageArrayPort [-ArrayName] <String> [-Wwnn] <String> [-Wwpn] 
    <String> [-MaxQueueDepth <Int64>] [-UtilizationPolicy <String>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Set-NcStorageArrayPort [-ArrayName] <String> [-MaxQueueDepth <Int64>] 
    [-UtilizationPolicy <String>] [-TargetIqn] <String> [-Tpgt] <String> 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Update an array port with new or changed information.
