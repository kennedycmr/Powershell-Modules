Cmdlet: New-NcAntivirusLogEntry


NAME
    New-NcAntivirusLogEntry
    
SYNOPSIS
    Create a new antivirus log entry.
    
SYNTAX
    New-NcAntivirusLogEntry [-VendorId] <Int64> [-VendorString] <String> 
    [-TimeStamp <DateTime>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Create a new antivirus log entry.
    

PARAMETERS
    -VendorId <Int64>
        Vendor ID.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -VendorString <String>
        Vendor string.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -TimeStamp <DateTime>
        When the event is logged.  If not provided, the current DateTime is 
        used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value
