Cmdlet category: Cluster

Add-NcCluster
Start the process of joining this node to a cluster.
Add-NcClusterNode
This cmdlet can be used to add new nodes to cluster.
Get-NcCluster
Returns the cluster identity information.
Get-NcClusterContact
Returns the contact information.
Get-NcClusterCreateProgress
Once the cluster create or cluster join operation has been started on this node, this cmdlet can be used to get the current status of the operation.
Get-NcClusterLogForward
Get information about log forwarding.
Get-NcClusterLogForwardStatistics
Get information about log forwarding statistics
Get-NcClusterNode
Get information about nodes in a cluster.
Get-NcClusterZoneinfo
Get zoneinfo information
Import-NcClusterZoneinfoFromUri
This commands loads a new set of timezone zoneinfo data to replace the version installed in the cluster.
New-NcCluster
Start the process to create a cluster using this node.
New-NcClusterLogForward
Create a log forwarding configuration.
Remove-NcCluster
Unjoins the cluster.
Remove-NcClusterLogForward
Delete a log forwarding configuration
Set-NcCluster
Modifies the cluster identity information.
Set-NcClusterContact
Modifies the contact information
Set-NcClusterLogForward
Modify the log forwarding configuration
Set-NcClusterNode
Modify state of a node in a cluster.
