Cmdlet: Remove-NcUser


NAME
    Remove-NcUser
    
SYNOPSIS
    Delete an existing user account object.
    
SYNTAX
    Remove-NcUser [-UserName] <String> -Vserver <String[]> -Application 
    <String[]> -AuthMethod <String> [-RemoteSwitchIpaddress <String>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Delete an existing user account object.
    

PARAMETERS
    -UserName <String>
        Name of the user to be removed.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String[]>
        Vserver that owns the user account.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Application <String[]>
        Name of the application.  Possible values: 'console', 'http', 'ontapi', 
        'snmp', 'sp', 'ssh'.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -AuthMethod <String>
        Authentication method for the application.  Possible values: 
        'community', 'password', 'publickey', 'domain', 'nsswitch' and 'usm'.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -RemoteSwitchIpaddress <String>
        This optionally specifies the IP Address of the remote switch.
        
        This parameter is available with DataONTAP 9.3 and up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: security
        API: security-login-delete
        Family: cluster
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NcUser vmware_admin | Remove-NcUser
    
    
    Remove all instances of user 'vmware_admin', regardless of vserver, 
    application type, or authentication method.
