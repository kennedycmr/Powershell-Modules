Cmdlet: Start-NcDiskReplace


NAME
    Start-NcDiskReplace
    
SYNOPSIS
    Initiate replacing a file system disk with an appropriate spare disk.
    
SYNTAX
    Start-NcDiskReplace [-Disk] <String> [-ReplacementDisk] <String> 
    [-AllowSameCarrier [<Boolean>]] [-Force] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Initiate replacing a file system disk with an appropriate spare disk.
    

PARAMETERS
    -Disk <String>
        Name of the file system disk to replace.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ReplacementDisk <String>
        Name of the spare disk to use as a replacement.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -AllowSameCarrier [<Boolean>]
        Using two disks from one carrier that houses multiple disks in one RAID 
        group is not desirable.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        Specify to allow replacement-disk to come from the opposite spare pool.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

    --------------  Example 1 --------------
    
    C:\PS>PS C:\PSTK\PSTK_Disk\depot\prod\diapason\main\NcCmdlet\bin\Debug> 
    Start-NcDiskReplace -Disk VMw-1.4 -ReplacementDisk VMw
    -1.8 -AllowSameCarrier $false
    
    
    Starts replacement of disk VMw-1.4 with spare replacement disk VMw-1.8.
