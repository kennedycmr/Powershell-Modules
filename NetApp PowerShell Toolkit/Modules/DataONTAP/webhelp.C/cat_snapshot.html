Cmdlet category: Snapshot

Add-NcSnapshotPolicySchedule
Add a new schedule to a snapshot policy.
Get-NcSnapshot
Get a list of snapshot objects.
Get-NcSnapshotAutodelete
Returns the current snapshot autodelete settings.
Get-NcSnapshotDelta
Returns the amount of space consumed between two snapshots or a snapshot and active filesystem.
Get-NcSnapshotPolicy
Get information about snapshot policies.
Get-NcSnapshotReclaimable
Returns the amount of space that would be freed when a set of snapshots are deleted from a specified volume.
Get-NcSnapshotReserve
Gets the percentage of disk space that is reserved for snapshots.
Get-NcSnapshotRestoreStatus
Get information about snapshot file restores on a given vserver.
New-NcSnapshot
Create a new snapshot on a specified volume.
New-NcSnapshotMulti
Create a snapshot with the specific name on each of the specified volumes.
New-NcSnapshotPolicy
Create a new snapshot scheduling policy.
Remove-NcSnapshot
Delete a snapshot on a specified volume.
Remove-NcSnapshotMulti
Delete the snapshot from the given flexible volumes.
Remove-NcSnapshotPolicy
Delete a snapshot scheduling policy.
Remove-NcSnapshotPolicySchedule
Remove a schedule from a snapshot policy.
Rename-NcSnapshot
Rename a specified snapshot to a new name on a specified volume.
Rename-NcSnapshotMulti
Rename Snapshot copies with a specific name to a new name across multiple volumes.
Restore-NcSnapshotFile
Reverts a single file to a revision from a specified snapshot.
Restore-NcSnapshotVolume
Reverts a volume to a specified snapshot.
Set-NcSnapshot
Modify the attributes of a snapshot.
Set-NcSnapshotAutodelete
Modifies the current snapshot autodelete settings.
Set-NcSnapshotPolicy
Modify a snapshot scheduling policy.
Set-NcSnapshotPolicySchedule
Modify a snapshot policy.
Set-NcSnapshotReserve
Sets the size of the indicated volume's snapshot reserve to the specified percentage.
Set-NcSnapshotSnaplockExpTime
Modifies snaplock expiry time of a snapshot copy.
Test-NcSnapshotMulti
Update-NcSnapshot
Modify the attributes of snapshot or a group of snapshot objects.
