Cmdlet: Get-NcEmsEventNotificationHistory


NAME
    Get-NcEmsEventNotificationHistory
    
SYNOPSIS
    Return EMS messages sent to a destination.
    
SYNTAX
    Get-NcEmsEventNotificationHistory [-Destination] <String> [[-Node] 
    <String[]>] [[-SeqNum] <Int64>] [[-Severity] <String[]>] [-Attributes 
    <EventNotificationHistoryInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcEmsEventNotificationHistory [-Destination] <String> -Template 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcEmsEventNotificationHistory [-Destination] <String> -Query 
    <EventNotificationHistoryInfo> [-Attributes <EventNotificationHistoryInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Return EMS messages sent to a destination.
