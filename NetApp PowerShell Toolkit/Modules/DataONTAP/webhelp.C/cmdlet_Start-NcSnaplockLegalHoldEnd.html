Cmdlet: Start-NcSnaplockLegalHoldEnd


NAME
    Start-NcSnaplockLegalHoldEnd
    
SYNOPSIS
    This API starts an operation to release legal-hold on files in  user 
    specified path. The progress of the operation can be  monitored using 
    snaplock-legal-hold-get-iter api.
    
SYNTAX
    Start-NcSnaplockLegalHoldEnd [-LitigationName] <String> [-Path] <String> 
    [-Volume] <String> [-VserverContext <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    This API starts an operation to release legal-hold on files in  user 
    specified path. The progress of the operation can be  monitored using 
    snaplock-legal-hold-get-iter api.
