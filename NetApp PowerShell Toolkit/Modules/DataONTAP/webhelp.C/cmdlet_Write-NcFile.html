Cmdlet: Write-NcFile


NAME
    Write-NcFile
    
SYNOPSIS
    Write data into a named file.
    
SYNTAX
    Write-NcFile [-Path] <String> -Data <String[]> [-NoClobber] [-Append] 
    [-AppendLine] [-NoTruncate] [-Separator <String>] [-Stream <String>] 
    [-VserverContext <String>] [-Controller <NcController[]>] [-Offset <Int64>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Write-NcFile [-Path] <String> -RawData <Byte[]> [-NoClobber] [-Append] 
    [-AppendLine] [-NoTruncate] [-Stream <String>] [-VserverContext <String>] 
    [-Controller <NcController[]>] [-Offset <Int64>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Write data into a named file.   If the file does not previously exist, it 
    will be created; the owner of the file will be root and it will not be 
    readable or writable by non-root users.  This cmdlet is limited to 
    transfers of 1 MB and should only be used on normal files. The results for 
    other file types such as LUNs is undefined.
    

PARAMETERS

        Pathname of the file to write.  For example: "/vol/<volume>/<file>".

        Accept wildcard characters?  false
        
    -Stream <String>
        A stream is an on disk structure which contains a sequence of bytes, 
        analogous to a file.  A stream is always associated with exactly one 
        file.  Characters that are valid for a file name are also valid for the 
        stream-name.  e.g., 'meta', 'resource1'.
        
        Use this parameter if the data should be written to the named stream 
        associated with the file.
        
        Default is empty stream name, in which case the data is written to the 
        file.
        
        This parameter is available in Data ONTAP 8.2 or later.

        Position?                    named
        Default value                
        Accept pipeline input?       false

    -VserverContext <String>

        direct all subsequent commands to that vserver, or supply the

        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
