Cmdlet: Import-NcPerfPreset


NAME
    Import-NcPerfPreset
    
SYNOPSIS
    Import Performance Preset configuration from source URI
    
SYNTAX
    Import-NcPerfPreset [-SourceUri] <String> [-UserName <String>] [-Password 
    <String>] [-Name] <String> [-Comment <String>] [-Privilege <String>] 
    [-Archived] [-ExpirationLengthMinutes <Int32>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Import-NcPerfPreset [-SourceUri] <String> [-Credential <PSCredential>] 
    [-Name] <String> [-Comment <String>] [-Privilege <String>] [-Archived] 
    [-ExpirationLengthMinutes <Int32>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Import Performance Preset configuration from source URI
