Cmdlet category: Fcp

Add-NcFcpService
Create an FCP Service in a Vserver. Each Vserver requires an online FCP Service in order to serve data via the Fibre Channel protocol.
Clear-NcFcpPortNameAlias
Remove an alias for a World Wide Port Name of an initiator.
Disable-NcFcp
Disable FCP service. When FCP service is stopped, the adapters are brought offline. Service will be unavaliable once the call returns with success.
Disable-NcFcpAdapter
Bring a Fibre Channel target adapter offline.
Enable-NcFcp
Enable FCP service. When FCP service is started, the adapters are brought online.
Enable-NcFcpAdapter
Bring a Fibre Channel target adapter online.
Get-NcFcpAdapter
Get a list of physical FC adapters.
Get-NcFcpAdapterStatistics
Get a list of fcp adapter statistics.
Get-NcFcpInitiator
Get all initiators connected to FCP target LIFs.
Get-NcFcpInterface
Get a list of FCP Target Logical Interfaces (LIF).
Get-NcFcpNameserver
Iterates over the list of entries in the fabric name server database.
Get-NcFcpNodeName
Get the current FCP World Wide Node Name (WWNN).
Get-NcFcpPingIgroup
Get a list of FCP ping igroups
Get-NcFcpPingInitiator
Get a list of FCP ping by FCP initiator objects.
Get-NcFcpPortName
Get a list of valid Fibre Channel target port names on a Vserver's FCP LIFs.
Get-NcFcpPortNameAlias
Get a list of assigned World Wide Port Name (WWPN) aliases.
Get-NcFcpService
Get a list of FCP Services.
Get-NcFcpTopology
Iterates over the list of FCP topology interconnect elements.
Get-NcFcpTopologyPort
Iterates over the list of entries in the fabric name server database.
Get-NcFcpZone
Iterates over the list of entries in the fabric name server database.
Remove-NcFcpService
Destroy the FCP Service in a Vserver.
Set-NcFcpAdapterSpeed
Set the speed on the Fibre Channel target adapter.
Set-NcFcpNodeName
Set the current FCP World Wide Node Name (WWNN).
Set-NcFcpPortName
Set a valid but unused port name on a Fibre Channel target interface.
Set-NcFcpPortNameAlias
Set an alias for a World Wide Port Name of an initiator that might login to the target.
Test-NcFcp
Get status of the FCP service, whether or not it is running.
