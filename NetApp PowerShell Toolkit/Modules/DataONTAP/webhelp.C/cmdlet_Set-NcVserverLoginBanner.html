Cmdlet: Set-NcVserverLoginBanner


NAME
    Set-NcVserverLoginBanner
    
SYNOPSIS
    Modify the login banner message.
    
SYNTAX
    Set-NcVserverLoginBanner [[-Message] <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcVserverLoginBanner [[-Message] <String>] [-Vservers] <String[]> 
    [-ContinueOnFailure] [-MaxFailureCount <Int32>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcVserverLoginBanner [[-Message] <String>] [-Query] 
    <VserverLoginBannerInfo> [-ContinueOnFailure] [-MaxFailureCount <Int32>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Set-NcVserverLoginBanner [[-Message] <String>] [-Controller 
    <NcController[]>] -Template [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify the login banner message. This message is displayed during SSH and 
    console device login, just before the password prompt is, or would be, 
    displayed. There is a cluster-wide default login banner that is used for 
    every incoming connection regardless of LIF type. Each data Vserver can 
    over-ride this default. To restore the default for a data Vserver, set the 
    message to '-'.

NOTES
    
    
        Category: Vserver

        Family: cluster, vserver
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NcVserverLoginBanner "Are you sure you want to log in to vserver 
    'pstk'?"
    
    
    Set a login banner for the vserver you are currently connected to.
    
    
    
    Vserver : pstk
    Message : Are you sure you want to log in to vserver 'pstk'?
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Set-NcVserverLoginBanner "welcome to the vserver" pstk_san,pstk
    
    
    Set the same login banner for multiple vservers
    
    
    
    Vserver : pstk
    Message : welcome to the vserver
    
    
    Vserver : pstk_san
    Message : welcome to the vserver
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Set-NcVserverLoginBanner "This is the login banner" -Query $query
    
    
    Using a query. The query object is created as
    PS> $query = Set-NcVserverLoginBanner -Template
    PS> $query.Vserver="pstk_san|pstk"
    
    
    
    Vserver : pstk
    Message : This is the login banner
    
    
    Vserver : pstk_san
    Message : This is the login banner
