Cmdlet category: Lun

Add-NcLunMap
Maps a LUN to all the initiators in the specified initiator group.
Add-NcLunMapReportingNodes
Add nodes to the list reporting the LUN map.
Clear-NcLunPersistentReservation
Clear the SCSI-2 reservation or SCSI-3 persistent reservation information for a given LUN.
Confirm-NcLunHasScsiReservation
Queries for all types of SCSI reservations covering both iSCSI and FCP.
Confirm-NcLunInitiatorLoggedIn
Determine if an initiator is logged in via FCP or iSCSI.
Get-NcLun
Get LUNs in a cluster or vserver.
Get-NcLunAlignment
Get a list of LUN alignment objects.
Get-NcLunAttribute
Get named attributes for a given LUN.
Get-NcLunCopy
Get details of lun-copy jobs on the controller.
Get-NcLunGeometry
Get SCSI disk geometry for a given LUN.
Get-NcLunImport
Get details of lun-relationships on the controller.
Get-NcLunInquiry
Get the SCSI INQUIRY response data for vendor id (vid), product id (pid), and firmware revision (rev) based on the igroup that the LUN in question is mapped to.
Get-NcLunMap
List LUN map details.
Get-NcLunMapByInitiator
List all the LUNs mapped to an initiator.
Get-NcLunMaxSize
Returns the maximum possible size in bytes of a LUN on a given volume or qtree.
Get-NcLunMinSize
Returns the minimum possible size in bytes.
Get-NcLunMove
Get details of lun-move jobs on the controller.
Get-NcLunOsTypes
Displays the supported values and descriptions for lun-os-type.
Get-NcLunPersistentReservation
Get the persistent reservation information for a given LUN.
Get-NcLunSelect
Get the select attribute for the specified LUN.
Get-NcLunSerialNumber
Get the serial number for the specified LUN
Get-NcLunSignature
Get the MBR or GPT signature of a partitioned LUN.
Get-NcLunStatistics
Get block-protocol access statistics for LUNs.
New-NcLun
Create a LUN by one of two methods: by size or from file.
New-NcLunImportRelationship
Create a LUN import relationship with the purpose of importing foreign disk data into the LUN.
New-NcLunSerialNumber
Generates a LUN Serial Number for later use.
Remove-NcLun
Destroy the specified LUN.
Remove-NcLunImportRelationship
Deletes the import relationship of the specified LUN or the specified foreign disk.
Remove-NcLunMap
Unmaps a LUN from an igroup.
Remove-NcLunMapReportingNodes
Remove nodes from the list reporting the LUN map.
Rename-NcLun
Move (rename) a LUN.
Reset-NcLunAlignmentStatistics
Reset LUN Alignment Statistics. Reset will work only when the lun is mapped.
Resume-NcLunCopy
Resume a paused lun copy operation
Resume-NcLunImport
Resumes a paused lun import operation
Resume-NcLunMove
Resumes a paused LUN move operation.
Set-NcLun
Enable or disable block-protocol access to a LUN.
Set-NcLunAttribute
Set a named attribute for a given LUN.
Set-NcLunComment
Set the optional descriptive comment for a LUN.
Set-NcLunCopyMaxThroughput
Modifies the maximum throughput of an ongoing copy operation.
Set-NcLunDeviceId
Set a SCSI peripheral device identifying information value on a LUN.
Set-NcLunImportThrottle
Modify the max throughput limit for the specified import relationship
Set-NcLunMoveMaxThroughput
Modifies the maximum throughput of an ongoing move operation
Set-NcLunQosPolicyGroup
Set the QoS policy group for a LUN.
Set-NcLunSelect
Set the select attribute for the specified LUN.
Set-NcLunSerialNumber
Set the serial number for the specified LUN.
Set-NcLunSignature
Set the MBR or GPT signature of a partitioned LUN.
Set-NcLunSize
Changes the size of the LUN.
Set-NcLunSpaceAllocated
Sets the space allocation settings for the named LUN.
Set-NcLunSpaceReserved
Sets the space reservation settings for the named LUN.
Start-NcLunCopy
Starts copying a group of LUNs from one volume to another.
Start-NcLunImport
Start and import operation for the specified LUN.
Start-NcLunImportVerify
Start the verification for the specified LUN.
Start-NcLunMove
Starts a LUN move operation to move one or more LUNs from one volume to another within a Vserver.
Stop-NcLunCopy
Cancels an ongoing LUN copy operation prior to creation of the new LUN.
Stop-NcLunImport
Abort import operation for the specified LUN.
Stop-NcLunImportVerify
Stop the verification for the specified LUN.
Stop-NcLunMove
Cancels an ongoing LUN move operation prior to creation of the new LUN
Suspend-NcLunCopy
Pauses an ongoing LUN copy operation.
Suspend-NcLunImport
Pauses an ongoing lun import operation
Suspend-NcLunMove
Pauses an ongoing LUN move operation.
