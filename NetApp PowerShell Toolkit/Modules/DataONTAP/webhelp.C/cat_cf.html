Switch to Data ONTAP 7-mode documentation.

Cmdlet category: Cf

Disable-NcClusterHa
Disable the takeover capability of this filer in the cluster.
Enable-NcClusterHa
Enable the takeover capability of this filer in the cluster.
Get-NcClusterHa
Get useful information about the status of the clustering service.
Get-NcClusterHaGiveback
Get the giveback status of all the aggregates in the cluster which are taken over or were given back.
Get-NcClusterHaHwAssist
Get useful information about the status of hw_assist functionality.
Get-NcClusterHaHwAssistStats
Get useful information about statistics of hardware assisted takeover functionality.
Get-NcClusterHaInfo
Get a list of cf objects.
Get-NcClusterHaPartner
Get the host name of the partner. If the name is unknown, it will return partner-unknown.
Get-NcClusterHaTakeover
Get the takeover status of a node.
Invoke-NcClusterHaGiveback
Initiate a giveback of partner resources.
Invoke-NcClusterHaTakeover
Initiate a takeover of the storage partner.
Reset-NcClusterHaHwAssistStats
Get useful information about stats of hw_assist functionality.
Set-NcClusterHaMode
Enable or disable the high-availability (HA) for this node.
Test-NcClusterHaHwAssist
Validate the hardware-assisted takeover configuration functionality.
Update-NcClusterHa
Modify the attributes of cf or a group of cf objects.
