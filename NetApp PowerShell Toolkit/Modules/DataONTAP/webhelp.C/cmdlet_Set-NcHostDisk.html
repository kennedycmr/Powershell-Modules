Cmdlet: Set-NcHostDisk


NAME
    Set-NaHostDisk
    
SYNOPSIS
    Set basic Windows disk options (offline, online, partition style).
    
SYNTAX
    Set-NaHostDisk [-DiskIndex] <UInt32> [-PartitionStyle] <PartitionStyle> 
    [<CommonParameters>]
    
    Set-NaHostDisk [-DiskIndex] <UInt32> [-Online] [-ReadOnly] 
    [<CommonParameters>]
    
    Set-NaHostDisk [-DiskIndex] <UInt32> [-Offline] [<CommonParameters>]
    
    
DESCRIPTION
    Set basic Windows disk options (offline, online, partition style).
    
    This cmdlet must be run with administrative privileges and is only 
    supported on Windows Vista sp1, Server 2008 or later.
    

PARAMETERS
    -DiskIndex <UInt32>
        A windows disk index/number
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -PartitionStyle <PartitionStyle>
        Partition style (Gpt or Mbr)
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Online
        Bring the disk online
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ReadOnly
        Bring the disk online in a read-only state
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Offline
        Bring the disk offline
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.HostUtilities.DiskDiscovery.DiskResource
        
        
    
     
    
NOTES
    
    
        Category: host
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaHostDisk 4 -Offline
    
    
    Set disk 4 offline
