Cmdlet: Get-NcTemplateManagementTemplateParameter


NAME
    Get-NcTemplateManagementTemplateParameter
    
SYNOPSIS
    Get the parameter of a Template.
    
SYNTAX
    Get-NcTemplateManagementTemplateParameter [[-Name] <String[]>] 
    [[-TemplateName] <String[]>] [-Attributes <TemplateParameterInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcTemplateManagementTemplateParameter -Template [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcTemplateManagementTemplateParameter -Query <TemplateParameterInfo> 
    [-Attributes <TemplateParameterInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the parameter of a Template.
    

PARAMETERS
    -Name <String[]>
        Name of the parameter
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -TemplateName <String[]>
        Name of the template
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <TemplateParameterInfo>
        Specify the attributes that should be returned.
        If not present, all attributes for which information is available will 
        be returned.
        
        If present, only the desired attributes for which information is 
        available will be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object. This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used. In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty AggrObjectStore object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <TemplateParameterInfo>
        For advanced queries, provide a TemplateParameterInfo object with 
        various fields set to indicate which records to retrieve.  Other fields 
        should remain set to null.  Use 
        "Get-NcTemplateManagementTemplateParameter -Template" to get the 
        initially empty TemplateParameterInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.TemplateManagement.TemplateParameterInfo
