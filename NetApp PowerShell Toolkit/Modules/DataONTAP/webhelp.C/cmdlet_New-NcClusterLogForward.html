Cmdlet: New-NcClusterLogForward


NAME
    New-NcClusterLogForward
    
SYNOPSIS
    Create a log forwarding configuration.
    
SYNTAX
    New-NcClusterLogForward [-Destination] <String> [[-Port] <UInt16>] 
    [-Facility <String>] [-Force] [-Protocol <String>] [-IsServerVerified 
    [<Boolean>]] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Create a log forwarding configuration.
    

PARAMETERS
    -Destination <String>
        Destination address that the log messages will be forwarded to. Can be 
        a hostname or IP address.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Port <UInt16>
        The destination port used to forward the message.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Facility <String>
        Specify one or more facilities to restrict results
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        Specify to skip connectivity test.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Protocol <String>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -IsServerVerified [<Boolean>]
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
