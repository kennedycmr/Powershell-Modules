Cmdlet: Start-NcClusterImageUpdate


NAME
    Start-NcClusterImageUpdate
    
SYNOPSIS
    Validates and installs prerequisite component updates, and installs Data 
    ONTAP to all  specified nodes within a cluster in a non-disruptive manner.
    
SYNTAX
    Start-NcClusterImageUpdate [-PackageVersion] <String> [[-Nodes] <String[]>] 
    [-StabilizeMinutes <Int64>] [-PauseAfter <String>] 
    [-IgnoreValidationWarning] [-ForceRolling] [-EstimateOnly] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Validates and installs prerequisite component updates, and installs Data 
    ONTAP to all  specified nodes within a cluster in a non-disruptive manner. 
    
    Note: The package must already be downloaded using 
    Start-NcClusterImagePackageDownload cmdlet
    

PARAMETERS
    -PackageVersion <String>
        Specify the package version(s) to update to.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Nodes <String[]>
        Specify the nodes to be updated.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -StabilizeMinutes <Int64>
        Specify the number of minutes needed to stabilize node before update.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -PauseAfter <String>
        One or more pause options can be specified : all or none
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -IgnoreValidationWarning
        Specify to allow the update to continue if warnings are encountered 
        during validation phase.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ForceRolling
        Specify to force a rolling upgrade on the cluster.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -EstimateOnly
        Specify to generate a report of the steps involved in the update 
        process. THIS DOES NOT UPDATE the cluster.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
