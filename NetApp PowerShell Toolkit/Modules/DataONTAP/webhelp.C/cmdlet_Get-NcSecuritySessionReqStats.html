Cmdlet: Get-NcSecuritySessionReqStats


NAME
    Get-NcSecuritySessionReqStats
    
SYNOPSIS
    Get session statistics per-request.
    
SYNTAX
    Get-NcSecuritySessionReqStats [[-NodeName] <String[]>] [[-Interface] 
    <String[]>] [[-Request] <String[]>] [-Attributes <SessionRequestStatsInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcSecuritySessionReqStats -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcSecuritySessionReqStats -Query <SessionRequestStatsInfo> [-Attributes 
    <SessionRequestStatsInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
