Cmdlet: Get-NcClusterNode


NAME
    Get-NcClusterNode
    
SYNOPSIS
    Get information about nodes in a cluster.
    
SYNTAX
    Get-NcClusterNode [[-Name] <String[]>] [-Attributes <ClusterNodeInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcClusterNode -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcClusterNode -Query <ClusterNodeInfo> [-Attributes <ClusterNodeInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Get information about nodes in a cluster.
    

PARAMETERS
    -Name <String[]>
        Name of one or more nodes to get.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <ClusterNodeInfo>
        For improved scalability in large clusters, provide a ClusterNodeInfo 
        object with various fields set to indicate which fields to return on 
        all objects.  Undesired fields should remain set to null.  Use 
        "Get-NcClusterNode -Template" to get the initially empty 
        ClusterNodeInfo object.  If not specified, all data is returned for 
        each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty ClusterNodeInfo object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <ClusterNodeInfo>
        For advanced queries, provide a ClusterNodeInfo object with various 
        fields set to indicate which records to retrieve.  Other fields should 
        remain set to null.  Use "Get-NcClusterNode -Template" to get the 
        initially empty ClusterNodeInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Cluster.ClusterNodeInfo
