Cmdlet: Get-NcAggrAutobalanceAggrState


NAME
    Get-NcAggrAutobalanceAggrState
    
SYNOPSIS
    Get a list of aggr autobalance objects.
    
SYNTAX
    Get-NcAggrAutobalanceAggrState [[-Node] <String[]>] [[-Aggregate] 
    <String[]>] [-Attributes <AggrAutobalanceAggregateState>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcAggrAutobalanceAggrState -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcAggrAutobalanceAggrState -Query <AggrAutobalanceAggregateState> 
    [-Attributes <AggrAutobalanceAggregateState>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of aggr autobalance objects.
    

PARAMETERS
    -Node <String[]>
        The name of one or more nodes containing the aggregates to get.  Data 
        ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Aggregate <String[]>
        The name of one or more aggregates to get.  Data ONTAP wildcards are 
        permitted.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <AggrAutobalanceAggregateState>
        For improved scalability in large clusters, provide a 
        AggrAutobalanceAggregateState object with various fields set to 
        indicate which fields to return on all objects.  Undesired fields 
        should remain set to null.  Use "Get-NcAggrAutobalanceAggrState 
        -Template" to get the initially empty AggrAutobalanceAggregateState 
        object.  If not specified, all data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty AggrAutobalanceAggregateState object for use 
        with advanced queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <AggrAutobalanceAggregateState>
        For advanced queries, provide a AggrAutobalanceAggregateState object 
        with various fields set to indicate which records to retrieve.  Other 
        fields should remain set to null.  Use "Get-NcAggrAutobalanceAggrState 
        -Template" to get the initially empty AggrAutobalanceAggregateState 
        object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
