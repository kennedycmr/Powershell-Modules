Cmdlet: Get-NcNetRoutingGroupRoute


NAME
    Get-NcNetRoutingGroupRoute
    
SYNOPSIS
    Get network routing group routes.
    
SYNTAX
    Get-NcNetRoutingGroupRoute [[-RoutingGroup] <String[]>] [-Destination 
    <String[]>] [-Vserver <String[]>] [-Gateway <String[]>] [-Metric 
    <String[]>] [-Attributes <RoutingGroupRouteInfo>] [-VserverContext 
    <String>] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcNetRoutingGroupRoute -Template [-VserverContext <String>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcNetRoutingGroupRoute -Query <RoutingGroupRouteInfo> [-Attributes 
    <RoutingGroupRouteInfo>] [-VserverContext <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get network routing group routes.
