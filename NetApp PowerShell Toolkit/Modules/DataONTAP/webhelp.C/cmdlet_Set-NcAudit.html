Cmdlet: Set-NcAudit


NAME
    Set-NcAudit
    
SYNOPSIS
    Set the cluster administrative audit settings.
    
SYNTAX
    Set-NcAudit [-CliSet [<Boolean>]] [-CliGet [<Boolean>]] [-HttpSet 
    [<Boolean>]] [-HttpGet [<Boolean>]] [-SnmpSet [<Boolean>]] [-SnmpGet 
    [<Boolean>]] [-OntapiSet [<Boolean>]] [-OntapiGet [<Boolean>]] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcAudit [-Attributes] <Object> [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Set the cluster administrative audit settings.
    

PARAMETERS

        If true, enable auditing of CLI set operations.  If false, disable 
        auditing of CLI set operations.  If not specified, the setting will 
        remain unchanged.

        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

        Accept wildcard characters?  false

NOTES
    
    
        Category: audit

    --------------  Example 1 --------------
    
    C:\PS>Set-NcAudit -CliSet $false -HttpSet $false -HttpGet $true
    
    
    Set the cluster to have the given audit settings.  Any settings not 
    specified remain unchanged.
    
    
    
    CliGet             : False
    CliSet             : False
    HttpGet            : True
    HttpSet            : False
    OntapiGet          : False
    OntapiSet          : True
    SnmpGet            : False
    SnmpSet            : True
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>$t = Get-NcAudit -Template
    $t.CliGet = $true
    $t.OntapiGet = $true
    $t.SnmpGet = $true
    Set-NcAudit -Attributes $t
    
    
    Set the cluster to have the given audit settings using an AuditInfo object. 
     Any settings not specified remain unchanged.
    
    
    
    CliGet             : True
    CliSet             : False
    HttpGet            : True
    HttpSet            : False
    OntapiGet          : True
    OntapiSet          : True
    SnmpGet            : True
    SnmpSet            : True
