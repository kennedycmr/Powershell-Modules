Cmdlet: Invoke-NcMetroclusterHeal


NAME
    Invoke-NcMetroclusterHeal
    
SYNOPSIS
    Heal aggregates and partner roots in preparation for a DR switchback.
    
SYNTAX
    Invoke-NcMetroclusterHeal -Aggregates [-StartOnNodeAndSkipNodeVersionChecks 
    <String>] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Invoke-NcMetroclusterHeal -RootAggregates 
    [-StartOnNodeAndSkipNodeVersionChecks <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Heal aggregates and partner roots in preparation for a DR switchback.
    
    Healing of aggregates and partner roots is done in two phases by the admin. 
    DR partner nodes must be powered off and remote disk shelves must be 
    powered on before calling this cmdlets.
    
    A job will be spawned to operate on the MetroCluster and the job id will be 
    returned. The progress of the job can be tracked using the job cmdlets.
    

PARAMETERS
    -Aggregates
        If specified, invoke the first healing phase.  The first phase heals 
        aggregates by resyncing mirrored plexes and replaying the nvlogs of 
        non-mirrored aggregates.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -StartOnNodeAndSkipNodeVersionChecks <String>
        Specify on which Node job starts and skip Node Version checks
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RootAggregates
        If specified, invoke the second healing phase.  The second phase heals 
        the root aggregates of partner nodes.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
