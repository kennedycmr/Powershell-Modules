Cmdlet: Get-NcTapeMc


NAME
    Get-NcTapeMc
    
SYNOPSIS
    Iterate over the list of Tape/MC objects
    
SYNTAX
    Get-NcTapeMc [[-DeviceId] <String[]>] [-NodeName <String[]>] [-Attributes 
    <TapeMcInfo>] [-Controller <NcController[]>] [-DeviceType <String[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcTapeMc -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcTapeMc -Query <TapeMcInfo> [-Attributes <TapeMcInfo>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Iterate over the list of Tape/MC objects
    

PARAMETERS

        Device Id(s) for which information is to be retrieved.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -NodeName <String[]>

        Required?                    false

        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <TapeMcInfo>
        Specify to restrict which data fields are returned by Data ONTAP
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -DeviceType <String[]>
        Type(s) of devices for which information is to be retrieved
