Cmdlet: Set-NcStorageArray


NAME
    Set-NcStorageArray
    
SYNOPSIS
    Update an array profile with new or changed information.
    
SYNTAX
    Set-NcStorageArray [-ArrayName] <String> [-AllPathFailDelay <String>] 
    [-IsUpgradePending [<Boolean>]] [-LunQueueDepth <Int64>] [-MaxQueueDepth 
    <Int64>] [-Model <String>] [-Options <String>] [-PathFailoverTime <String>] 
    [-Prefix <String>] [-Vendor <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Update an array profile with new or changed information.
    

PARAMETERS
    -ArrayName <String>
        The name of the array profile to update.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -AllPathFailDelay <String>
        The time delay to hold IO during All Path Failure Event. Format 
        <number><units>, eg 2s, 2m and 2h.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -IsUpgradePending [<Boolean>]
        Used to indicate that the specified array will under go an upgrade in 
        the near future.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -LunQueueDepth <Int64>
        The queue depth assigned to array LUNs from this array.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -MaxQueueDepth <Int64>
        The target port queue depth for all target ports on this array.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Model <String>
        The model of the array.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Options <String>
        Array profile specific options.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -PathFailoverTime <String>
        The time delay before switching the I/O path when the path is deleted. 
        Format <number><units>, eg 2s, 2m and 2h
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Prefix <String>
        A short label used to refer to this array.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vendor <String>
        The name of the array's manufacturer.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
