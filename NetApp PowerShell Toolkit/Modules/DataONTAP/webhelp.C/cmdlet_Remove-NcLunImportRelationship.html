Cmdlet: Remove-NcLunImportRelationship


NAME
    Remove-NcLunImportRelationship
    
SYNOPSIS
    Deletes the import relationship of the specified LUN or the specified 
    foreign disk.
    
SYNTAX
    Remove-NcLunImportRelationship [-Controller <NcController[]>] [-Path] 
    <String> [-Vserver] <String> [-Force] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Remove-NcLunImportRelationship [-Controller <NcController[]>] 
    [-ForeignDisk] <String> [-Force] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Deletes the import relationship of the specified LUN or the specified 
    foreign disk.
    

PARAMETERS
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Path <String>
        The path to the lun
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String>
        The Vserver containing the LUN which has an import relationship with 
        the foreign disk.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        Specify to delete the import relationship even if import operation is 
        in progress.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ForeignDisk <String>
        The serial number of the foreign disk
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Lun.LunImportInfo
