Dismount-NcVol

Get-NcVol
Get a list of volumes.
Get-NcVolAutobalance
Displays information about Auto Balance Volume operations for an Infinite Volume.
Get-NcVolAutosize
Given the name of a flexible volume, get the autosize settings.
Get-NcVolClone
Display a list of FlexClone volumes.
Get-NcVolCloneSplit
Display the progress in separating clones from their underlying parent volumes and snapshots.
Get-NcVolCloneSplitEstimate
Display an estimate of additional storage required in the underlying aggregate to perform a volume clone split operation.
Get-NcVolContainer
Return the name of the containing aggregate for the named flexible volume.
Get-NcVolFlexgroupAggrLayout
Get the aggregate layout that will be created when deploying a FlexGroup.
Get-NcVolFlexgroupExpandLimits
Get the minimum and maximum expand limits for a FlexGroup.
Get-NcVolFlexgroupLimits
Get the minimum and maximum create limits for a FlexGroup.
Get-NcVolFlexgroupResizeLimits
Get the minimum and maximum resize limits for a FlexGroup.
Get-NcVolFootprint
Get a list of volumes and their data and metadata footprints in their parent aggregate.
Get-NcVolLanguage
Get the given volume's language mapping.
Get-NcVolLimit
Return calculated volume limits, which are based on the current configuration of the vserver.
Get-NcVolMove
Get status of move operation.
Get-NcVolMoveTargetAggr
Scans aggregates and returns a list of compatible target aggregates for the given volume move operation.
Get-NcVolOption
Get the options that have been set for the specified volume.
Get-NcVolRoot
Return the "namespace root" volume on the currently connected vserver.
Get-NcVolSize
Get the size of a volume.
Get-NcVolSpace
Get a list of volumes and a breakup of their used space.
Get-NcVolStorageService
Get a list of volume storage service objects.
Get-NcVolumeEncryption
Display the encryption status of a specific volume

Invoke-NcVolumeEncryptionRekey
Rekey a Volume

Remove-NcVol
Destroy the specified volume.
Rename-NcVol

Rename-NcVolStorageService
Rename a volume storage service.
Resume-NcVolumeEncryption

Resume-NcVolumeMove

Set-NcVol

Set-NcVolAutosize

Set-NcVolSize
Set the size of a volume.

Start-NcVolAutobalance

Start-NcVolCloneSplit

Start-NcVolCopy

Start-NcVolEncryptionPrepareToDowngrade

Start-NcVolExpand

Start-NcVolFlexgroupDeploy

Start-NcVolMove

Start-NcVolReHost

Start-NcVolTransitionDowngrade
Verifies that there are no volumes actively transitioning from 7-mode to clustered Data ONTAP, and configures the transition feature for downgrade.
