Get-NcShelf
Get information about one or more shelves.
Get-NcShelfBay
Get information about storage shelf bays and ports.
Get-NcShelfEnvironment
Get the environmental information for storage shelves.
Get-NcStorageShelf
Get shelf information about one or more storage shelves.
Get-NcStorageShelfAcp
Return the connectivity of the node.
Get-NcStorageShelfAcpModule
Return the module list
Get-NcStorageShelfDrawer
Get drawer information about a specified storage shelf drawer.
Get-NcStorageShelfDrawerPhy
Get phy information about a specified storage shelf drawer phy.
Get-NcStorageShelfDrawerSlot
Get a map between bay number and drawer/slot number.
Get-NcStorageShelfError
Lists all known errors associated with a storage shelf.
Get-NcStorageShelfFirmwareUpdateInfo
Storage shelf firmware update status records.
Get-NcStorageShelfLocationLed
Get the Location LED status of one or more shelves.
Set-NcStorageShelfStorageAcp
Configure connection.
Start-NcStorageShelfFirmwareUpdate
Start shelf firmware download process to update firmware on disk shelves.
Update-NcStorageShelfAcpFirmware
Start shelf firmware download process to update firmware on disk shelves. This operation is asynchronous, and therefore returns no errors that might occur during the download process. This operation will only update firmware on shelves that do not have the latest firmware revision.
Update-NcStorageShelfLocationLed
Modify the state of the Location LED of one or more shelves/
