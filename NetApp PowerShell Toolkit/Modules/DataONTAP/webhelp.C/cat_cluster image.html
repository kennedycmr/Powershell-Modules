Cmdlet category: Cluster Image

Get-NcClusterImage
Get information about image on each node of the cluster
Get-NcClusterImageDownload
Check the progress of current image download started by the Start-NcClusterImagePackageDownload cmdlet
Get-NcClusterImagePackage
Get record of locally available packages
Get-NcClusterImageUpdateHistory
Get history records of update operations on this cluster
Get-NcClusterImageUpdateLog
Get details from the update log
Get-NcClusterImageUpdateProgress
Get the status of all phases of update that is in progress.
Get-NcClusterImageUpdateProgressOverview
Get information on a node's image update progress
Get-NcClusterImageUpgradePlan
The cluster-image-get-upgrade-plan API generates an Upgrade Plan and Backout Plan for the specified target image. The target package-version must be provided.
Remove-NcClusterImagePackage
Remove a package from the cluster image repository
Resume-NcClusterImageUpdate
Resumes a package update that is currently paused.
Start-NcClusterImagePackageDownload
Downloads the specified package.
Start-NcClusterImageUpdate
Validates and installs prerequisite component updates, and installs Data ONTAP to all specified nodes within a cluster in a non-disruptive manner.
Stop-NcClusterImagePackageDownload
Abort an ongoing package download operation.
Stop-NcClusterImageUpdate
Cancel a package update that is currently in progress.
Suspend-NcClusterImageUpdate
Pauses a package update that is currently in progress.
Test-NcClusterImage
Validates the various logical and physical components of a cluster to ensure that a non-disruptive update can occur.
