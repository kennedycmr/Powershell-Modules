Cmdlet category: Sis

Disable-NcSis
Disable SIS (dedupe) on a volume.
Enable-NcSis
Enable SIS (dedupe) on a volume.
Get-NcSis
Get the SIS (dedupe) status for one or more volumes.
Get-NcSisPolicy
Get the SIS (dedupe) policies defined on one or more vservers.
New-NcSisPolicy
Create a new SIS (dedupe) policy.
Remove-NcSisPolicy
Delete a SIS (dedupe) policy.
Set-NcSis
Set up or modify SIS (dedupe) policy, schedule or options for a volume.
Set-NcSisPolicy
Modify the attributes of a SIS (dedupe) policy.
Start-NcSis
Start a SIS (dedupe) operation on a volume.
Stop-NcSis
Abort any currently active SIS (dedupe) operation on the volume.
