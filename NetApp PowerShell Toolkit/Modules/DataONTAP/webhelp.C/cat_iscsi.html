Cmdlet category: Iscsi

Add-NcIscsiAuthInitiatorAddressRanges
Add one or more Initiator IP Address Ranges
Add-NcIscsiInterfaceAccess
Add the iSCSI LIF to the accesslist of the specified initiator.
Add-NcIscsiService
Add an iSCSI Service in a Vserver. Each Vserver requires an online iSCSI Service in order to serve data via the iSCSI protocol.
Add-NcIscsiTargetPortalGroupInterface
Add an interface to a target portal group. Interfaces may only be added to a user defined target portal group.
Add-NcIsnsService
Create the iSNS Service for a Vserver.
Disable-NcIscsi
Disable the iSCSI service. Service will be not be available once the call returns with success.
Disable-NcIscsiInterface
Disable an interface for use by iSCSI.
Disable-NcIsns
Stops iSNS service.
Enable-NcIscsi
Enable iSCSI service. Service will be available once the call returns with success.
Enable-NcIscsiInterface
Enable an interface for use by iSCSI
Enable-NcIsns
Start iSNS service.
Get-NcIscsiConnection
Get a list of iSCSI Connection objects.
Get-NcIscsiInitiator
Get a list of remote iSCSI initiator objects.
Get-NcIscsiInitiatorAuth
Get a list of iSCSI initiator authentication configuration objects.
Get-NcIscsiInterface
Get a list of iSCSI target interface objects.
Get-NcIscsiInterfaceAccess
Get a list of iSCSI Interface Accesslist objects.
Get-NcIscsiNodeName
Get the current iscsi node name.
Get-NcIscsiService
Get the list of iSCSI Services.
Get-NcIscsiSession
Get a list of iSCSI session objects.
Get-NcIscsiStatistics
Get a list of iscsi statistics.
Get-NcIscsiTargetAlias
Get the current iscsi target alias.
Get-NcIscsiTargetPortalGroup
Get iSCSI Target Portal Groups.
Get-NcIsns
Gets iSNS service configuration for a Vserver.
New-NcIscsiChapPassword
Generate a 128 bit random password that can be used as a CHAP secret.
New-NcIscsiTargetPortalGroup
Create a new user defined target portal group.
Remove-NcIscsiAuthInitiatorAddressRanges
Remove one or more Initiator IP Address Ranges
Remove-NcIscsiInitiatorAuth
Delete initiator from the authentication list.
Remove-NcIscsiInterfaceAccess
Remove the iSCSI LIF from the accesslist of the specified initiator.
Remove-NcIscsiService
Remove the iSCSI Service in a Vserver.
Remove-NcIscsiTargetPortalGroup
Destroy a target portal. Only user defined target portal groups may be destroyed.
Remove-NcIscsiTargetPortalGroupInterface
Remove an interface from a target portal group. Interfaces may only be removed from a user defined target portal group. Removing an interface will return it to the system defined default group for the interface.
Remove-NcIsnsService
Destroy the iSNS Service for a Vserver.
Set-NcIscsiInitiatorAuth
Set initiator authentication method.
Set-NcIscsiInitiatorAuthChap
Modify CHAP parameters to an existing initiator authentication info whose authentication type is CHAP.
Set-NcIscsiInterface
Modify the attributes of an iSCSI interface object.
Set-NcIscsiNodeName
Set the current iscsi node name.
Set-NcIscsiService
Modify an iSCSI Service in a Vserver.
Set-NcIscsiTargetAlias
Set the current iscsi target alias.
Set-NcIsnsAddress
Set the IP address of the iSNS service to register.
Test-NcIscsi
Get status of the iSCSI service, whether or not it is running.
Update-NcIsns
Forces an update of the iSNS service configuration stored in the external iSNS server.
