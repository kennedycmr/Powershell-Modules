Cmdlet: Set-NcEmsConfig


NAME
    Set-NcEmsConfig
    
SYNOPSIS
    Set the EMS Configuration.
    
SYNTAX
    Set-NcEmsConfig [[-MailFrom] <String>] [[-MailServer] <String>] 
    [[-Suppression] [<Boolean>]] [[-Console] [<Boolean>]] [[-MaxLogShowSize] 
    <Int32>] [[-TraceLogSize] <Int32>] [[-ConsoleLogLevel] <String>] 
    [[-MaxTargetLogSize] <String>] [[-MaxTargetEventSize] <String>] 
    [-MaxFilterCount <Int64>] [-MaxFilterRuleCount <Int64>] 
    [-MaxNotificationCount <Int64>] [-MaxDestinationCount <Int64>] 
    [-AutoSuppressExemptFilter <String>] [-DuplicateSuppressionDuration 
    <String>] [-LogRotationSize <String>] [-ProxyUrl <String>] [-ProxyUser 
    <String>] [-RestDeliveryTimeout <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
