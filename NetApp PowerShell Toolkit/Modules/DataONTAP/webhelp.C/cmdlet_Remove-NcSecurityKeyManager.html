Cmdlet: Remove-NcSecurityKeyManager


NAME
    Remove-NcSecurityKeyManager
    
SYNOPSIS
    Delete a key manager server from Data ONTAP
    
SYNTAX
    Remove-NcSecurityKeyManager [-KeyManagerIPAddress] <String> [[-Nodes] 
    <String[]>] [-Controller <NcController[]>] [-ContinueOnFailure] 
    [-MaxFailureCount <Int32>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Remove-NcSecurityKeyManager [-Query] <KeyManagerInfo> [-Controller 
    <NcController[]>] [-ContinueOnFailure] [-MaxFailureCount <Int32>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Remove-NcSecurityKeyManager -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Delete a key manager server from Data ONTAP
