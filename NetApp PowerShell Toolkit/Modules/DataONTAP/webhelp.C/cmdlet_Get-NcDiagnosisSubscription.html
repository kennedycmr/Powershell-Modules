Cmdlet: Get-NcDiagnosisSubscription


NAME
    Get-NcDiagnosisSubscription
    
SYNOPSIS
    Get a list of Subscriptions for Notifications objects.
    
SYNTAX
    Get-NcDiagnosisSubscription [[-Node] <String[]>] [[-SubscriptionId] 
    <String[]>] [-Monitor <String[]>] [-NotifyDestNode <String[]>] 
    [-NotifyDestHm <String[]>] [-ClassName <String[]>] [-EventType <String[]>] 
    [-Attributes <DiagnosisSubscriptionsInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcDiagnosisSubscription -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcDiagnosisSubscription -Query <DiagnosisSubscriptionsInfo> 
    [-Attributes <DiagnosisSubscriptionsInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of Subscriptions for Notifications objects.
