Cmdlet: Get-NcAggr


NAME
    Get-NcAggr
    
SYNOPSIS
    Gets the aggregates for the current controller.
    
SYNTAX
    Get-NcAggr [[-Name] <String[]>] [-Attributes <AggrAttributes>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcAggr -Template [-Fill] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcAggr -Query <AggrAttributes> [-Attributes <AggrAttributes>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Gets the aggregates for the current controller.
    

PARAMETERS
    -Name <String[]>
        The name of one or more aggregates to get.  Data ONTAP wildcards are 
        permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <AggrAttributes>
        For improved scalability in large clusters, provide an AggrAttributes 
        object with various fields set to indicate which fields to return on 
        all objects.  Undesired fields should remain set to null.  Use 
        "Get-NcAggr -Template" to get the initially empty AggrAttributes 
        object.  If not specified, all data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty AggrAttributes object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Fill
        Specify when using the Template parameter to recursively fill in all 
        object types.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <AggrAttributes>
        For advanced queries, provide an AggrAttributes object with various 
        fields set to indicate which records to retrieve.  Other fields should 
        remain set to null.  Use "Get-NcAggr -Template" to get the initially 
        empty AggrAttributes object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Aggr.AggrAttributes
        
        
    
     
    
NOTES
    
    
        Category: aggr
        API: aggr-get-iter

    --------------  Example 1 --------------
    
    C:\PS>Get-NcAggr
    
    
    Get all aggregates.
    
    
    
    Name                      State       TotalSize  Used  Available Disks 
    RaidType        RaidSize  Volumes
    ----                      -----       ---------  ----  --------- ----- 
    --------        --------  -------
    aggr0                     online       119.5 GB   95%     5.6 GB   3   
    raid_dp,normal     16           1
    aggr0_fas3070cluster01... online       119.5 GB   95%     5.6 GB   3   
    raid_dp,normal     16           1
    aggr1                     online       717.1 GB    3%   696.9 GB   8   
    raid_dp,normal     16           4
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NcAggr aggr1 | select State, TotalSize, Used | ft -AutoSize
    
    
    Get aggregate 'aggr1' and display a subset of its attributes.
    
    
    
    State     TotalSize Used
    -----     --------- ----
    online 769958141952    3
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Get-NcAggr -Attributes space,volumecount | select Name, TotalSize, 
    Used, Volumes | ft -AutoSize
    
    
    Restrict the amount of data transmitted by using the Attributes field.
    
    
    
    Name                           TotalSize Used Volumes
    ----                           --------- ---- -------
    aggr0                       128326377472   95       1
    aggr0_fas3070cluster01_02_0 128326377472   95       1
    aggr1                       769958141952    3       4
    
    
    
    --------------  Example 4 --------------
    
    C:\PS>$a = Get-NcAggr -Template
    Initialize-NcObjectProperty $a AggrSpaceAttributes
    $a.AggrSpaceAttributes.PercentUsedCapacity = ">85"
    Get-NcAggr -Query $a | select Name, Used | ft -AutoSize
    
    
    Using an advanced query, discover which aggregates are more than 85% full.
    
    
    
    Name                        Used
    ----                        ----
    aggr0                         95
    aggr0_fas3070cluster01_02_0   95
