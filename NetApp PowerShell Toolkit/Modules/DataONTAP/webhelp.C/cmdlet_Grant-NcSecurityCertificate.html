Cmdlet: Grant-NcSecurityCertificate


NAME
    Grant-NcSecurityCertificate
    
SYNOPSIS
    Sign a certificate using self-signed root CA.
    
SYNTAX
    Grant-NcSecurityCertificate [-CertificateAuthority] <String> 
    [-CaSerialNumber] <String> [-Csr] <String> [-Vserver <String>] [-ExpireDays 
    <Int32>] [-Format <String>] [-Destination <String>] [-HashFunction 
    <String>] [-PrivateKey <String>] [-FilePassword <String>] [-Username 
    <String>] [-Password <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Sign a certificate using self-signed root CA.

    --------------  Example 1 --------------

    2048 -HashFunction sha256
