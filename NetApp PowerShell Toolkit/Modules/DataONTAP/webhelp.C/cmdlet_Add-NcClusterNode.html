Cmdlet: Add-NcClusterNode


NAME
    Add-NcClusterNode
    
SYNOPSIS
    This cmdlet can be used to add new nodes to cluster.
    
SYNTAX
    Add-NcClusterNode [-NodeIp] <String> [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Add-NcClusterNode [-NodeCount] <Int64> [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    This cmdlet can be used to discover and add new nodes to cluster.  You may 
    provide either NodeCount or NodeIp. To run this cmdlet you must be 
    connected to the cluster. If you are connected to the node (in pre-cluster 
    mode), please use Add-NcCluster to perform cluster join.
    
    A job will be spawned to operate on the cluster and the job id will be 
    returned. The progress of the job can be tracked using the Get-NcJob cmdlet.
