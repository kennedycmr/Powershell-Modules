Cmdlet: Get-NcStorageShelfLocationLed


NAME
    Get-NcStorageShelfLocationLed
    
SYNOPSIS
    Get the Location LED status of one or more shelves.
    
SYNTAX
    Get-NcStorageShelfLocationLed [[-ShelfName] <String[]>] [[-LedStatus] 
    <String[]>] [-Attributes <StorageShelfLocationLedInfo>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcStorageShelfLocationLed -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcStorageShelfLocationLed -Query <StorageShelfLocationLedInfo> 
    [-Attributes <StorageShelfLocationLedInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the Location LED status of one or more shelves
