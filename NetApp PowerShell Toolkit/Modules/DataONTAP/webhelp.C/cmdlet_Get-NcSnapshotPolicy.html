Cmdlet: Get-NcSnapshotPolicy


NAME
    Get-NcSnapshotPolicy
    
SYNOPSIS
    Get information about snapshot policies.
    
SYNTAX
    Get-NcSnapshotPolicy [[-Name] <String[]>] [-Attributes 
    <SnapshotPolicyInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcSnapshotPolicy -Template [-Fill] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcSnapshotPolicy -Query <SnapshotPolicyInfo> [-Attributes 
    <SnapshotPolicyInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get information about snapshot policies.
    

PARAMETERS
    -Name <String[]>
        Restrict results to the specified policies.  Data ONTAP wildcards are 
        permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <SnapshotPolicyInfo>
        For improved scalability in large clusters, provide a 
        SnapshotPolicyInfo object with various fields set to indicate which 
        fields to return on all objects.  Undesired fields should remain set to 
        null.  Use "Get-NcSnapshotPolicy -Template" to get the initially empty 
        SnapshotPolicyInfo object.  If not specified, all data is returned for 
        each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty SnapshotPolicyInfo object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Fill
        Specify when using the Template parameter to recursively fill in all 
        object types.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <SnapshotPolicyInfo>
        For advanced queries, provide a SnapshotPolicyInfo object with various 
        fields set to indicate which records to retrieve.  Other fields should 
        remain set to null.  Use "Get-NcSnapshotPolicy -Template" to get the 
        initially empty SnapshotPolicyInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Snapshot.SnapshotPolicyInfo
