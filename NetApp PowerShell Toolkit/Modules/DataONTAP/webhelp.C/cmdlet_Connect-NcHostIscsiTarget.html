Cmdlet: Connect-NcHostIscsiTarget


NAME
    Connect-NaHostIscsiTarget
    
SYNOPSIS
    Establish a connection to a target discovered by the iSCSI Initiator.
    
SYNTAX
    Connect-NaHostIscsiTarget [-TargetName] <String> [-TargetIp <String>] 
    [-TargetPort <UInt16>] [-InitiatorIp <String>] [-Persistent] 
    [-HeaderDigest] [-DataDigest] [-EnableMultipath] [<CommonParameters>]
    
    Connect-NaHostIscsiTarget [-TargetName] <String> [-TargetIp <String>] 
    [-TargetPort <UInt16>] [-InitiatorIp <String>] [-Persistent] [-Credential 
    <PSCredential>] [-MutualChap] [-HeaderDigest] [-DataDigest] 
    [-EnableMultipath] [-RadiusVerify] [<CommonParameters>]
    
    Connect-NaHostIscsiTarget [-TargetName] <String> [-TargetIp <String>] 
    [-TargetPort <UInt16>] [-InitiatorIp <String>] [-Persistent] [-MutualChap] 
    [-HeaderDigest] [-DataDigest] [-EnableMultipath] [-RadiusGenerate] 
    [-RadiusVerify] [<CommonParameters>]
    
    
DESCRIPTION
    Establish a connection to a target discovered by the iSCSI Initiator.
    

PARAMETERS
    -TargetName <String>
        The Iqn of the target to connect.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -TargetIp <String>
        IP address of the target portal to connect to.  If not specified, the 
        default IP address will be used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -TargetPort <UInt16>
        The target port to connect to.  The default is 3260.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -InitiatorIp <String>
        Ip address the initiator should use to connect to the target.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Persistent
        Specify this flag to add this target to the list of favorite targets.  
        The iSCSI Initiator service ensures that all volumes and devices on a 
        favorite target are available.  It attempts to restore connections to 
        these targets when this machine restarts.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -HeaderDigest
        If specified, header digest will be enabled.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DataDigest
        If specified, data digest will be enabled.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -EnableMultipath
        If specified, multi-path will be enabled for the established connection.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Credential <PSCredential>
        The CHAP username/password to use to authenticate the target.  If not 
        specified, no authentication will be used when connecting to the target.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -MutualChap
        If specified, mutual CHAP will be used.  To use mutual CHAP, an 
        initiator CHAP secret must be set on the Configuration page of the 
        iSCSI Initiator Properties.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RadiusVerify
        If specified, use RADIUS to authenticate target credentials.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RadiusGenerate
        If specified, use RADIUS to generate user authentication credentials.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
