Cmdlet: Get-NcNetNdpActiveNeighbor


NAME
    Get-NcNetNdpActiveNeighbor
    
SYNOPSIS
    Get a list of network ndp active neighbor objects.
    
SYNTAX
    Get-NcNetNdpActiveNeighbor [[-Neighbor] <String[]>] [[-Node] <String[]>] 
    [[-Vserver] <String[]>] [[-SubnetGroup] <String[]>] [[-Port] <String[]>] 
    [-Attributes <NetNdpNeighborCacheInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcNetNdpActiveNeighbor -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcNetNdpActiveNeighbor -Query <NetNdpNeighborCacheInfo> [-Attributes 
    <NetNdpNeighborCacheInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of network ndp active neighbor objects.
    

PARAMETERS
    -Neighbor <String[]>
        One or more neighbor IPv6 addresses.  Data ONTAP wildcards are accepted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Node <String[]>
        One or more nodes.  Data ONTAP wildcards are accepted.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String[]>
        Restrict results to one or more vservers.  Data ONTAP wildcards are 
        accepted.
        
        Required?                    false
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -SubnetGroup <String[]>
        Name of one or more subnet groups.  Data ONTAP wildcards are accepted.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Port <String[]>
        One or more physical ports.  Data ONTAP wildcards are accepted.
        
        Required?                    false
        Position?                    5
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <NetNdpNeighborCacheInfo>
        For improved scalability in large clusters, provide a 
        NetNdpNeighborCacheInfo object with various fields set to indicate 
        which fields to return on all objects.  Undesired fields should remain 
        set to null.  Use "Get-NcNetNdpActiveNeighbor -Template" to get the 
        initially empty NetNdpNeighborCacheInfo object.  If not specified, all 
        data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty NetNdpNeighborCacheInfo object for use with 
        advanced queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <NetNdpNeighborCacheInfo>
        For advanced queries, provide a NetNdpNeighborCacheInfo object with 
        various fields set to indicate which records to retrieve.  Other fields 
        should remain set to null.  Use "Get-NcNetNdpActiveNeighbor -Template" 
        to get the initially empty NetNdpNeighborCacheInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

        This cmdlet supports the common parameters: Verbose, Debug,
