Cmdlet: Set-NcAntivirusRemedy


NAME
    Set-NcAntivirusRemedy
    
SYNOPSIS
    Set the quarantine information of the Anti-Virus engine installed.
    
SYNTAX
    Set-NcAntivirusRemedy [-Action <String>] [-Option <String>] [-Extension 
    <String>] [-Repair <String>] [-Directory <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Set the quarantine information of the Anti-Virus engine installed.
    

PARAMETERS
    -Action <String>
        The remedy action desired. Possible values: "none" - no action will be 
        taken; "repair" - file will be reparied; "delete" - file will be 
        deleted; "quarantine" -  file will be quarantined.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Option <String>
        The option of the remedy action desired. Possible values: "move" - file 
        will be moved; "add_extension" - an extension will be added to the file.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Extension <String>
        The file extension to use for the remedy action.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Repair <String>
        The failed repair option to try. Possible values; "none" - no action 
        will be taken; "delete" - file will be deleted; "quarantine" - file 
        will be quarantined.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Directory <String>
        The pathname of the directory to use for quarantine.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Antivirus.AvRemedyInfo
        
        
    
     
    
NOTES
    
    
        Category: antivirus
        API: av-set-remedy-info
        Family: cluster
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NcAntivirusRemedy -Option add_extension -Extension .vir
    
    
    Set the remedy option to "add_extension" and set the extension to ".vir".
    
    
    
    NcController : 10.61.172.155
    Action       : quarantine

    Extension    : .vir
    Repair       : quarantine
    Directory    : /quarantine
