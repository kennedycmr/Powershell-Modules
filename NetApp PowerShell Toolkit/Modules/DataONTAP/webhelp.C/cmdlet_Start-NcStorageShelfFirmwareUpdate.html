Cmdlet: Start-NcStorageShelfFirmwareUpdate


NAME
    Start-NcStorageShelfFirmwareUpdate
    
SYNOPSIS
    Start shelf firmware download process to update firmware on disk shelves.
    
SYNTAX
    Start-NcStorageShelfFirmwareUpdate [[-Shelf] <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Start-NcStorageShelfFirmwareUpdate [[-ModuleType] <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Start shelf firmware download process to update firmware on disk shelves. 
    
    This operation is asynchronous, and therefore returns no errors that might 
    occur during the download process. This operation will only update firmware 
    on shelves that do not have the latest firmware revision.
