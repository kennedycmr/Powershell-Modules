Cmdlet category: Clone

Add-NcCloneDeletionExtension
Add new supported file extensions for file(s) and lun(s) on the specified volume to be deleted with clone-delete
Disable-NcCloneAutodelete
Disable autodelete on a LUN clone.
Enable-NcCloneAutodelete
Enable autodelete on a LUN clone.
Enable-NcCloneSplit
Enable clone split on split disabled entry.
Get-NcCloneAutodelete
Get the information about file or LUN clone autodeletion
Get-NcCloneDeletion
List clone deletion information
Get-NcCloneLoadSplit
Gets information about load on all nodes due to file or LUN clone split operations.
Get-NcCloneToken
Get information about the reserved token
New-NcClone
Clone a file or LUN.
New-NcCloneToken
Create a token to reserve available split load on a node.
Remove-NcCloneDeletionExtension
Remove the no longer supported file extensions for file(s) and lun(s) on the specified volume to be deleted with clone-delete
Remove-NcCloneToken
Delete token to increase the allowed split load by a node.
Set-NcCloneDeletionMinSize
Required minimum size of the file(s) and lun(s) on the specified volume to be deleted with clone-delete.
Set-NcCloneTokenExpiry
Modify the allowed expiry limit for tokens.
