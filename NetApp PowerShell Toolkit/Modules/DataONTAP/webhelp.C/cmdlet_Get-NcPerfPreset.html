Cmdlet: Get-NcPerfPreset


NAME
    Get-NcPerfPreset
    
SYNOPSIS
    Get performance presets.
    
SYNTAX
    Get-NcPerfPreset [[-Name] <String[]>] [-Attributes <PresetInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcPerfPreset [[-Name] <String[]>] -Detailed [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcPerfPreset -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcPerfPreset -Query <PresetInfo> [-Attributes <PresetInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Get performance presets.
    

PARAMETERS
    -Name <String[]>
        Name of one or more presets.  Data ONTAP wildcards are accepted if the 
        Detailed switch is not present.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <PresetInfo>
        For improved scalability in large clusters, provide an PresetInfo 
        object with various fields set to indicate which fields to return on 
        all objects.  Undesired fields should remain set to null.  Use 
        "Get-NcPerfPreset -Template" to get the initially empty PresetInfo 
        object.  If not specified, all data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  true
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Detailed
        If specified, get more detailed information about the performance 
        presets.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty PresetInfo object for use with advanced queries 
        involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <PresetInfo>
        For advanced queries, provide an PresetInfo object with various fields 
        set to indicate which records to retrieve.  Other fields should remain 
        set to null.  Use "Get-NcPerfPreset -Template" to get the initially 
        empty PresetInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Perf.PresetInfo
