Cmdlet category: Storage Initiator

Get-NcStorageInitiatorDiskPath
Get path information and statistics for a given disk or all disks.
Get-NcStorageInitiatorError
Get all known disk/configuration errors associated with an array or shelves acting like array.
Get-NcStorageInitiatorLoad
Gets disk I/O rates for a given fibre channel initiator port or for all initiator ports.
Get-NcStorageInitiatorPath
Get information and statistics on all known paths to back end storage.
Invoke-NcStorageInitiatorBalance
Balances primary/secondary array LUN paths across available initiator ports based on I/O load.
Invoke-NcStorageInitiatorPathQuiesce
Quiesce an array LUN on a path.
Invoke-NcStorageInitiatorPathResume
Resumes I/O to array LUN on a path that was previously quiesced.
