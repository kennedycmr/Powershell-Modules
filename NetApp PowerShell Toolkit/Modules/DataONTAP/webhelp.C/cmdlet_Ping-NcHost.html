Cmdlet: Ping-NcHost


NAME
    Ping-NcHost
    
SYNOPSIS
    Ping an IP address to verify whether the interface is alive.
    
SYNTAX
    Ping-NcHost [-Host] <String> [-Node] <String> [-Count <Int32>] 
    [-WaitSeconds <Int32>] [-Raw] [-WaitResponse <Int32>] [-PacketSize <Int32>] 
    [-BufferSize <Int32>] [-DisableFragmentation] 
    [-DisableFragmentationUnicastOnly] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Ping-NcHost [-Host] <String> -InterfaceOwner <String> -Interface <String> 
    [-Count <Int32>] [-WaitSeconds <Int32>] [-Raw] [-UseSourcePort] 
    [-WaitResponse <Int32>] [-PacketSize <Int32>] [-BufferSize <Int32>] 
    [-DisableFragmentation] [-DisableFragmentationUnicastOnly] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Ping an IP address to verify whether the interface is alive.
    

PARAMETERS
    -Host <String>
        The address of the destination.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Node <String>
        Specifies the node from which the ping request is to be sent.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Count <Int32>
        The number of pings to try before giving up.  Default is 20.
        
        Required?                    false
        Position?                    named
        Default value                20
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -WaitSeconds <Int32>
        The number of seconds to wait between pings.  Default is 1.
        
        Required?                    false
        Position?                    named
        Default value                1
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Raw
        Specify to get an unwrapped boolean value.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -WaitResponse <Int32>
        Specifies the maximum number of milliseconds to wait for each ping 
        response. Default: 10000 (10 seconds)
        This parameter is applicable only when pinging an IPv4 address.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -PacketSize <Int32>
        The size of the ping packet in bytes. This parameter is applicable for 
        pinging of IPv6 addresses only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -BufferSize <Int32>
        The socket buffer size in bytes. This parameter is applicable for 
        pinging of IPv6 addresses only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DisableFragmentation
        Specify to fragment packets to fit into the minimum IPv6 MTU size for 
        both unicast and multicast addresses.. This parameter is applicable for 
        pinging of IPv6 addresses only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DisableFragmentationUnicastOnly
        Specify to fragment packets to fit into the minimum IPv6 MTU size for 
        unicast addresses only. This parameter is applicable for pinging of 
        IPv6 addresses only.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -InterfaceOwner <String>
        Specifies the name of an interface owner.  It may be a node or Vserver 
        name.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Interface <String>
        Specifies the interface from which the ping request is to be sent.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -UseSourcePort
        If true, send out ping via the port which is hosting the lif.  Only 
        applicable when interface parameter when specified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

NOTES

    --------------  Example 1 --------------
    
    C:\PS>Ping-NcHost -Host 10.61.167.22 -Node Cerebus-03 -Raw
    
    
    Ping a host by IP address and write the raw boolean to the pipeline.
    
    
    
    True
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NcNode | Ping-NcHost 10.61.167.22 -Count 1
    
    
    Ensure each node can ping a host.
    
    
    
    Host                      Success Node
    ----                      ------- ----
    10.61.167.22               True   Cerebus-03
    10.61.167.22               True   Cerebus-04
    10.61.167.22               True   Cerebus-05
    10.61.167.22               True   Cerebus-06
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Ping-NcHost -Host 10.61.167.22 -InterfaceOwner Lionetti01 -Interface 
    Lionetti01_mgmt
    
    
    Ping a host from a specific interface defined on a vserver.
    
    
    
    NcController   : cerebus
    Host           : 10.61.167.22
    Success        : True
    Node           :
    InterfaceOwner : Lionetti01
    Interface      : Lionetti01_mgmt
