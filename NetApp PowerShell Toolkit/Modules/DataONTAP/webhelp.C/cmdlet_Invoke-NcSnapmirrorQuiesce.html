Cmdlet: Invoke-NcSnapmirrorQuiesce


NAME
    Invoke-NcSnapmirrorQuiesce
    
SYNOPSIS
    Disables future transfers to a SnapMirror destination.
    
SYNTAX
    Invoke-NcSnapmirrorQuiesce [-DestinationCluster <String>] 
    -DestinationVserver <String> -DestinationVolume <String> [-SourceCluster 
    <String>] [-SourceVserver <String>] [-SourceVolume <String>] [-Passthru] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Invoke-NcSnapmirrorQuiesce [-Destination] <String> [[-Source] <String>] 
    [-Passthru] [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Invoke-NcSnapmirrorQuiesce [-Query] <SnapmirrorInfo> [-ContinueOnFailure] 
    [-MaxFailureCount <Int32>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Disables future transfers to a SnapMirror destination.  If there is no 
    transfer in progress, the SnapMirror relationship becomes 'Quiesced'.  If 
    there is a transfer in progress, the SnapMirror relationship becomes 
    'Quiescing' until the transfer completes.  If the current transfer aborts, 
    it will be treated like a future transfer and will not restart.  When a 
    SnapMirror relationship is quiesced, it remains in that state across 
    reboots and fail-overs.
    
    The relationship must exist on the destination and you must specify the 
    destination endpoint. 
    
    This cmdlet must be issued on the destination cluster.
    
    If applied to a load-sharing (LS) SnapMirror relationship, all the 
    relationships in the set will be quiesced.

    display the SnapMirror object.
    
    
    
    SourceLocation                    DestinationLocation                 
    Status   MirrorState
    --------------                    -------------------                 
    ------   -----------
    VxeRubble://costea01/unitTestLuns VxeRubble://costea02/unitTestLunsSM 
    quiesced snapmirrored

    C:\PS>$q = Get-NcSnapmirror -Template
    $q.SourceVserver = "beam01"
    Invoke-NcSnapmirrorQuiesce -Query $q
