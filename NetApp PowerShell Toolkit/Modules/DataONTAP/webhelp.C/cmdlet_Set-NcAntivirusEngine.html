Cmdlet: Set-NcAntivirusEngine


NAME
    Set-NcAntivirusEngine
    
SYNOPSIS
    This sets the attributes of the antivirus engine installed.
    
SYNTAX
    Set-NcAntivirusEngine [-LicenseKey <String>] [-Vendor <String>] 
    [-ProductInfo <String>] [-LicenseType <String>] [-LicenseExpiration 
    <DateTime>] [-Seats <Int32>] [-UpdateUrl <String>] [-DesiredActivationCode 
    <String>] [-CurrentActivationCode <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcAntivirusEngine [-Enable] [-LicenseKey <String>] [-Vendor <String>] 
    [-ProductInfo <String>] [-LicenseType <String>] [-LicenseExpiration 
    <DateTime>] [-Seats <Int32>] [-UpdateUrl <String>] [-DesiredActivationCode 
    <String>] [-CurrentActivationCode <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Set-NcAntivirusEngine [-Disable] [-LicenseKey <String>] [-Vendor <String>] 
    [-ProductInfo <String>] [-LicenseType <String>] [-LicenseExpiration 
    <DateTime>] [-Seats <Int32>] [-UpdateUrl <String>] [-DesiredActivationCode 
    <String>] [-CurrentActivationCode <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    This sets the attributes of the antivirus engine installed.
