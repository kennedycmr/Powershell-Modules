Cmdlet: Set-NcEmsDestination


NAME
    Set-NcEmsDestination
    
SYNOPSIS
    Modify the specified EMS destination.
    
SYNTAX
    Set-NcEmsDestination [-Name] <String> [-Mail <String[]>] [-Snmp <String[]>] 
    [-Syslog <String[]>] [-SyslogFacility <String>] [-SnmpCommunity <String>] 
    [-HideParameters [<Boolean>]] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify the specified EMS destination.
