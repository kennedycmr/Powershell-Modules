Cmdlet: Invoke-NcSysstat


NAME
    Invoke-NcSysstat
    
SYNOPSIS
    Get clustered ONTAP controller performance statistics.

DESCRIPTION
    Get clustered ONTAP controller performance statistics.
    
    The cmdlet can be used to retrieve the following performance statistics:  
    System, FCP, NFSv3, NFSv4, CIFS, iSCSI, Volume, Ifnet, LUN, and Disk.
    
    The cmdlet works in the cluster context.  Data ONTAP 8.2 or higher is 
    required to run the cmdlet in the vserver context.
    

PARAMETERS
    -Node <String[]>
        Restrict the returned instances to the given node or nodes.  
        
        If used with the Aggregate switch, get aggregated performance 
        statistics for the given node or nodes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Interval <Int32>
        Sampling frequency.  The default is 5 seconds.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Count <Int32>
        Number of samples to take.  If not specified, samples will be taken 
        indefinitely.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Name <String[]>
        The name of one or more perf object instances for which to collect 
        data.  Wildcards are accepted.  The meaning of the name depends on the 
        type of performance statistics being collected.
        
        FCP:
        -       Name of the fcp interface in the format 
        <vserver_name>.<lif_name> for which to get FCP performance statistics.
        
        NFSv3:
        -       Name of the NFS server Vserver for which to get NFSv3 
        performance statistics.
        
        NFSv4:
        -       Name of the NFS server Vserver for which to get NFSv4 
        performance statistics.
        
        CIFS:
        -       Name of the CIFS server Vserver for which to get CIFS 
        performance statistics.
        
        iSCSI:
        -       Name of the iSCSI interface in the format 
        <vserver_name>.<lif_name> for which to get iSCSI performance statistics.
        
        Volume:
        -       Name of the volume for which to get performance statistics.
        
        Ifnet:
        -       Name of the network interface port for which to get ifnet 
        performance statistics.
        
        LUN:
        -       LUN path of the LUN for which to get performance statistics.
        
        Disk:
        -       Name of the disk for which to get disk performance statistics.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Nfsv3
        If specified, retrieve NFSv3 performance statistics.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Nfsv4

        Position?                    named
