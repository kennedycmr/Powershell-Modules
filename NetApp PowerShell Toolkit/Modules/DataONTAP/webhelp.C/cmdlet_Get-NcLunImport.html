Cmdlet: Get-NcLunImport


NAME
    Get-NcLunImport
    
SYNOPSIS
    Get details of lun-relationships on the controller.
    
SYNTAX
    Get-NcLunImport [-Attributes <LunImportInfo>] [-Controller 
    <NcController[]>] [[-Path] <String[]>] [-Vserver <String[]>] [-ForeignDisk 
    <String[]>] [-ImportOnly] [-VerifyOnly] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    Get-NcLunImport -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcLunImport -Query <LunImportInfo> [-Attributes <LunImportInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Get details of lun-relationships on the controller.
    

PARAMETERS
    -Attributes <LunImportInfo>
        Specify to restrict which data fields are returned by Data ONTAP
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Path <String[]>
        Restrict results to one or more operations by naming the destination 
        luns
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String[]>
        Restrict results to one or more operations where destination lun is in 
        the same vserver.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ForeignDisk <String[]>
        Restrict results to one or more operations by naming the source lun 
        array serial number.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ImportOnly
        "Specify to view only those entries where the operation is import.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -VerifyOnly
        Specify to view only those entries where the operation is verify.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Can be used in Powershell 2 to obtain an empty query element.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <LunImportInfo>
        Specify to query using a object with restricting fields set.
