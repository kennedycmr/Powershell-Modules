Cmdlet category: Snapmirror

Add-NcSnapmirrorSnapshotOwner
Add an owner to preserve a Snapshot copy for a SnapMirror mirror-to-vault cascade configuration.
Disable-NcSnapmirrorAutoProtect
Disable SnapMirror auto protection for a source Vserver.
Enable-NcSnapmirrorAutoProtect
Enable SnapMirror auto protection for a source Vserver.
Enable-NcSnapmirrorProtect
The snapmirror-protect API protects the specified list of volumes. The snapmirror-protect API must be used from the destination cluster. It is only supported for FlexVol endpoints. It is not supported for FlexGroups, FlexGroup constituents, Infinite Volumes, Infinite Volume constituents, Vserver endpoints, or non-Data ONTAP endpoints. This API is available in Data ONTAP 9.3 and later. A job will be spawned to operate on the snapmirror and the job id will be returned. The progress of the job can be tracked using the job APIs.
Get-NcSnapmirror
List SnapMirror relationships.
Get-NcSnapmirrorCapabilities
The snapmirror-capabilities-get API returns capabilities information for an endpoint type. It is only supported for the FlexVol endpoint type. It is not supported for FlexGroup, Infinite Volume, or non-Data ONTAP endpoint types. This API is available in Data ONTAP 9.3 and later.
Get-NcSnapmirrorCRClusterStorageConfiguration
Returns information regarding the configuration of the cluster-wide storage associated with cross-cluster replication of Async Vserver DR configuration data.
Get-NcSnapmirrorCRStatus
Obtain the results of the SnapMirror config replication status check.
Get-NcSnapmirrorCRStatusAggregateEligibility
Returns information regarding the eligibility of aggregates to host configuration replication volumes used for the replication of snapmirror configuration data.
Get-NcSnapmirrorCRStatusComm
Obtain the results of the SnapMirror config replication communication status check.
Get-NcSnapmirrorDestination
Get information about one or more SnapMirror relationships whose source endpoints are in the Vserver or the cluster the cmdlet is issued on.
Get-NcSnapmirrorDpPersonality
This cmdlet returns the Data Protection Personality (DP Personality) of the cluster.
Get-NcSnapmirrorHistory
Get information for one or several SnapMirror operations.
Get-NcSnapmirrorSnapshotOwner
Get owners used to preserve a Snapshot copy for a SnapMirror mirror-to-vault cascade configuration.
Get-NcSnapmirrorUnprotected
The snapmirror-list-unprotected API returns a list of unprotected volumes for the specified source Vserver.
Invoke-NcSnapmirrorAbort
Stops ongoing transfers for a SnapMirror relationship.
Invoke-NcSnapmirrorBreak
Breaks a SnapMirror relationship between a source and destination volume of a data protection mirror.
Invoke-NcSnapmirrorCheck
Starts an operation to compare the contents of a snapshot between the source volume and destination volume.
Invoke-NcSnapmirrorInitialize
Performs the initial update of a SnapMirror relationship.
Invoke-NcSnapmirrorLsInitialize
Performs the initial manual update of a set of load-sharing mirrors.
Invoke-NcSnapmirrorLsPromote
Performs a failover to the destination volume of a load-sharing SnapMirror relationship.
Invoke-NcSnapmirrorLsUpdate
Updates destination volumes of a set of load-sharing mirrors.
Invoke-NcSnapmirrorQuiesce
Disables future transfers to a SnapMirror destination.
Invoke-NcSnapmirrorRelease
Remove a SnapMirror relationship on the source endpoint.
Invoke-NcSnapmirrorRestore
Restore the contents of a selected Snapshot copy from the source volume to the active file system of the destination volume.
Invoke-NcSnapmirrorResume
Enables future transfers for a SnapMirror relationship that has been quiesced.
Invoke-NcSnapmirrorResync
Re-establishes a mirroring relationship between a source volume and a destination volume.
Invoke-NcSnapmirrorUpdate
Updates the destination endpoint of the SnapMirror relationship.
New-NcSnapmirror
Creates a SnapMirror relationship between a source and destination volume.
Remove-NcSnapmirror
Removes the SnapMirror relationship of a source volume and a destination volume.
Remove-NcSnapmirrorSnapshotOwner
Delete an owner used to preserve a Snapshot copy for a SnapMirror mirror-to-vault cascade configuration.
Set-NcSnapmirror
Changes one or more parameters of a SnapMirror relationship.
Set-NcSnapmirrorCRClusterStorageConfiguration
Modifies information regarding the configuration of the cluster-wide storage associated with cross-cluster replication of Async Vserver DR configuration data.
Test-NcSnapmirrorVolume
Test the snapmirror status for the given volume.
Update-NcSnapmirror
Changes one or more parameters of one or more SnapMirror relationships.
