Cmdlet: Get-NcSecurityKeyManagerKeys


NAME
    Get-NcSecurityKeyManagerKeys
    
SYNOPSIS
    Return the authentication passphrase and key ID pairs stored in all of the 
    key managers known to the cluster and whether restored (present in the 
    nodes' key tables).
    
SYNTAX
    Get-NcSecurityKeyManagerKeys [-Node <String[]>] [-KeyManagerIPAddress 
    <String[]>] [-KeyManagerServerStatus <String[]>] [-Attributes 
    <KeyManagerQueryInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcSecurityKeyManagerKeys -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcSecurityKeyManagerKeys -Query <KeyManagerQueryInfo> [-Attributes 
    <KeyManagerQueryInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Return the authentication passphrase and key ID pairs stored in all of the 
    key managers known to the cluster and whether restored (present in the 
    nodes' key tables).
