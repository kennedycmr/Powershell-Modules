Cmdlet: Get-NcClusterLogForwardStatistics


NAME
    Get-NcClusterLogForwardStatistics
    
SYNOPSIS
    Get information about log forwarding statistics
    
SYNTAX
    Get-NcClusterLogForwardStatistics [[-Destination] <String[]>] [[-Node] 
    <String[]>] [[-Port] <UInt64[]>] [-Facility <String[]>] [-Log <String[]>] 
    [-Attributes <ClusterLogForwardStatistics>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcClusterLogForwardStatistics -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcClusterLogForwardStatistics -Query <ClusterLogForwardStatistics> 
    [-Attributes <ClusterLogForwardStatistics>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get information about log forwarding statistics
