Cmdlet: Get-NcQosStream


NAME
    Get-NcQosStream
    
SYNOPSIS
    Get a list of streams.
    
SYNTAX
    Get-NcQosStream [[-StreamId] <String[]>] [[-WorkloadUuid] <String[]>] 
    [[-Vserver] <String[]>] [[-Volume] <String[]>] [[-Qtree] <String[]>] 
    [[-Lun] <String[]>] [-Attributes <QosStreamInfo>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcQosStream -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcQosStream -Query <QosStreamInfo> [-Attributes <QosStreamInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of streams.
    
    This cmdlet is not available for Data ONTAP 8.2 and up.
    

PARAMETERS
    -StreamId <String[]>
        Identifier for this stream.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -WorkloadUuid <String[]>
        Universally unique identifier for the workload to which the stream is 
        attached.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String[]>
        Vserver stream characteristic.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Volume <String[]>
        Volume name stream characteristic.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Qtree <String[]>
        Qtree name stream characteristic.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    5
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Lun <String[]>
        Lun name stream characteristic.  Data ONTAP wildcards are permitted.
        
        Required?                    false
        Position?                    6
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <QosStreamInfo>
        For improved scalability in large clusters, provide a QosStreamInfo 
        object with various fields set to indicate which fields to return on 
        all objects.  Undesired fields should remain set to null.  Use 
        "Get-NcQosStream -Template" to get the initially empty QosStreamInfo 
        object.  If not specified, all data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty QosStreamInfo object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <QosStreamInfo>
        For advanced queries, provide a QosStreamInfo object with various 
        fields set to indicate which records to retrieve.  Other fields should 
        remain set to null.  Use "Get-NcQosStream -Template" to get the 
        initially empty QosStreamInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Qos.QosStreamInfo
