Cmdlet category: Ntp_Server

Get-NcNtpServer
Get the configuration of all the external time servers used by NTP.
Get-NcNtpServerSecurity
Get the NTP security settings.
Get-NcNtpServerStatus
Get the status of the selection of an external time server.
New-NcNtpServer
Create a configuration for an external time server to be used by NTP.
Remove-NcNtpServer
Remove a configuration for an external time server used by NTP.
Reset-NcNtpServer
Reset the external NTP server list to the desired default values. The table is emptied and then populated to the desired default values.
Set-NcNtpServer
Modify a configuration for an external time server used by NTP.
Set-NcNtpServerSecurity
Set the NTP security settings.
Test-NcNtpServer
Validate an existing NTP external time server and report any validation problems.
