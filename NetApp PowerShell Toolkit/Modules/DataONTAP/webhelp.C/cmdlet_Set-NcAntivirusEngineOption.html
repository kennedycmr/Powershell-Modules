Cmdlet: Set-NcAntivirusEngineOption


NAME
    Set-NcAntivirusEngineOption
    
SYNOPSIS
    Modify the attributes of antivirus engine options.
    
SYNTAX
    Set-NcAntivirusEngineOption [-GroupArchiveUnpack [<Boolean>]] 
    [-MaxRecursionDepth <Int64>] [-HeuristicAnalysis [<Boolean>]] 
    [-MacroAnalysis [<Boolean>]] [-ScanMime [<Boolean>]] [-MimeLinesToScan 
    <Int64>] [-CacheSize <String>] [-ProxyHost <String>] [-ProxyPort <Int64>] 
    [-ProxyLogin <String>] [-ProxyPassword <String>] 
    [-DecompressedFileCountLimit <Int64>] [-DecompressedFileSizeLimit 
    <Decimal>] [-DecompressionLayersLimit <Int64>] [-DecompressionSizeFactor 
    <Int64>] [-IsSpywareEnabled [<Boolean>]] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify the attributes of antivirus engine options.
