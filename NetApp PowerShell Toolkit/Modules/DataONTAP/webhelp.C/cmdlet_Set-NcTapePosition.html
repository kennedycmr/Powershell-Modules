Cmdlet: Set-NcTapePosition


NAME
    Set-NcTapePosition
    
SYNOPSIS
    Position tape cartridge
    
SYNTAX
    Set-NcTapePosition [-NodeName] <String> [-DeviceName] <String> [-Operation] 
    <String> [[-Count] <Int64>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Position tape cartridge
    

PARAMETERS
    -NodeName <String>
        Name of owning node.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -DeviceName <String>
        Name of device to be positioned.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Operation <String>
        Tape operation.
        
        Possible values: 
         * 'weof'     - Write end-of-file,
         *  'fsf'      - Forward space files,
         *  'bsf'      - Backward space files,
         *  'fsr'      - Forward space records,
         *  'bsr'      - Backward space records,
         *  'rewind'   - Rewind tape cartridge,
         *  'erase'    - Erase tape media,
         *  'eom'      - End of tape media
        
        Required?                    true
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Count <Int64>
        Count for Positioning.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: tape_mc
        API: tape-mc-info-position
        Family: cluster
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NcTapePosition -NodeName aparajirssd-01 -DeviceName v8.2 
    -Operation weof
    
    
    Set the tape v8.2 for operation write-end-of-file
