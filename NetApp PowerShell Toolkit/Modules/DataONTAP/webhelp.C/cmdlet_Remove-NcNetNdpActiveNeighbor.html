Cmdlet: Remove-NcNetNdpActiveNeighbor


NAME
    Remove-NcNetNdpActiveNeighbor
    
SYNOPSIS
    Delete an existing network ndp active neighbor object.
    
SYNTAX
    Remove-NcNetNdpActiveNeighbor [-Neighbor] <String> [-SubnetGroup] <String> 
    [-Node] <String> [-Port] <String> [-Vserver] <String> [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Delete an existing network ndp active neighbor object.
    

PARAMETERS
    -Neighbor <String>
        The neighbor IPv6 address.  Example: fd20:13::1
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -SubnetGroup <String>
        The subnet group the neighbor belongs to.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Node <String>
        The name of the node.
        
        Required?                    true
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Port <String>
        The physical port.
        
        Required?                    true
        Position?                    4
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String>
        The name of the Vserver.
        
        Required?                    true
        Position?                    5
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
