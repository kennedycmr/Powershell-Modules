Cmdlet: Get-NcNetPlacementCache


NAME
    Get-NcNetPlacementCache
    
SYNOPSIS
    Get the LIF placement cached information.
    
SYNTAX
    Get-NcNetPlacementCache [-Node <String[]>] [-Vserver <String[]>] [-Port 
    <String[]>] [-Address <String[]>] [-NetmaskLength <String[]>] [-Attributes 
    <NetPlacementCache>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcNetPlacementCache -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcNetPlacementCache -Query <NetPlacementCache> [-Attributes 
    <NetPlacementCache>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the LIF placement cached information.  Use Update-NcNetPlacementCache 
    to update the placement cache.
