Cmdlet: Restart-NcNode


NAME
    Restart-NcNode
    
SYNOPSIS
    Reboot the specified node.
    
SYNTAX
    Restart-NcNode [-Name] <String> [-Reason <String>] [-Controller 
    <NcController[]>] [-AllowInhibitTakeover] [-DontDump] 
    [-AllowLifMigrationBeforeReboot] [-IgnoreQuorumWarnings] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Restart-NcNode [-Name] <String> [-Controller <NcController[]>] [-UseSp] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Reboot the specified node.  Only an admin can reboot the node.
    

PARAMETERS
    -Name <String>
        The node name.  This is the name defined during setup.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Reason <String>
        Reason for rebooting the node.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -AllowInhibitTakeover
        If specified, storage takeover by partner will be allowed. By default 
        it is not allowed.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DontDump
        If specified, no core dump will be created.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -AllowLifMigrationBeforeReboot
        If specified, allows migration of LIFs away from this node prior to 
        reboot.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IgnoreQuorumWarnings
        If specified, Ignore quorum warnings before reboot.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -UseSp
        Specify to PowerCycle the node using the Service Processor firmware
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
