Cmdlet category: Disk Encrypt

Clear-NcEncryptedDisk
Cryptographically erases one or more self-encrypting disks (SEDs) that are in the spare or broken state.
Get-NcDiskEncrypt
Return data key ID, FIPS key ID and protection mode of one or more self-encrypting disks (SEDs).
Get-NcDiskEncryptStatus
Return information about the latest storage encryption disk command for one or more nodes.
Set-NcDiskEncrypt
Alters encryption parameters on one or more self-encrypting disks (SEDs).
