Cmdlet: Set-NcRoleConfig


NAME
    Set-NcRoleConfig
    
SYNOPSIS
    Modify the specified attributes of role configuration object.
    
SYNTAX
    Set-NcRoleConfig [-Role] <String> -Vserver <String[]> [-MinUsernameSize 
    <Int32>] [-MinPasswordSize <Int32>] [-LastPasswordsDisallowedCount <Int32>] 
    [-ChangePasswordDurationInDays <Int32>] [-RequireUsernameAlphaNumeric 
    [<Boolean>]] [-RequirePasswordAlphaNumeric [<Boolean>]] 
    [-MinPasswordSpecialCharacter <Int32>] [-PasswordExpirationDuration 
    <Int32>] [-MaxFailedLoginAttempts <Int32>] [-LockoutDurationDays <Int32>] 
    [-RequireInitialPasswordUpdate [<Boolean>]] [-DelayAfterFailedLogin 
    <Int32>] [-PasswdMinLowercaseChars <Int32>] [-PasswdMinUppercaseChars 
    <Int32>] [-PasswdMinDigits <Int32>] [-PasswdExpiryWarnTime <String>] 
    [-AccountExpiryTime <String>] [-AccountInactiveLimit <String>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify the specified attributes of role configuration object.  Omitted 
    (optional) fields will not be changed.
    

PARAMETERS

        Name of the role to modify.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vserver <String[]>

        Accept wildcard characters?  false
        
    -MinUsernameSize <Int32>
        The minimum length of the user name.  Possible values range from 3 to

        Required?                    false

        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -MinPasswordSize <Int32>

    -ChangePasswordDurationInDays <Int32>
