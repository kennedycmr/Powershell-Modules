 Data ONTAP® PowerShell Toolkit
Switch to Data ONTAP 7-mode documentation.

Cmdlet category: Host

Add-NcHostIscsiConnection
Add a connection to an existing iSCSI session.
Add-NcHostIscsiTargetPortal
Add a Data ONTAP storage controller to the iSCSI Initiator target portals. This portal is used to discover iSCSI targets.
Connect-NcHostIscsiTarget
Establish a connection to a target discovered by the iSCSI Initiator.
ConvertTo-NcLun
Create a LUN from a virtual hard disk (VHD or VHDX).
ConvertTo-NcVhd
Convert a VMDK file, a VHDX file, a dynamic VHD file, or a Data ONTAP LUN into a VHD file.
ConvertTo-NcVhdx
Convert a VMDK file, a fixed VHD file, or a Data ONTAP LUN into a VHDX file.
ConvertTo-NcVmdk
Convert a VHD or VHDX file into a VMDK file.
Copy-NcHostFile
Copy a file between LUNs mounted on a Windows host, or within a LUN mounted on a windows host.
Disconnect-NcHostIscsiTarget
Disconnect the host from an iSCSI target.
Dismount-NcHostVolume
Remove one or all mount points from a Windows volume.
Dismount-NcVirtualDisk
Dismounts a VHD file from the local host.
Format-NcHostVolume
Perform a quick NTFS format on an existing Windows volume. Optionally, this can trigger a reclaim of any data previously held by the volume.
Get-NcHostDisk
Lists all disks on the local host that are served by Data ONTAP LUNs.
Get-NcHostFcAdapter
Get FC adapters on the host machine.
Get-NcHostFcConnectivity
Get information about the host fibre channel connections to the current controller.
Get-NcHostIscsiAdapter
Get information about the Microsoft iSCSI Initiator on the host machine.
Get-NcHostIscsiSession
Get current iSCSI sessions on the host machine.
Get-NcHostIscsiTarget
Get the targets discovered by the iSCSI Initiator.
Get-NcHostVolume
Get Windows volume information.
Get-NcHyperV
List Hyper-V virtual machines running on the local host along with all Data ONTAP storage resources that they are using.
Get-NcVirtualDiskAlignment
Get the disk partitions of the given Virtual Hard Disk and determine if they are aligned.
Initialize-NcHostDisk
Initialize a Windows disk with basic partition layout information.
Invoke-NcHostVolumeSpaceReclaim
Reclaim free space on a Windows volume that is hosted on a Data ONTAP LUN.
Invoke-NcVirtualDiskSpaceReclaim
Reclaim space on a virtual disk residing on a Data ONTAP LUN.
Mount-NcHostVolume
Add a drive letter or empty folder mount point to an existing Windows volume.
Mount-NcVirtualDisk
Mounts a VHD file on the local host.
New-NcHostVolume
Create a new Windows volume on an existing disk.
New-NcVirtualDisk
Rapidly create a VHD file on a LUN or CIFS share mounted on a Windows host.
Remove-NcHostFile
Delete a file, completely reclaiming all of the space it consumed on the storage controller.
Remove-NcHostIscsiConnection
Remove a connection from an iSCSI session.
Remove-NcHostVolume
Remove a Windows volume from a disk.
Repair-NcVirtualDiskAlignment
Perform an alignment operation on a misaligned virtual hard disk file. If the virtual hard disk file contains only aligned partitions the cmdlet will do nothing.
Restore-NcHostFile
Restore a single file from a snapshot.
Set-NcHostDisk
Set basic Windows disk options (offline, online, partition style).
Set-NcHostVolumeSize
Grow or shrink a Windows volume.
Set-NcVirtualDiskSize
Resize a VHD or VHDX format virtual disk.
Start-NcHostDiskRescan
Start a Windows disk rescan to force enumeration of any recently added or removed disks.
Wait-NcHostDisk
Wait for a specific disk to appear, or wait for ongoing changes to the disk list to settle.
