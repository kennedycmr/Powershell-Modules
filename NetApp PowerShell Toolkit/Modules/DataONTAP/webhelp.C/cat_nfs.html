Cmdlet category: Nfs

Add-NcNfsExport
Enables pathnames for mounting according to the rules specified.
Add-NcNfsService
Create a new NFS configuration.
Disable-NcNfs
Stop the Vserver's NFS service.
Enable-NcNfs
Start the Vserver's NFS service.
Get-NcNfsExport
Get the rules associated with exports.
Get-NcNfsSecurityFlavor
Get a list of currently supported security flavors.
Get-NcNfsService
Get the NFS server configuration.
Remove-NcNfsExport
Removes the rules for a set of pathnames.
Remove-NcNfsService
Delete an NFS configuration.
Set-NcNfsExport
Functionally similar to Add-NcNfsExport except it returns with an error if the rule does not already exist.
Set-NcNfsService
Modify an NFS configuration.
Test-NcNfs
Get the status of the NFS server, whether or not it is running.
Test-NcNfsExportPermission
Test if the host IP has mount permissions for a specified path.
