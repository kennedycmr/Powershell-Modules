Cmdlet: Get-NcHostIscsiTarget


NAME
    Get-NaHostIscsiTarget
    
SYNOPSIS
    Get the targets discovered by the iSCSI Initiator.
    
SYNTAX
    Get-NaHostIscsiTarget [-IpAddress <String>] [-Port <Int32>] [[-TargetName] 
    <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the targets discovered by the iSCSI Initiator.
    

PARAMETERS
    -IpAddress <String>
        Ip Address of a target portal used to discover this target.  If 
        omitted, all discovered targets are retrieved.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Port <Int32>
        The port that was used to discover this target.  Default is 3260.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -TargetName <String>
        The name of the target to get.  If omitted, all targets are returned.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.PowerShell.SDK.Cmdlets.Toolkit.Windows.IscsiTargetInfo
