Cmdlet: Get-NcPlatformCap


NAME
    Get-NcPlatformCap
    
SYNOPSIS
    Iterate over a list of Platform Capability Support objects.
    
SYNTAX
    Get-NcPlatformCap [[-Node] <String[]>] [[-CapabilityId] <Int64>] 
    [-Attributes <PlatformCapInfo>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcPlatformCap -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcPlatformCap -Query <PlatformCapInfo> [-Attributes <PlatformCapInfo>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Iterate over a list of Platform Capability Support objects.
    

PARAMETERS
    -Node <String[]>
        Specify to filter by Node.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -CapabilityId <Int64>
        Specify to filter by CapabilityId
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <PlatformCapInfo>
        Specify the attributes that should be returned. If not present, all 
        attributes for which information is available will be returned. If 
        present, only the desired attributes for which information is available 
        will be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        The number of times to retry commands that return with errors that may 
        succeed after a retry
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Can be used to obtain an empty query element, which can be used along 
        with Query or Attributes parameter.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <PlatformCapInfo>
        Specify to query using a object with restricting fields set.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.PlatformCap.PlatformCapInfo
