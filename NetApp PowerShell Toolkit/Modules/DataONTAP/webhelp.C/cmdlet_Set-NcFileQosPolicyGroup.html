Cmdlet: Set-NcFileQosPolicyGroup


NAME
    Set-NcFileQosPolicyGroup
    
SYNOPSIS
    Manage the association of QoS policy group to a specified file.
    
SYNTAX
    Set-NcFileQosPolicyGroup [-Path] <String> [-Volume] <String> [-Vserver] 
    <String> [[-QosPolicyGroup] <String>] [[-CachingPolicy] <String>] 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Manage the association of QoS policy group to a specified file.
    
    QoS Policy groups define measurable Service Level Objectives (SLOs) that 
    apply to the storage object with which the policy group is associated. In 
    Data ONTAP 8.2 hierarchical QoS is not supported i.e. QoS policy groups 
    cannot be assigned to multiple objects in the file's storage hierarchy - 
    such as the parent LUN, volume or Vserver.
