Cmdlet category: Aggr

Add-NcAggr
Add disks to the specified aggregate.
Add-NcAggrObjectStore
Attach an object store to an aggregate
Confirm-NcAggrSpareLow
Return true if there is no suitable spare disk available for any filesystem (parity or data) disk.
Disable-NcAggrAutobalance
Disable aggregate autobalance.
Enable-NcAggrAutobalance
Enable aggregate autobalance.
Get-NcAggr
Gets the aggregates for the current controller.
Get-NcAggrAutobalance
Get the configuration for Auto Balance Aggregate.
Get-NcAggrAutobalanceAggrState
Get a list of aggr autobalance objects.
Get-NcAggrAutobalanceNotification
Get the nodes that notify Auto Balance Aggregate about aggregates that need to be balanced.
Get-NcAggrAutobalanceVolumeState
Get the aggregate autobalance volume state.
Get-NcAggrEfficiency
Display the all Flash optimized aggregates and their storage efficiency.
Get-NcAggrEfficiencyCumulated
Display the cumulated efficiency ratio for the specified aggregates
Get-NcAggrNodeInfo
Get information on what possibilities and parameters exist for aggregates on a given controller.
Get-NcAggrObjectStore
Display a list of aggregates which object stores are attached
Get-NcAggrObjectStoreConfig
Get the attributes of an object store configuration.
Get-NcAggrObjectStoreConfigProvider
Returns a list of supported object store providers.
Get-NcAggrObjectStoreVolBtuuids
Display a list of volume buftree UUIDs in a FabricPool aggregate.
Get-NcAggrOption
Get the options that have been set for the specified aggregate.
Get-NcAggrRelocation
Get aggregate relocation status.
Get-NcAggrResyncOptions
Get aggregate resynchronization options for a node in the cluster
Get-NcAggrScrub
Get the status of parity scrubbing on the named aggregate.
Get-NcAggrSpace
Display a list of aggregates and a breakup of their used space.
Get-NcAggrSpare
Get a list of spare disks.
Get-NcAggrStatus
Get status and topology information about one or more aggregates from RAID subsystem.
Get-NcAggrVerify
Get the status of RAID mirror verification on the named aggregate.
Move-NcAggr
Relocate one or more aggregates from on node to another.
New-NcAggr
Create a new aggregate with the given name.
New-NcAggrMirror
Turn an unmirrored aggregate into a mirrored aggregate.
New-NcAggrObjectStoreConfig
Create an object store configuration.
Remove-NcAggr

Remove-NcAggrObjectStoreConfig
Delete an object store configuration
Rename-NcAggr
Rename the specified aggregate.
Resume-NcAggrScrub
Resume parity scrubbing on the named aggregate, plex, or RAID group.
Resume-NcAggrVerify
Resume RAID mirror verification on the named aggregate.

Set-NcAggrAutobalance

Suspend-NcAggrScrub

Suspend-NcAggrVerify
