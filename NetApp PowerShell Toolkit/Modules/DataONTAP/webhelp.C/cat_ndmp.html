Disable-NcNdmpVserver

Enable-NcNdmp
Enable the NDMP service on the given node.
Enable-NcNdmpVserver
Enable NDMP service on the vserver.

Get-NcNdmpVserverAttr
Get NDMP attributes for vserver.
Get-NcNdmpVserverSession
Get details of all vserver-scoped NDMP sessions from the cluster
New-NcNdmpPassword
Generate NDMP password for the given user in a specific Vserver context.

Modify NDMP attributes for vserver.
Stop-NcNdmpNodeSession
Kill a NDMP session on a node
Stop-NcNdmpVserverSession
