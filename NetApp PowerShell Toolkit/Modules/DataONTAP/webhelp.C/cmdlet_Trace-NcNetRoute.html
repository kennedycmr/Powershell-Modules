Cmdlet: Trace-NcNetRoute


NAME
    Trace-NcNetRoute
    
SYNOPSIS
    Trace the route IPv6 packets take to a host.
    
SYNTAX
    Trace-NcNetRoute [-Host] <String> [-Node] <String> [-FirstHop <Int16>] 
    [-HopLimit <Int16>] [-Port <Int64>] [-Nqueries <Int64>] [-WaitTime <Int32>] 
    [-PacketSize <Int64>] [-Controller <NcController[]>] [-DebugMode] 
    [-UseICMP6] [-PrintNumericAddressOnly] [-PrintVerbose] [-Gateway <String>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Trace-NcNetRoute [-Host] <String> -Interface <String> [-InterfaceOwner 
    <String>] [-FirstHop <Int16>] [-HopLimit <Int16>] [-Port <Int64>] 
    [-Nqueries <Int64>] [-WaitTime <Int32>] [-PacketSize <Int64>] [-Controller 
    <NcController[]>] [-DebugMode] [-UseICMP6] [-PrintNumericAddressOnly] 
    [-PrintVerbose] [-Gateway <String>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Trace the route IPv6 packets take to a host. The trace may start at a 
    particular node, or a specified lif.
