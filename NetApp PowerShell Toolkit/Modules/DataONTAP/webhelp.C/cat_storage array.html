Cmdlet category: Storage Array

Get-NcStorageArray
Retrieves a list of all array profiles known to the controller.
Get-NcStorageArrayConfig
Obtain a high level summary of array LUN pathing (connectivity) information.
Get-NcStorageArrayPort
Generate a list of online array ports and their associated arrays.
Rename-NcStorageArray
Rename an array profile on the controller.
Set-NcStorageArray
Update an array profile with new or changed information.
Set-NcStorageArrayPort
Update an array port with new or changed information.
