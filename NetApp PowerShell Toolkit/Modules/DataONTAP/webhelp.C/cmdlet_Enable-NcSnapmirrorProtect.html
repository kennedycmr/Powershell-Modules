Cmdlet: Enable-NcSnapmirrorProtect


NAME
    Enable-NcSnapmirrorProtect
    
SYNOPSIS
    The snapmirror-protect API protects the specified list of volumes. The 
    snapmirror-protect API must be used from the destination cluster. It is 
    only supported for FlexVol endpoints.  It is not supported for FlexGroups, 
    FlexGroup constituents, Infinite Volumes, Infinite Volume constituents, 
    Vserver endpoints, or non-Data ONTAP endpoints. This API is available in 
    Data ONTAP 9.3 and later.  A job will be spawned to operate on the 
    snapmirror and the job id will be returned. The progress of the job can be 
    tracked using the job APIs.
    
SYNTAX
    Enable-NcSnapmirrorProtect [-Policy] <String> [-PathList] <String[]> 
    [-Schedule <String>] [-AutoInitialize [<Boolean>]] 
    [-DestinationVolumePrefix <String>] [-DestinationVolumeSuffix <String>] 
    [-DestinationVserver <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    The snapmirror-protect API protects the specified list of volumes. The 
    snapmirror-protect API must be used from the destination cluster. It is 
    only supported for FlexVol endpoints.  It is not supported for FlexGroups, 
    FlexGroup constituents, Infinite Volumes, Infinite Volume constituents, 
    Vserver endpoints, or non-Data ONTAP endpoints. This API is available in 
    Data ONTAP 9.3 and later.  A job will be spawned to operate on the 
    snapmirror and the job id will be returned. The progress of the job can be 
    tracked using the job APIs.
