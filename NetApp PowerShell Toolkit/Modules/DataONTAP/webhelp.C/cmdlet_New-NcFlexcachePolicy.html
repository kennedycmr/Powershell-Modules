Cmdlet: New-NcFlexcachePolicy


NAME
    New-NcFlexcachePolicy
    
SYNOPSIS
    Create a new FlexCache cache policy.
    
SYNTAX
    New-NcFlexcachePolicy [-Name] <String> [-FileTimeToLive <Int64>] 
    [-DirTimeToLive <Int64>] [-MetafilesTimeToLive <Int64>] 
    [-SymbolicTimeToLive <Int64>] [-OtherTimeToLive <Int64>] 
    [-DelegationLruTimeout <Int64>] [-PreferLocalCache] [-Vserver <String>] 
    [-VserverContext <String>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Create a new FlexCache cache policy.
    
    The cmdlet works in the vserver context for Data ONTAP versions prior to 
    8.2.  In Data ONTAP 8.2, the cmdlet works in the cluster context.
    Flexcache feature is not supported on Data ONTAP 8.3, so this cmdlet works 
    on Data ONTAP 8.2 or lower only.
