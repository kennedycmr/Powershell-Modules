Cmdlet: Set-NcDiagnosisSubscription


NAME
    Set-NcDiagnosisSubscription
    
SYNOPSIS
    Modify system health subscription.
    
SYNTAX
    Set-NcDiagnosisSubscription -Node <String> -SubscriptionId <String> 
    -Monitor <String> -NotifyDestNode <String> -NotifyDestHm <String> 
    -ClassName <String> -EventType <String> [-InstanceName <String>] 
    [-NotifyTable <String>] [-FailThreshold <Int64>] [-PscOption [<Boolean>]] 
    [-TimeGapNotify <Int32>] [-MaxNotifyPeriod <Int32>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify system health subscription.
