Cmdlet: New-NcSecurityCertificate


NAME
    New-NcSecurityCertificate
    
SYNOPSIS
    Create a new digital security certificate.
    
SYNTAX
    New-NcSecurityCertificate [-CommonName] <String> [-Type] <String> [-Size 
    <String>] [-Country <String>] [-State <String>] [-Locality <String>] 
    [-Organization <String>] [-OrganizationUnit <String>] [-EmailAddress 
    <String>] [-ExpireDays <Int32>] [-HashFunction <String>] [-Vserver 
    <String>] [-SystemInternalCertificate] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Create a new digital security certificate.  If a certificate is created 
    with the existing Common Name then a new certificate is generated with a 
    different serial number.

    -Size <String>
