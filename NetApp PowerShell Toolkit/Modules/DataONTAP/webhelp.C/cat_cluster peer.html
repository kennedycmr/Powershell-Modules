Cmdlet category: Cluster Peer

Add-NcClusterPeer
Create a cluster peer relationship.
Disconnect-NcClusterPeer
Terminate a current peering connection.
Get-NcClusterPeer
Returns a list of cluster peer relationships.
Get-NcClusterPeerAuthOffer
Get the specified peer cluster relationship authentication offer information.
Get-NcClusterPeerConnection
Get a list of connections to cluster peers.
Get-NcClusterPeerHealth
Get the health of all local and peer nodes sequentially.
Get-NcClusterPeerPolicy
Get the cluster peering policy configuration.
Ping-NcClusterPeer
Ping the specified peer nodes.
Remove-NcClusterPeer
Delete a cluster peer relationship.
Remove-NcClusterPeerAuthOffer
Cancel the outstanding offer to authenticate with a peer cluster.
Set-NcClusterPeer
Modifies a cluster peer relationship.
Set-NcClusterPeerAuthOffer
Modify the cluster peer relationship authentication offer.
Set-NcClusterPeerLocalName
Modify the local name for a cluster peer.
Set-NcClusterPeerPolicy
Modify the cluster peering policy configuration.
