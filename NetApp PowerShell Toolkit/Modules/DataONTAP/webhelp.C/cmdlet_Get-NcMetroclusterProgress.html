Cmdlet: Get-NcMetroclusterProgress


NAME
    Get-NcMetroclusterProgress
    
SYNOPSIS
    Get a list of MetroCluster progress table info objects.
    
SYNTAX
    Get-NcMetroclusterProgress [[-Index] <UInt64[]>] [-Node <String[]>] 
    [-Attributes <MetroclusterProgressTableInfo>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcMetroclusterProgress -Template [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Get-NcMetroclusterProgress -Query <MetroclusterProgressTableInfo> 
    [-Attributes <MetroclusterProgressTableInfo>] [-Controller 
    <NcController[]>] [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of MetroCluster progress table info objects.
    

PARAMETERS
    -Index <UInt64[]>
        One or more metrocluster progress table entry indices.  Data ONTAP 
        wildcards are accepted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Node <String[]>
        One or more node names.  Data ONTAP wildcards are accepted.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <MetroclusterProgressTableInfo>
        For improved scalability in large clusters, provide a 
        MetroclusterProgressTableInfo object with various fields set to 
        indicate which fields to return on all objects.  Undesired fields 
        should remain set to null.  Use "Get-NcMetroclusterProgress -Template" 
        to get the initially empty MetroclusterProgressTableInfo object.  If 
        not specified, all data is returned for each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty MetroclusterProgressTableInfo object for use 
        with advanced queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <MetroclusterProgressTableInfo>
        For advanced queries, provide a MetroclusterProgressTableInfo object 
        with various fields set to indicate which records to retrieve.  Other 
        fields should remain set to null.  Use "Get-NcMetroclusterProgress 
        -Template" to get the initially empty MetroclusterProgressTableInfo 
        object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Metrocluster.MetroclusterProgressTableInfo
