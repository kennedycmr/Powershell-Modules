Cmdlet category: Security Key Manager

Add-NcSecurityKeyManager
Add a key server to each node in the cluster.
Add-NcSecurityKeyManagerServer
Add a key management server to Data ONTAP.
Get-NcSecurityKeyManager
Lists registered key managers known to the cluster.
Get-NcSecurityKeyManagerBackup
Get backup of the key hierarchy.
Get-NcSecurityKeyManagerKeys
Return the authentication passphrase and key ID pairs stored in all of the key managers known to the cluster and whether restored (present in the nodes' key tables).
Get-NcSecurityKeyManagerQueryV2
Return the key IDs stored in a key management server and whether restored (present in the node's key table).
Get-NcSecurityKeyManagerRestore
Restore the authentication passphrase and key ID pairs from all the key managers known to this cluster.
Initialize-NcSecurityKeyManager
Configure key manager connectivity
New-NcSecurityKeyManagerKey
Create a new authentication passphrase with the key managers
Remove-NcSecurityKeyManager
Delete a key manager server from Data ONTAP
Remove-NcSecurityKeyManagerKmipConfig
Deletes the KMIP configuration
Remove-NcSecurityKeyManagerServer
Delete a key management server from Data ONTAP
Remove-NcSecurityKeyMangerKeys
Deletes the key hierarchy for onboard key manager
Test-NcSecurityKeyManagerVolumeEncryptionSupport
Checks if Volume encryption is supported on this node
Update-NcSecurityKeyManagerCertificate
Update Key Manager SSL Certificate
Update-NcSecurityKeyManagerPassphrase
Update passphrase for onboard key management
