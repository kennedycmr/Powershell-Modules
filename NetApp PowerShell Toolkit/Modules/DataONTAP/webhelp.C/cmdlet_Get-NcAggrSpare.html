Cmdlet: Get-NcAggrSpare


NAME
    Get-NcAggrSpare
    
SYNOPSIS
    Get a list of spare disks.
    
SYNTAX
    Get-NcAggrSpare [[-Name] <String[]>] [[-Owner] <String[]>] [-Attributes 
    <AggrSpareDiskInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcAggrSpare -Template [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    Get-NcAggrSpare -Query <AggrSpareDiskInfo> [-Attributes 
    <AggrSpareDiskInfo>] [-Controller <NcController[]>] [-ZapiRetryCount 
    <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of spare disks.
    

PARAMETERS
    -Name <String[]>
        The name of one or more spare disks to get.  Data ONTAP wildcards are 
        permitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Owner <String[]>
        The name of one or more owners of the spare disks to get.  Data ONTAP 
        wildcards are permitted.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Attributes <AggrSpareDiskInfo>
        For improved scalability in large clusters, provide a AggrSpareDiskInfo 
        object with various fields set to indicate which fields to return on 
        all objects.  Undesired fields should remain set to null.  Use 
        "Get-NcAggrSpare -Template" to get the initially empty 
        AggrSpareDiskInfo object.  If not specified, all data is returned for 
        each object.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NcController[]>
        A clustered Data ONTAP controller to receive this cmdlet, as embodied 
        by an NcController object.  This parameter is returned by the 
        Connect-NcController cmdlet.  If not specified, the value in the global 
        variable CurrentNcController is used.  In the latter case, if 
        CurrentNcController contains a collection of NcController objects, this 
        cmdlet is invoked on each controller in succession.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ZapiRetryCount <Int32>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Template
        Specify to get an empty AggrSpareDiskInfo object for use with advanced 
        queries involving the Query attribute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Query <AggrSpareDiskInfo>
        For advanced queries, provide a AggrSpareDiskInfo object with various 
        fields set to indicate which records to retrieve.  Other fields should 
        remain set to null.  Use "Get-NcAggrSpare -Template" to get the 
        initially empty AggrSpareDiskInfo object.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.C.Types.Aggr.AggrSpareDiskInfo
        
        
    
     
    
NOTES
    
    
        Category: aggr
        API: aggr-spare-get-iter
        Family: cluster
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NcAggrSpare
    
    
    Get the spare disks on the cluster.
    
    
    
    Disk                          TotalSize DiskType        IsDiskShared   
    IsDiskZeroed  IsDiskZeroing  OriginalOwner
    ----                          --------- --------        ------------   
    ------------  -------------  -------------
    VMw-1.12                        88.9 MB vmdisk             False          
    False          False      beam-fs-ps-01
    VMw-1.13                        88.9 MB vmdisk             False          
    False          False      beam-fs-ps-01
    VMw-1.14                        88.9 MB vmdisk             False          
    False          False      beam-fs-ps-01
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NcAggrSpare -Query @{IsDiskZeroed=$false} | Start-NcDiskZeroSpare
    
    
    Get all of the non-zeroed spare disks and start zeroing them.
