Cmdlet: Reset-NcStoragePortDevice


NAME
    Reset-NcStoragePortDevice
    
SYNOPSIS
    Resets a device behind a storage port
    
SYNTAX
    Reset-NcStoragePortDevice [-Node] <String> [-Port] <String> [-BayId 
    <Int64>] [-ShelfName <String>] [-Controller <NcController[]>] 
    [-ZapiRetryCount <Int32>] [<CommonParameters>]
    
    Reset-NcStoragePortDevice [-Node] <String> [-Port] <String> -LoopId <Int64> 
    [-Controller <NcController[]>] [-ZapiRetryCount <Int32>] 
    [<CommonParameters>]
