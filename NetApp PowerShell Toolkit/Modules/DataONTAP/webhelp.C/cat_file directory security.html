Cmdlet category: File Directory Security

Add-NcFileDirectorySecurityNtfsDacl
Add a discretionary access control entry to NTFS security descriptor.
Add-NcFileDirectorySecurityNtfsSacl
Add a system/audit access control entry to NTFS security descriptor.
Add-NcFileDirectorySecurityPolicyTask
Add a file security policy task.
Get-NcFileDirectoryEffectivePermissions
Gives information about the maximal access for the mentioned user on given path.
Get-NcFileDirectorySecurity
Get security information of a file/folder.
Get-NcFileDirectorySecurityNtfs
Get information about one or more NTFS security descriptor.
Get-NcFileDirectorySecurityNtfsDacl
Get information about one or more discretionary access control entries.
Get-NcFileDirectorySecurityNtfsSacl
Get information about one or more system/audit access control entries.
Get-NcFileDirectorySecurityPolicy
Get information about one or more security policies.
Get-NcFileDirectorySecurityPolicyTask
Get information about one or more policy tasks.
New-NcFileDirectorySecurityNtfs
Create a new NTFS security descriptor.
New-NcFileDirectorySecurityPolicy
Add a file security policy task. A task maps security descriptor with a file or folder.
Remove-NcFileDirectorySecurityNtfs
Delete a NTFS security descriptor.
Remove-NcFileDirectorySecurityNtfsDacl
Remove a discretionary access control entry from NTFS security descriptor.
Remove-NcFileDirectorySecurityNtfsSacl
Remove a system/audit access control entry from NTFS security descriptor.
Remove-NcFileDirectorySecurityPolicy
Delete a file/folder security policy.
Remove-NcFileDirectorySecurityPolicyTask
Remove a task from the policy of a vserver.
Remove-NcFileDirectorySecuritySlag
Removes SLAG from the specified volume junction path.
Set-NcFileDirectorySecurity
Apply security settings of a policy.
Set-NcFileDirectorySecurityNtfs
Modify the NTFS security descriptor.
Set-NcFileDirectorySecurityNtfsDacl
Modify an discretionary access control entry of a file security descriptor.
Set-NcFileDirectorySecurityNtfsSacl
Modify an system/audit access control entry of a file security descriptor.
Set-NcFileDirectorySecurityPolicyTask
Remove a task from the policy of a vserver.
