Cmdlet category: Snapmirror

Disable-NaSnapmirror
Disables SnapMirror data transfers and turns off the SnapMirror scheduler.
Enable-NaSnapmirror
Enables SnapMirror data transfers and turns on the SnapMirror scheduler.
Get-NaSnapmirror
Return the SnapMirror status.
Get-NaSnapmirrorConnection
Get connection information for a given snapmirror connection or all connections. The connections are in /etc/snapmirror.conf.
Get-NaSnapmirrorDestination
Returns a list of destination locations and information about SnapMirror relationships for given source locations, which can be a volume name or qtree path.
Get-NaSnapmirrorSchedule
Returns the schedule for a given destination or all destinations.
Get-NaSnapmirrorSyncSchedule
Returns a synchronous schedule for a given destination or all destinations.
Invoke-NaSnapmirrorAbort
Cancels any transfer for a destination.
Invoke-NaSnapmirrorBreak
Breaks a snapmirrored relationship.
Invoke-NaSnapmirrorInitialize
Starts an initial transfer over the network for a specific destination.
Invoke-NaSnapmirrorQuiesce
Pauses transfers to the destination.
Invoke-NaSnapmirrorRelease
Informs SnapMirror that a direct mirror is no longer going to make requests.
Invoke-NaSnapmirrorResume
Resumes transfers to the destination that were quiesced.
Invoke-NaSnapmirrorResync
Kicks off a resync of a broken snapmirrored pair.
Invoke-NaSnapmirrorThrottle
Changes the max transfer rate of an active transfer.
Invoke-NaSnapmirrorUpdate
Starts a transfer over the network for a specific destination.
Remove-NaSnapmirror
Completely delete a SnapMirror relationship.
Remove-NaSnapmirrorConnection
Remove a snapmirror connection. The connections are in /etc/snapmirror.conf.
Remove-NaSnapmirrorSchedule
Delete the schedule for a given destination.
Remove-NaSnapmirrorSyncSchedule
Delete a synchronous schedule for a given destination.
Set-NaSnapmirrorConnection
Set up a connection. Set-NaSnapmirrorConnection will add a new connection or modify an existing one. This cmdlet must be executed on the destination controller. The connections are saved in /etc/snapmirror.conf.
Set-NaSnapmirrorSchedule
Sets the schedule for a given destination.
Set-NaSnapmirrorSyncSchedule
Establishes a synchronous schedule.
Test-NaSnapmirror
Returns true if Snapmirror is available and enabled, false otherwise.
Test-NaSnapmirrorVolume
Return SnapMirror status values for a given volume.
