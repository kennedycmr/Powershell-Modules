Cmdlet: Set-NaSnmpLocation


NAME
    Set-NaSnmpLocation
    
SYNOPSIS
    Sets the SNMP location.
    
SYNTAX
    Set-NaSnmpLocation [-Location] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Sets the location name returned by the  SNMP  agent as the 
    System.sysLocation.0 MIB-II variable.
    

PARAMETERS
    -Location <String>
        The new location value.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snmp.SnmpStatusInfo
        
        
    
     
    
NOTES
    
    
        Category: snmp
        API: system-cli
        Privilege: security-priv-admin, cli-snmp-location
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaSnmpLocation "Main lab"
    
    
    Set the SNMP location value.
    
    
    
    Communities   : {public}
    Contact       : Bill Smith
    IsTrapEnabled : True
    Location      : Main lab
