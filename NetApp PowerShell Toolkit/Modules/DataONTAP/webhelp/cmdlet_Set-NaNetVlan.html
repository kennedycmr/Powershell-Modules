Cmdlet: Set-NaNetVlan


NAME
    Set-NaNetVlan
    
SYNOPSIS
    Configure GVRP (GARP VLAN Registration Protocol) on an interface.
    
SYNTAX
    Set-NaNetVlan [-ParentInterface] <String> [-GvrpOn] [-GvrpOff] 
    [-ActiveOnly] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Configure GVRP (GARP VLAN Registration Protocol) on an interface.
    

PARAMETERS
    -ParentInterface <String>
        The name of a VLAN's parent interface.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -GvrpOn
        Specify to enable GVRP.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -GvrpOff
        Specify to disable GVRP.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ActiveOnly
        If specified, only the active config is modified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Net.InterfaceConfigInfo
        
        
    
     
    
NOTES
    
    
        Category: net
        API: system-cli
        Privilege: security-priv-admin, cli-vlan-modify
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaNetVlan e0c -GvrpOn
    
    
    Enable GVRP on interface 'e0c'.
    
    
    
    Interface            Ipspace               Enabled   PrimaryAddresses
    ---------            -------               -------   ----------------
    e0c                  default-ipspace        False
