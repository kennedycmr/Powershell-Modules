Cmdlet category: Aggr

Add-NaAggr
Add disks to the specified aggregate.
Confirm-NaAggrSpareLow
Return true if there is no suitable spare disk available for any filesystem (parity or data) disk.
Get-NaAggr
Gets the aggregates for the current controller.
Get-NaAggrFilerInfo
Get information on what possibilities and parameters exist for aggregates on a given filer.
Get-NaAggrMediaScrub
Get the status of media scrubbing on the named aggregate, plex, or RAID group.
Get-NaAggrOption
Get the options that have been set for the specified aggregate.
Get-NaAggrScrub
Get the status of parity scrubbing on the named aggregate, plex, or RAID group.
Get-NaAggrSpace
Show the space usage of the aggregate on a per flexible volume basis.
Get-NaAggrVerify
Get the status of RAID mirror verification on the named aggregate.
New-NaAggr
Create a new aggregate.
New-NaAggrMirror
Turns an unmirrored aggregate into a mirrored aggregate by adding a plex to it.
Remove-NaAggr
Destroys the specified aggregate or plex.
Rename-NaAggr
Rename the specified aggregate.
Resume-NaAggrScrub
Resume parity scrubbing on the named aggregate, plex, or RAID group.
Resume-NaAggrVerify
Resume RAID mirror verification on the named aggregate.
Set-NaAggr
Set the specified aggregate or plex online, offline, or restricted.
Set-NaAggrOption
Set the specified option for the given aggregate.
Set-NaAggrRaidType
Split-NaAggrMirror
Remove the specified plex from a mirrored aggregate and create a new unmirrored aggregate with the specified name that contains the plex.
Start-NaAggrScrub
Start parity scrubbing on the named aggregate, plex, or RAID group.
Start-NaAggrVerify
Start RAID mirror verification on the named aggregate.
Stop-NaAggrScrub
Stop parity scrubbing on the named aggregate, plex, or RAID group.
Stop-NaAggrVerify
Stop RAID mirror verification on the named aggregate.
Suspend-NaAggrScrub
Suspend parity scrubbing on the named aggregate, plex, or RAID group.
Suspend-NaAggrVerify
Suspend RAID mirror verification on the named aggregate.
