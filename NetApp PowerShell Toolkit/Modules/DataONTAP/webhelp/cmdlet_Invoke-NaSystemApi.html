Cmdlet: Invoke-NaSystemApi


NAME
    Invoke-NaSystemApi
    
SYNOPSIS
    Sends a raw API request to Data ONTAP.
    
SYNTAX
    Invoke-NaSystemApi [-Request] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    Invoke-NaSystemApi [-RequestXML] <XmlDocument> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Sends a raw API request to Data ONTAP.  The request must be formatted as 
    XML.  See the Manage ONTAP SDK documentation for details.
    

PARAMETERS
    -Request <String>
        The Zephyr API (ZAPI) request.  This argument accepts the request as a 
        string, but it must be a well-formed XML fragment.  Do not include the 
        XML header or the <netapp> wrapper tag.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RequestXML <XmlDocument>
        The Zephyr API (ZAPI) request.  This argument accepts the request as a 
        .NET XmlDocument object.  Do not include the XML header or the <netapp> 
        wrapper tag when building the request.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    System.Xml.XmlDocument
        
        
    
     
    
OUTPUTS
    System.Xml.XmlDocument
