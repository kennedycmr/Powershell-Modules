Cmdlet: Add-NaIscsiInterfaceAccess


NAME
    Add-NaIscsiInterfaceAccess
    
SYNOPSIS
    Add the named network interfaces to the accesslist for the specified 
    initiator.
    
SYNTAX
    Add-NaIscsiInterfaceAccess [-Initiator] <String> [-Interface] <String[]> 
    [-Controller <NaController>] [<CommonParameters>]
    
    Add-NaIscsiInterfaceAccess [-Initiator] <String> [-All] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Add the named network interfaces to the accesslist for the specified 
    initiator. If there is no accesslist, one will be created.
    
    Creating or modifying an accesslist may require shutting down existing 
    iSCSI sessions associated with network interfaces that no longer appear on 
    the accesslist. For example, creating a new accesslist via the add 
    operation may cause sessions to be shut down on network interfaces that are 
    not in the new accesslist. Likewise, removing network interfaces from an 
    existing accesslist via the remove operation may also cause sessions to be 
    shut down.  Note that adding all interfaces (using -All) and removing all 
    interfaces (using -All) will not affect any iSCSI sessions.
    

PARAMETERS
    -Initiator <String>
        The initiator for which to add to the accesslist.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Interface <String[]>
        One or more interfaces to add to the accesslist.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -All
        If specified, all the interfaces will be added to the accesslist.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see
