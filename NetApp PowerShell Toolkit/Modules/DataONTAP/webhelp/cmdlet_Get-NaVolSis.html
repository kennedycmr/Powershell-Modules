Cmdlet: Get-NaVolSis


NAME
    Get-NaVolSis
    
SYNOPSIS
    Get the dedupe status of an SIS volume.
    
SYNTAX
    Get-NaVolSis [[-Path] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Get the dedupe status of an SIS volume.  This cmdlet returns the SisInfo 
    data returned by the volume-list-info API.
    

PARAMETERS
    -Path <String>
        The path of the SIS volume, i.e. /vol/<vol_name>.  Only one path may be 
        specified at a time.  If the path variable is not used then the SIS 
        status for volumes in the controller will be returned.  The /vol/ 
        prefix may be omitted.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Volume.SisInfo
        
        
    
     
    
NOTES
    
    
        Category: volume
        API: volume-list-info
