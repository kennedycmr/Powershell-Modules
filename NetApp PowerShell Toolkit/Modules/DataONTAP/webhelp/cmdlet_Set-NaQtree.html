Cmdlet: Set-NaQtree


NAME
    Set-NaQtree
    
SYNOPSIS
    Modify qtree security style and/or opportunistic locks setting.
    
SYNTAX
    Set-NaQtree [-Path] <String> [-OpLocks <String>] [-SecurityStyle <String>] 
    [-Controller <NaController>] [<CommonParameters>]
    
    Set-NaQtree [-QtreeInfo] <QtreeInfo> [-OpLocks <String>] [-SecurityStyle 
    <String>] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Modify qtree security style and/or opportunistic locks setting.
    

PARAMETERS
    -Path <String>
        Path of an existing qtree. The path should be in this format: 
        /vol/<volume>/<qtree>, or /vol/<volume>/ to select the volume qtree.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -OpLocks <String>
        The new oplocks settings.  Values are 'enabled' or 'disabled'.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -SecurityStyle <String>
        The new security style.  Values are 'unix', 'ntfs' or 'mixed'.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -QtreeInfo <QtreeInfo>
        Alternate parameter set allows for a qtree to be passed in via the 
        pipeline.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Qtree.QtreeInfo
        
        
    
     
    
NOTES
    
    
        Category: qtree
        API: system-cli
        Privilege: security-priv-admin, cli-qtree-oplocks, cli-qtree-security
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaQtree /vol/vol2/qtree2 -OpLocks enabled -SecurityStyle mixed
    
    
    Enable oplocks on qtree '/vol/vol2/qtree2' and set its security style to 
    'mixed'.
    
    
    
    Volume Qtree  Status Security OpLocks
    ------ -----  ------ -------- -------
    vol2   qtree2 normal mixed    enabled
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Set-NaQtree /vol/vol2/ -SecurityStyle ntfs
    
    
    Set the security style on the volume qtree for volume 'vol2' to 'ntfs'.
    
    
    
    Volume Qtree Status Security OpLocks
    ------ ----- ------ -------- -------
    vol2         normal ntfs     enabled
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Get-NaQtree vol2 | Set-NaQtree -OpLocks enabled
    
    
    Enable oplocks on all qtrees in volume 'vol2'.
    
    
    
    Volume Qtree     Status Security OpLocks
    ------ -----     ------ -------- -------
    vol2             normal ntfs     enabled
    vol2   qtree2    normal mixed    enabled
    vol2   testqtree normal mixed    enabled
