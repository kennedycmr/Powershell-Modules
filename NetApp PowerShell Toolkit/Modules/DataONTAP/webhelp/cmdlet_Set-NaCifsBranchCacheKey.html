Cmdlet: Set-NaCifsBranchCacheKey


NAME
    Set-NaCifsBranchCacheKey
    
SYNOPSIS
    Sets the server secret for BranchCache.
    
SYNTAX
    Set-NaCifsBranchCacheKey [-ServerSecret] <String> [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Sets the server secret for BranchCache.
    

PARAMETERS
    -ServerSecret <String>
        A binary string that provides cryptographic data used by the 
        BranchCache content server to generate hashes. Content servers that are 
        serving the same data and are expected to provide the same hash values 
        must use the same key. After changing this value, any existing cached 
        content will be identified as stale and retrieved from the content 
        server again.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: cifs
        API: cifs-branchcache-set-key

    C:\PS>Set-NaCifsBranchCacheKey ABC123
    
    
    Change the server secret used by the BranchCache server to ABC123.
