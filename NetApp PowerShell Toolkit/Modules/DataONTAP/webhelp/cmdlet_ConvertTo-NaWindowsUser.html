Cmdlet: ConvertTo-NaWindowsUser


NAME
    ConvertTo-NaWindowsUser
    
SYNOPSIS
    Obtain windows user information for a given unix user name or user id, 
    using whatever mechanism the controller's nameservice is currently using.
    
SYNTAX
    ConvertTo-NaWindowsUser -Username <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    ConvertTo-NaWindowsUser -Uid <Int64> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Obtain windows user information for a given unix user name or user id, 
    using whatever mechanism the controller's nameservice is currently using.
    

PARAMETERS
    -Username <String>
        A Unix user name.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Uid <Int64>
        A Unix user Id.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Nameservice.WindowsUser
        
        
    
     
    
NOTES
    
    
        Category: nameservice
        API: nameservice-map-unix-to-windows, nameservice-map-uid-to-user-name
    
    --------------  Example 1 --------------
    
    C:\PS>ConvertTo-NaWindowsUser -Uid 65531
    
    
    Get the Windows user that the Unix user with UID 65531 maps to.
    
    
    
    UserName                                                    Domain
    --------                                                    ------
    homer                                                       KENNETH
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>ConvertTo-NaWindowsUser -Username simpson
    
    
    Get the Windows user that the Unix user with username simpson maps to.
    
    
    
    UserName                                                    Domain
    --------                                                    ------
    homer                                                       KENNETH
