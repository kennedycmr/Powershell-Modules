Cmdlet: Wait-NaHostDisk


NAME
    Wait-NaHostDisk
    
SYNOPSIS
    Wait for a specific disk to appear, or wait for ongoing changes to the disk 
    list to settle.
    
SYNTAX
    Wait-NaHostDisk [-DiskIndex] <UInt32> [-Timeout <Int64>] 
    [<CommonParameters>]
    
    Wait-NaHostDisk [-ControllerLunPath] <String> [-ControllerName] <String> 
    [-Timeout <Int64>] [<CommonParameters>]
    
    Wait-NaHostDisk -SerialNumber <String> [-Timeout <Int64>] 
    [<CommonParameters>]
    
    Wait-NaHostDisk -SettlingTime <Int32> [-Timeout <Int64>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Wait for a specific disk to appear, or wait for ongoing changes to the disk 
    list to settle.
    

PARAMETERS
    -DiskIndex <UInt32>
        The Windows disk index of a specific disk being sought.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Timeout <Int64>
        The length of time in milliseconds to wait for a specific disk, or for 
        stability in the disk list, before giving up.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ControllerLunPath <String>
        The path of the LUN that backs the host disk being sought, in the form 
        /vol/<volume>/<lun> or /vol/<volume>/<qtree>/<lun>.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ControllerName <String>
        The name of the controller hosting the LUN that backs the host disk 
        being sought.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -SerialNumber <String>
        The serial number of the host disk being sought.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -SettlingTime <Int32>
        Length of disk population settling interval, in milliseconds.  If 
        specified, this cmdlet will wait for the host disk list to remain 
        stable for at least this length of time.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.HostUtilities.DiskDiscovery.DiskResource
        
        
    
     
    
NOTES
    
    
        Category: host
    
    --------------  Example 1 --------------
    
    C:\PS>Start-NaHostDiskRescan; Wait-NaHostDisk -SettlingTime 5000
    
    
    Initiate a disk rescan and wait until any changes to the disk population 
    are complete.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Start-NaHostDiskRescan; Wait-NaHostDisk -ControllerLunPath 
    /vol/vol1/lun3 -ControllerName benson | New-NaHostVolume
    
    
    Start a VDS rescan, wait for the given disk to become available and create 
    a volume.
    
    
    
    MountPoints Disk     Size GuidPaths
    ----------- ----     ---- ---------
    {L:\}         11 994.0 MB 
    {\\?\Volume{e092a3af-4176-4a48-808f-e865f8b35c48}\}
