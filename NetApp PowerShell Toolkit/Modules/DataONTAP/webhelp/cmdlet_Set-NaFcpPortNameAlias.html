Cmdlet: Set-NaFcpPortNameAlias


NAME
    Set-NaFcpPortNameAlias
    
SYNOPSIS
    Set an alias for a World Wide Port Name of an initiator that might login to 
    the target.
    
SYNTAX
    Set-NaFcpPortNameAlias [-PortName] <String> [-Alias] <String> [-Force] 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Set an alias for a World Wide Port Name of an initiator that might login to 
    the target.
    

PARAMETERS
    -PortName <String>
        WWPN for which alias is being set.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Alias <String>
        Alias to be set for the given WWPN.  The alias can be 32-characters 
        long and may contain: A-Z, a-z, 0-9, _,-,.,{,} and no spaces.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        When set to true, the WWPN associated with the alias will be 
        over-written.  Default value is false.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: fcp
        API: fcp-wwpnalias-set
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaFcpPortNameAlias 50:0a:09:81:97:09:3b:1f port1
    
    
    Set the alias for the specified WWPN to 'port1'.
