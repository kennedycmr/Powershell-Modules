Cmdlet: Get-NaAggr


NAME
    Get-NaAggr
    
SYNOPSIS
    Gets the aggregates for the current controller.
    
SYNTAX
    Get-NaAggr [[-Name] <String[]>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    The Get-NaAggr cmdlet retrieves one or more aggregates from a Data ONTAP 
    controller.  Wildcards are supported for the aggregate name.
    

PARAMETERS
    -Name <String[]>
        
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    System.String
        
        The name of the aggregate to retrieve.
    
     
    
OUTPUTS
    DataONTAP.Types.Aggr.AggrInfo
        
        The AggrInfo class describes a Data ONTAP aggregate.
    
     
    
NOTES
    
    
        Category: aggr
        API: aggr-list-info
    
    --------------  All aggregates --------------
    
    C:\PS>Get-NaAggr
    
    
    Retrieves all aggregates from the current controller.
    
    
    
    
    
    --------------  A specific aggregate --------------
    
    C:\PS>Get-NaAggr aggr1
    
    
    Retrieves the aggregate named 'aggr1'.
    
    
    
    
    
    --------------  Wildcards --------------
    
    C:\PS>Get-NaAggr aggr1*
    
    
    Retrieves all aggregates whose names begin with 'aggr1'.
