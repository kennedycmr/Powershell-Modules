Cmdlet: Get-NaLicense


NAME
    Get-NaLicense
    
SYNOPSIS
    Returns information about the current list of licensed Data ONTAP services.
    
SYNTAX
    Get-NaLicense [[-Name] <String[]>] [-Installed] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns information about the current list of licensed Data ONTAP services, 
    their codes, the type of license, and, if it is a time limited license, the 
    expiration date. It also tells the services that are not licensed for your 
    appliance, or if a time limited licensed service has expired.
    
    For Data ONTAP 8.2 and up, only current licensed packages are returned.  
    Prior to Data ONTAP 8.2, use the -Installed switch to list only installed 
    services.
    

PARAMETERS
    -Name <String[]>
        The names of the licenses desired, or blank to list them all.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Installed
        Specify to retrieve only installed licenses.
        
        For Data ONTAP 8.2 and up, this parameter has no effect.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
