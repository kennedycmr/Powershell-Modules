Cmdlet category: Volume

Get-NaVol
Get volume status.
Get-NaVolAutosize
Given the name of a flexible volume, get the autosize settings.
Get-NaVolCharmap
Return charmap information for a specified volume.
Get-NaVolCloneSplit
Display the progress in separating clones from their underlying parent volumes and snapshots.
Get-NaVolCloneSplitEstimate
Display an estimate of additional storage required in the underlying aggregate to perform a volume clone split operation.
Get-NaVolContainer
Return the name of the containing aggregate for the named flexible volume.
Get-NaVolFootprint
Get a list of volumes and a breakdown of their data and metadata footprints in their parent aggregates.
Get-NaVolLanguage
Get the given volume's language mapping.
Get-NaVolMove
Obtain the status of a volume move. If a volume name is not specified, all in progress volume move operations are returned.
Get-NaVolOption
Get the options that have been set for the specified volume.
Get-NaVolRoot
Get the root volume for the current controller.
Get-NaVolSis
Get the dedupe status of an SIS volume.
Get-NaVolSize
Given the name of a flexible volume, return its current size in bytes.
Get-NaVolSpace
Get a list of volumes and a breakdown of their space usage.
Invoke-NaVolMoveCutover
Initiate manual cutover. If cutover cannot be initiated or completed, the API will return with an error. The move will pause and an EMS message will be printed. The Get-NaVolMove cmdlet will show the state of the move as move(paused). The user can resume or abort the move.
New-NaVol
Create a new flexible volume with the given name and characteristics.
New-NaVolClone
Create a flexible volume that is a clone of a "backing" or "parent" flexible volume.
Remove-NaVol
Destroy the specified volume or plex.
Rename-NaVol
Renames the specified volume.
Resume-NaVolMove
Resume a previously paused move.
Set-NaVol
Set the specified volume online, offline, or restricted.
Set-NaVolAutosize
Given the name of a flexible volume, set the autosize settings.
Set-NaVolCharmap
Associate a charmap description with a specified volume.
Set-NaVolLanguage
Set the given volume's language mapping.
Set-NaVolOption
Set the option named 'Key' to the value specified by 'Value' in the specified volume.
Set-NaVolSize
Given the name of a flexible volume, set the volume's size to the stated amount.
Set-NaVolTotalFiles
Set a volume's 'total-files' value to the given quantity.
Start-NaVolCloneSplit
Begin the process by which the given clone is split off from its underlying parent volume and snapshot.
Start-NaVolMove
Start-NaVolMove will move a single 7-mode flexvol between two aggregates on the same controller.
Stop-NaVolCloneSplit
Stop the process of splitting off a clone from its parent volume and snapshot.
Stop-NaVolMove
Stop the volume move operation.
Suspend-NaVolMove
Suspends the volume move operation.
