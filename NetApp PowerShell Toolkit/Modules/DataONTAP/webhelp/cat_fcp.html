Cmdlet category: Fcp

Clear-NaFcpAdapterStatistics
Resets the stats for the specified Fibre Channel Target Adapter.
Clear-NaFcpPortNameAlias
Remove an alias for a World Wide Port Name of an initiator.
Disable-NaFcp
Stops FCP service.
Disable-NaFcpAdapter
Bring the adapter offline.
Enable-NaFcp
Starts FCP service.
Enable-NaFcpAdapter
Bring the adapter online.
Get-NaFcpAdapter
Gets information such as nodename/portname and link state about the specified Fiber Channel target adapter, or if no adapter is specified, about all the FC target adapters.
Get-NaFcpAdapterStatistics
Get statistics about the Fibre Channel target adapters.
Get-NaFcpClusterMode
Get the current cfmode setting for the system.
Get-NaFcpInitiator
Get the list of initiators currently connected to the specified Fibre Channel target adapter.
Get-NaFcpNodeName
Get the current FCP node name.
Get-NaFcpPortName
Get the list of valid Fibre Channel target port names on a filer's local adapters.
Get-NaFcpPortNameAlias
Get the list of WWPN for a given alias or an alias for for a given WWPN or complete list of all current alias-WWPN mappings.
Ping-NaFcp
Send an ELS ECHO frame to a Fibre Channel address or WWPN. Returns true if the ping was successful.
Ping-NaFcpInfo
Send an ELS ECHO frame to a Fibre Channel address or WWPN. The cmdlet pings the address or WWPN count times and returns the number of successful pings and times.
Set-NaFcpAdapterMediaType
Sets the link type on the Fibre Channel target adapter.
Set-NaFcpAdapterPartner
Sets the name of the partner adapter which the local adapter should takeover.
Set-NaFcpAdapterSpeed
Sets the speed on the Fibre Channel target adapter.
Set-NaFcpClusterMode
Set the current cfmode setting for the system.
Set-NaFcpNodeName
Set the current FCP node name.
Set-NaFcpPortName
Set a valid but unused port name on a local Fibre Channel target adapter.
Set-NaFcpPortNameAlias
Set an alias for a World Wide Port Name of an initiator that might login to the target.
Switch-NaFcpPortName
Swap port names of two local Fibre Channel target adapters.
Test-NaFcp
Get status of the FCP service, whether or not it is running.
