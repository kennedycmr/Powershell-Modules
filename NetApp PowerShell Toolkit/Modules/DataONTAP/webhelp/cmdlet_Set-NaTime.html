Cmdlet: Set-NaTime


NAME
    Set-NaTime
    
SYNOPSIS
    Set current date and time  to the specified date and time.
    
SYNTAX
    Set-NaTime [-UTC] [-Controller <NaController>] [<CommonParameters>]
    
    Set-NaTime [-Time] <Int32> [-UTC] [-Controller <NaController>] 
    [<CommonParameters>]
    
    Set-NaTime [-DateTime] <DateTime> [-UTC] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Set current date and time  to the specified date and time.
    

PARAMETERS
    -UTC
        Specify if time values are presented as universal coordinated time.  
        Otherwise, the time should be the local time.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Time <Int32>
        The new time value as a unix time (seconds since 1/1/1970).
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -DateTime <DateTime>
        The new time value as a .NET DateTime object.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Clock.Time
