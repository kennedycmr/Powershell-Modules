Cmdlet: ConvertTo-NaUnixUser


NAME
    ConvertTo-NaUnixUser
    
SYNOPSIS
    Obtain a unix user name and user id from a windows username or SID, using 
    whatever mechanism the controller's nameservice is currently using.
    
SYNTAX
    ConvertTo-NaUnixUser -Domain <String> -Username <String> [-Controller 
    <NaController>] [<CommonParameters>]
    
    ConvertTo-NaUnixUser -Sid <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Obtain a unix user name and user id from a windows username or SID, using 
    whatever mechanism the controller's nameservice is currently using.
    

PARAMETERS
    -Domain <String>
        Windows domain of the user.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Username <String>
        A Windows user name.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Sid <String>
        A Windows SID.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Nameservice.UnixUser
        
        
    
     
    
NOTES
    
    
        Category: nameservice
        API: nameservice-map-windows-to-unix, nameservice-map-sid-to-uid, 
        nameservice-map-user-name-to-uid, nameservice-map-uid-to-user-name
    
    --------------  Example 1 --------------
    
    C:\PS>ConvertTo-NaUnixUser -Domain KENNETH -Username steven
    
    
    Get the unix user that the Windows user "steven" in the "KENNETH" domain 
    maps to.
    
    
    
    UserName            UserId
    --------            ------
    pcuser              65534
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>ConvertTo-NaUnixUser -Sid 
    S-1-5-21-1088268088-1183557120-1454356382-500
    
    
    Get the unix user that the Windows user with the given SID maps to.
    
    
    
    UserName            UserId
    --------            ------
    root                0
