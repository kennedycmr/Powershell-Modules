Cmdlet: Get-NaShelfEnvironment


NAME
    Get-NaShelfEnvironment
    
SYNOPSIS
    Returns the environmental information for a specified shelf or channel.
    
SYNTAX
    Get-NaShelfEnvironment [-Channel] <String> [[-Shelf] <Int32>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns the environmental information for a specified shelf or channel.
    

PARAMETERS
    -Channel <String>
        Get environment information for a shelf on this channel.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Shelf <Int32>
        Get environment information for the specified Shelf ID on the specified 
        Channel.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Ses.ShelfEnvironChannelInfo
        
        
    
     
    
NOTES
    
    
        Category: ses
        API: storage-shelf-environment-list-info
