Cmdlet: Remove-NaCredential


NAME
    Remove-NaCredential
    
SYNOPSIS
    Remove saved login credentials for a Data ONTAP controller.
    
SYNTAX
    Remove-NaCredential [-SystemScope] [<CommonParameters>]
    
    Remove-NaCredential -Controller <NaController> [-SystemScope] 
    [<CommonParameters>]
    
    Remove-NaCredential [-Name] <String> [-SystemScope] [<CommonParameters>]
    
    Remove-NaCredential [-Credential] <NaCredential> [<CommonParameters>]
    
    
DESCRIPTION
    Remove saved login credentials for a Data ONTAP controller.
    

PARAMETERS
    -SystemScope
        Remove credentials for the specified controller that were saved using 
        system scope.  If not specified, remove credentials that were saved 
        with user scope.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        An NaController object, as returned by Connect-NaController.  If 
        neither this parameter nor -Name is specified, this cmdlet will use the 
        value in $global:CurrentNaController (if set).
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Name <String>
        The name or address of a storage controller.  If a host name is 
        specified, it must be resolvable to an address.  If neither this 
        parameter nor -Controller is specified, this cmdlet will use the value 
        in $global:CurrentNaController (if set).
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Credential <NaCredential>
        An NaCredential object, as returned by Get-NaCredential.  If specified, 
        that record is removed from the credentials cache (including the 
        correct value of system scope).
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: toolkit
    
    --------------  Example 1 --------------
    
    C:\PS>Remove-NaCredential dunn
    
    
    Remove the cached credentials for storage controller 'dunn' that were saved 
    with user scope.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaCredential | Remove-NaCredential
    
    
    Clear all entries from the credentials cache that are visible to the 
    current user, including those with system scope.
