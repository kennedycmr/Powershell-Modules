Cmdlet: Initialize-NaSecureAdminSsh


NAME
    Initialize-NaSecureAdminSsh
    
SYNOPSIS
    Initialize the controller's SSH server, generating host and server keys.
    
SYNTAX
    Initialize-NaSecureAdminSsh [[-Ssh1HostKeySize] <Int32>] 
    [[-Ssh1ServerKeySize] <Int32>] [[-Ssh2HostKeySize] <Int32>] [-Force] 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Initialize the controller's SSH server, generating host and server keys.
    

PARAMETERS
    -Ssh1HostKeySize <Int32>
        SSH1 protocol host key size.  Key size must be greater than 384 bits 
        and less than 2048 bits.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Ssh1ServerKeySize <Int32>
        SSH1 protocol server key size.  Key size must be greater than 384 bits 
        and less than 2048 bits.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Ssh2HostKeySize <Int32>
        SSH2 protocol host key size.  Key size must be greater than 768 bits 
        and less than 2048 bits.
        
        Required?                    false
        Position?                    3
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force
        Specify to force SSH setup even if the SSH server has already been 
        configured.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: secureadmin
        API: system-cli
        Privilege: security-priv-advanced, cli-secureadmin-setup
    
    --------------  Example 1 --------------
    
    C:\PS>Initialize-NaSecureAdminSsh -Ssh1HostKeySize 2048 -Ssh1ServerKeySize 
    1920 -Ssh2HostKeySize 2048
    
    
    Initialize SSH and generate a 2048 bit SSH1 host key, a 1920 bit SSH1 
    server key, and a 2048 bit SSH2 host key.
