Cmdlet: Get-NaCifsHomeDirectory


NAME
    Get-NaCifsHomeDirectory
    
SYNOPSIS
    Return path to the user's CIFS home directory if it exists, or return the 
    paths to all home directories.
    
SYNTAX
    Get-NaCifsHomeDirectory [[-User] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Return path to the user's CIFS home directory if it exists, or return the 
    paths to all home directories.
    

PARAMETERS
    -User <String>
        Name of the user. If the filer is using the "mapped" CIFS home 
        directory naming style, then a mapped Unix name is provided. For the 
        domain naming style, a domain/user is provided. Otherwise the filer 
        will expect the NT name for the user. The filer's CIFS home directory 
        naming style can be obtained with the Get-NaOption cmdlet, using 
        parameter "cifs.home_dir_namestyle".
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.String
        
        
    
     
    
NOTES
    
    
        Category: cifs
        API: cifs-homedir-paths-get, cifs-homedir-path-get-for-user
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaCifsHomeDirectory
    
    
    Return the list of home directories on the current controller.
    
    
    
    /vol/vol0/user1
    /vol/vol0/user2
