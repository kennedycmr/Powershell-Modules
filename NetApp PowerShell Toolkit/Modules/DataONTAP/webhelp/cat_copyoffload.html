Cmdlet category: Copyoffload

Disable-NaCopyOffload
Disable the copy offload feature.
Enable-NaCopyOffload
Enable the copy offload feature.
Get-NaCopyOffload
Provides status on an on-going copyoffload-copy-file request. The response will indicate the state of the on-going operation and the number of bytes copied thus far.
Start-NaCopyOffload
Copy specified whole/sub source file to specified destination file. This operation is handled asynchronously.
Stop-NaCopyOffload
Abort an on-going copyoffload-copy-file request.
Test-NaCopyOffload
Show current copy offload feature state.
