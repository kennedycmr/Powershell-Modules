Cmdlet: Set-NaNetInterface


NAME
    Set-NaNetInterface
    
SYNOPSIS
    Configure network interface.
    
SYNTAX
    Set-NaNetInterface [-InterfaceName] <String> [-Alias] [-RemoveAlias] 
    [-NoDdns] [[-Address] <String>] [-Netmask <String>] [-PrefixLength 
    <UInt32>] [-Broadcast <String>] [-MediaType <String>] [-FlowControl 
    <String>] [-DadAttempts <UInt32>] [-MtuSize <UInt32>] [-Up] [-Down] 
    [-Trusted [<Boolean>]] [-Wins [<Boolean>]] [-Nfo [<Boolean>]] [-Partner 
    <String>] [-ActiveOnly] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Configure network interface.  This affects both the current configuration 
    and the persistent configuration in /etc/rc.
    

PARAMETERS
    -InterfaceName <String>
        The name of the interface to configure.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Alias
        Specify to add an address as an alias.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RemoveAlias
        Specify to remove an address as an alias.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -NoDdns
        Specify to prevent DDNS advertisement.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Address <String>
        IP address.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Netmask <String>
        Netmask for the specified address.  Mutually exclusive with 
        PrefixLength.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -PrefixLength <UInt32>
        The number of bits reserved for subdividing networks into subnetworks.  
        Mutually exclusive with Netmask.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Broadcast <String>
        The broadcast address for this network address.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -MediaType <String>
        Specifies the Ethernet media type used.  Values are tp, tp-fd, 100tx, 
        100tx-fd, 1000fx, 10g-sr, auto.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -FlowControl <String>
        Flow control type.  Values are none, receive, send, full.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -DadAttempts <UInt32>
        (Inet6 only) Specifies Duplicate Address Detection-attempts to use for 
        the multicast interface.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -MtuSize <UInt32>
        The MTU (maximum transmission unit) to use for the network interface.  
        Range: 296-9196
        
        Required?                    false
        Position?                    named
        Default value                1500
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Up
        Specify to bring this interface online.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Down
        Specify to take this interface offline.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Trusted [<Boolean>]
        Specifies that the network to which the interface is attached is 
        trusted relative to firewall-style security (default).
        
        Required?                    false
        Position?                    named
        Default value                true
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Wins [<Boolean>]
        Specifies that the network interface is to be registered with Windows 
        Internet Name Services (default).
        
        Required?                    false
        Position?                    named
        Default value                true
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Nfo [<Boolean>]
        Specifies that negotiated failover is to be enabled for the network 
        interface.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Partner <String>
        Name of cluster partner interface that will failover to this interface. 
         Specify an empty string to clear the partner interface.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ActiveOnly
        If specified, only the active config is modified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
