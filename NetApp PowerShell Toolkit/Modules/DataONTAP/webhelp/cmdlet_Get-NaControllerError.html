Cmdlet: Get-NaControllerError


NAME
    Get-NaControllerError
    
SYNOPSIS
    Get a description of a Data ONTAP error code.
    
SYNTAX
    Get-NaControllerError [[-Code] <Int32>] [<CommonParameters>]
    
    Get-NaControllerError [-Name] <String> [<CommonParameters>]
    
    
DESCRIPTION
    Get a description of a Data ONTAP error code.  A specific error may be 
    retrieved by numerical code or by its shorthand name, both of which are 
    normally returned by Data ONTAP if an API error occurs.  Specify neither to 
    get a listing of all defined error codes.
    

PARAMETERS
    -Code <Int32>
        The numerical error code returned by Data ONTAP.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Name <String>
        An error name returned by Data ONTAP, such as 'EDEST_IS_LS_VOLUME2'.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Toolkit.Utils.ControllerErrorInfo
        
        
    
     
    
NOTES
    
    
        Category: toolkit
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaControllerError 1004
    
    
    Look up a Data ONTAP error code by number.
    
    
    
    Name        : ENOPARTNERIFC
    Code        : 1004
    Description : Partner interface configured does not exist on partner.
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaControllerError EQUOTADIRECTIVE
    
    
    Look up a Data ONTAP error code by name.
    
    
    
    Name        : EQUOTADIRECTIVE
    Code        : 13054
    Description : Cannot process quota directives in /etc/quotas.
