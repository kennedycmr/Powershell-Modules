Cmdlet: Get-NaQtree


NAME
    Get-NaQtree
    
SYNOPSIS
    Returns a list of qtrees.
    
SYNTAX
    Get-NaQtree [[-VolumeName] <String>] [[-QtreeName] <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns a list of qtrees.
    

PARAMETERS
    -VolumeName <String>
        Name of the volume to be queried.  Wildcards are supported.  
        Optionally, this parameter may also take the form /vol/<volume>/<qtree> 
        or /vol/<volume>/; omit the QtreeName parameter if using this form.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -QtreeName <String>
        Name of the qtree to be queried.  Wildcards are supported.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Qtree.QtreeInfo
        
        
    
     
    
NOTES
    
    
        Category: qtree
        API: qtree-list, qtree-list-iter-start, qtree-list-iter-next,
