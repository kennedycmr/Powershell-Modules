Cmdlet: Add-NaNetVlan


NAME
    Add-NaNetVlan
    
SYNOPSIS
    Create a new VLAN interface.
    
SYNTAX
    Add-NaNetVlan [-Interface] <String> -Vlans <UInt16[]> [-Gvrp] [-ActiveOnly] 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Create a new VLAN interface.  Modifies active and persistent config.
    

PARAMETERS
    -Interface <String>
        The interface to which to add a VLAN.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Vlans <UInt16[]>
        The VLAN IDs to add.  Range: [1..4094]
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Gvrp
        If specified, GVRP will be enabled on all added VLANs.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ActiveOnly
        If specified, only the active config is modified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Net.VlanInfo
        
        
    
     
    
NOTES
    
    
        Category: net
        API: net-vlan-create
    
    --------------  Example 1 --------------
    
    C:\PS>Add-NaNetVlan e0c 40 50
    
    
    Add VLAN tags 40 and 50 to interface 'e0c'.
    
    
    
    Interface            ParentInterface       Tag  GVRP
    ---------            ---------------       ---  ----
    e0c-40               e0c                    40  True
    e0c-50               e0c                    50  True
