Cmdlet: Get-NaNfsExport


NAME
    Get-NaNfsExport
    
SYNOPSIS
    Returns the rules associated with exports.
    
SYNTAX
    Get-NaNfsExport [[-Path] <String>] [-Persistent] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns the rules associated with exports, using the new security info 
    structure.  If a pathname is specified, the rules associated with the 
    export matching that pathname, are returned; otherwise, rules for all 
    exports are returned.
    

PARAMETERS
    -Path <String>
        The pathname, for whose matching export, the client wants a listing of 
        the associated rules. If this parameter is provided, the Persistent 
        parameter is ignored.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Persistent
        Default value is false. If specified, the export entries that are 
        present in the /etc/exports file are returned; otherwise, those loaded 
        in memory are returned. This parameter is ignored if the Path parameter 
        is provided.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Nfs.ExportsRuleInfo2
        
        
    
     
    
NOTES
    
    
        Category: nfs
        API: nfs-exportfs-list-rules-2
