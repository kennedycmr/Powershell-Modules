Cmdlet: Set-NaLunDeviceId


NAME
    Set-NaLunDeviceId
    
SYNOPSIS
    Set or clear the SCSI Device Identifier for the LUN.
    
SYNTAX
    Set-NaLunDeviceId [-Path] <String> [[-ID] <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Set or clear the SCSI Device Identifier for the LUN.
    

PARAMETERS
    -Path <String>
        Path of the LUN.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ID <String>
        SCSI Device Identifier.  Omit this parameter to clear the device ID.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Lun.LunInfo
        
        
    
     
    
NOTES
    
    
        Category: lun
        API: lun-set-device-id, lun-unset-device-id
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaLunDeviceId /vol/vol2clone/quorum 103 | Format-List 
    Path,DeviceId
    
    
    Set the SCSI device ID for LUN '/vol/vol2clone/quorum' to 103 and write the 
    result.
    
    
    
    Path     : /vol/vol2clone/quorum
    DeviceId : 103
