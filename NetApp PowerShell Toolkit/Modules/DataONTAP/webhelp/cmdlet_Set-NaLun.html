Cmdlet: Set-NaLun


NAME
    Set-NaLun
    
SYNOPSIS
    Sets the specified LUN online or offline.
    
SYNTAX
    Set-NaLun [-Path] <String> -Online [-Force] [-Controller <NaController>] 
    [<CommonParameters>]
    
    Set-NaLun [-Path] <String> -Offline [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Sets the specified LUN online or offline.
    
    Online:  Re-enables block-protocol accesses to the lun.
    
    Offline:  Disables block-protocol accesses to the LUN. Mappings, if any, 
    configured for the lun are not altered. Note that unless explicitly 
    offlined, a lun is online.
    

PARAMETERS
    -Path <String>
        Path of the LUN.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Online
        Specify -Online to set the LUN online.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force
        Forcibly online the lun, disabling mapping onflict checks with the 
        cluster partner.  If not specified all conflict checks are performed.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Offline
        Specify -Offline to set the LUN offline.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
