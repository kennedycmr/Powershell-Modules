Cmdlet: Update-NaNfsExport


NAME
    Update-NaNfsExport
    
SYNOPSIS
    Loads the etc/exports file into memory.
    
SYNTAX
    Update-NaNfsExport [-PersistentOnly] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Loads the etc/exports file into memory. Replaces exports rules already 
    residing in memory.
    

PARAMETERS
    -PersistentOnly
        Default value is false. If specified, atomically reloads each rule from 
        the exports file and unloads all other rules.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: nfs
        API: nfs-exportfs-load-exports
    
    --------------  Example 1 --------------
    
    C:\PS>Update-NaNfsExport -PersistentOnly
    
    
    Update each export rule from the exports file, unloading all other rules.
