Cmdlet: Remove-NaSnapvaultPriSnapshotSchedule


NAME
    Remove-NaSnapvaultPriSnapshotSchedule
    
SYNOPSIS
    Request the primary system to delete the specified snapshot schedules.
    
SYNTAX
    Remove-NaSnapvaultPriSnapshotSchedule [-VolumeName] <String> 
    [[-ScheduleName] <String>] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Request the primary system to delete the specified snapshot schedules. The 
    snapshot schedules that match the volume name and if specified the schedule 
    name, will be deleted.
    

PARAMETERS
    -VolumeName <String>
        The primary volume for which the schedules are to be deleted.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ScheduleName <String>
        Name of the schedule to be deleted.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

        Category: snapvault
