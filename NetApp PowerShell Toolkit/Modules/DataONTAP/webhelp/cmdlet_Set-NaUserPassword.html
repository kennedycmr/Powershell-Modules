Cmdlet: Set-NaUserPassword


NAME
    Set-NaUserPassword
    
SYNOPSIS
    Changes the password of a specified user.
    
SYNTAX
    Set-NaUserPassword [-User] <String> [-OldPassword] <String> [-NewPassword] 
    <String> [-Controller <NaController>] [<CommonParameters>]
    
    Set-NaUserPassword [-User] <String> [-NewPassword] <String> -ChangeOther 
    [-Controller <NaController>] [<CommonParameters>]
    
    Set-NaUserPassword [-OldCredential <PSCredential>] -NewCredential 
    <PSCredential> [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Changes the password of a specified user.
    
    If the password changes for the user whose credentials are being used to 
    run this cmdlet, the cmdlet attempts to update the credentials for that 
    controller connection.  If subsequent cmdlets fail with an authentication 
    error, call Connect-NaController with the new credentials to ensure the 
    credentials are updated.
    

PARAMETERS
    -User <String>
        The user who's password should be changed.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -OldPassword <String>
        Current password for the user. A user with the capability 
        'security-passwd-change-others' and at least the same capabilities as 
        the user being changed, does not need to enter the current password in 
        order to change it to a new one.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -NewPassword <String>
        New password for the user. Please see documentation for constraints on 
        the password.
        
        Required?                    true
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ChangeOther
        Specify if changing another user's password.  OldPassword will not be 
        required.
        
        Required?                    true
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -OldCredential <PSCredential>
        Old credentials for the user.  The username supplied with the old 
        credential must match the username supplied with the new credential.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -NewCredential <PSCredential>
        New credential for the user.  The username supplied with the old 
        credential must match the username supplied with the new credential.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
