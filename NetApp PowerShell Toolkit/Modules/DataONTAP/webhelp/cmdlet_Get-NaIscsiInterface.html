Cmdlet: Get-NaIscsiInterface


NAME
    Get-NaIscsiInterface
    
SYNOPSIS
    Lists interfaces and their status for iSCSI.
    
SYNTAX
    Get-NaIscsiInterface [-Interfaces <String[]>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Lists interfaces and their status for iSCSI.
    

PARAMETERS
    -Interfaces <String[]>
        Name of interface(s) (e.g. e4) to report; if not supplied, all 
        interfaces are listed.  Wildcards are supported.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Iscsi.IscsiInterfaceListEntryInfo
        
        
    
     
    
NOTES
    
    
        Category: iscsi
        API: iscsi-interface-list-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaIscsiInterface

    e0b                   True         1001 e0b_default
