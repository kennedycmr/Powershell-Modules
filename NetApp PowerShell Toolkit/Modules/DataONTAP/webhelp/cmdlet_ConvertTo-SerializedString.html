Cmdlet: ConvertTo-SerializedString


NAME
    ConvertTo-SerializedString
    
SYNOPSIS
    Convert any .NET object that is marked as Serializable to a Base64-encoded 
    string.
    
SYNTAX
    ConvertTo-SerializedString [-Input] <Object> [<CommonParameters>]
    
    
DESCRIPTION
    Convert any .NET object that is marked as Serializable to a Base64-encoded 
    string.
    
    This cmdlet is the companion of ConvertFrom-SerializedString.
    

PARAMETERS
    -Input <Object>
        The serializable object.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.String
