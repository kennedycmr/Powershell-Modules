Cmdlet: Set-NaNetIpspace


NAME
    Set-NaNetIpspace
    
SYNOPSIS
    Assign a list of interfaces to an ipspace.
    
SYNTAX
    Set-NaNetIpspace [-Name] <String> -Interfaces <String[]> [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Assign a list of interfaces to an ipspace. Modifies active and persistent 
    config.  To remove interfaces from an ipspace, use this cmdlet to assign 
    them back to the default ipspace.
    

PARAMETERS
    -Name <String>
        Name of ipspace to modify.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Interfaces <String[]>
        Interfaces to be assigned to the specified ipspace.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Net.IpspaceConfigInfo
        
        
    
     
    
NOTES
    
    
        Category: net
        API: net-ipspace-assign
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaNetIpspace testspace e0c e0d
    
    
    Assign interfaces 'e0c' and 'e0d' to ipspace 'testspace'.
    
    
    
    Ipspace                   Interfaces
    -------                   ----------
    testspace                 {e0c, e0d}
