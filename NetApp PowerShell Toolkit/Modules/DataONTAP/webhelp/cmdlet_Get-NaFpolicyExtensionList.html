Cmdlet: Get-NaFpolicyExtensionList


NAME
    Get-NaFpolicyExtensionList
    
SYNOPSIS
    Returns information on existing extension sets.
    
SYNTAX
    Get-NaFpolicyExtensionList [-Name] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Returns information on existing extension sets.  Both the include and 
    exclude sets are returned.
    

PARAMETERS
    -Name <String>
        Name of the fpolicy to get extension lists for.  This parameter accepts 
        wildcards.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Fpolicy.ExtensionList
        
        
    
     
    
NOTES
    
    
        Category: fpolicy
        API: fpolicy-extensions-list-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaFpolicyExtensionList policy1
    
    
    Gets the policy extension list for fpolicy policy1.
    
    
    
    Name    : policy1
    Include : {???, ??T, EXE, J??}
    Exclude : {TXT}
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaFpolicyExtensionList test*
    
    
    Gets the fpolicy extension list for all the policies whose names begin with 
    "test".
    
    
    
    Name    : test
    Include : {EXE, P?T, XNN}
    Exclude : {PPT}
    
    Name    : test2
    Include : {???}
    Exclude : {?X?, TXT}
