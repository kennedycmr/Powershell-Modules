Cmdlet: Invoke-NaSnapmirrorAbort


NAME
    Invoke-NaSnapmirrorAbort
    
SYNOPSIS
    Cancels any transfer for a destination.
    
SYNTAX
    Invoke-NaSnapmirrorAbort [-Destination] <String> [-Controller 
    <NaController>] [-ClearCheckpoint] [<CommonParameters>]
    
    
DESCRIPTION
    Cancels any transfer for a destination.  The API can be issued from either 
    the source or destination filer.
    

PARAMETERS
    -Destination <String>
        The destination location of the transfer to abort.  The destination 
        location is of the volume form: <filer>:<volume> or the qtree form: 
        <filer>:/vol/<volume>/<qtree>.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ClearCheckpoint
        If true, the restart checkpoint is cleared.  The default is false, not 
        cleared.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snapmirror.SnapmirrorStatusInfo

        Category: snapmirror
        API: snapmirror-abort
