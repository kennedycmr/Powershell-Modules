Cmdlet: Get-NaAggrSpace


NAME
    Get-NaAggrSpace
    
SYNOPSIS
    Show the space usage of the aggregate on a per flexible volume basis.
    
SYNTAX
    Get-NaAggrSpace [[-Name] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Show the space usage of the aggregate on a per flexible volume basis.
    

PARAMETERS
    -Name <String>
        The aggregate to get the space information for. If not specified the 
        space information for all aggregates is obtained.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Aggr.AggrSpaceInfo
        
        
    
     
    
NOTES
    
    
        Category: aggr
        API: aggr-space-list-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaAggrSpace aggr0
    
    
    Gets space info for aggregate 'aggr0'.

    SizeFree            : 5558628352
    SizeMetadata        : 1510596608

    SizeSnapUsed        : 1110417408
    SizeUsed            : 116351414272
    SizeVolumeAllocated : 116342484992
    SizeVolumeUsed      : 8156282880
    VolumeCount         : 1
    Volumes             : {vol0}
