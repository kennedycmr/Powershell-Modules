Cmdlet: Disable-NaSis


NAME
    Disable-NaSis
    
SYNOPSIS
    Disables SIS operation on a volume.
    
SYNTAX
    Disable-NaSis [-Path] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    Disable-NaSis [-Volume] <VolumeInfo> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Disables SIS operation on a volume. If the SIS operation is active on the 
    volume, it needs to be stopped by "Stop-NaSis" cmdlet before disabling.
    

PARAMETERS
    -Path <String>
        The full path of the SIS volume, /vol/<vol_name>. Only one path can be 
        specified at a time.  The /vol/ prefix may be omitted.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Volume <VolumeInfo>
        Rather than specifying the volume name, this cmdlet alternatively 
        accepts a VolumeInfo object.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Volume.VolumeInfo
