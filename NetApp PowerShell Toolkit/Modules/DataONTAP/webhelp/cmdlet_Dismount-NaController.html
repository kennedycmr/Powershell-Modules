Cmdlet: Dismount-NaController


NAME
    Dismount-NaController
    
SYNOPSIS
    Remove one or more PSDrive objects created by the DataONTAP PowerShell 
    Provider.
    
SYNTAX
    Dismount-NaController [-Scope <String>] [-Force] [<CommonParameters>]
    
    Dismount-NaController [-Name] <String> [-Scope <String>] [-Force] 
    [<CommonParameters>]
    
    Dismount-NaController [-Controller] <NaController> [-Scope <String>] 
    [-Force] [<CommonParameters>]
    
    Dismount-NaController [-Drive] <NaDriveInfo> [-Scope <String>] [-Force] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Remove one or more PSDrive objects created by the DataONTAP PowerShell 
    Provider.
    

PARAMETERS
    -Scope <String>
        The PowerShell scope from which the drive is being removed.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        Force PowerShell to remove the drive even if it is in use, or if the 
        provider throws an exception while trying to remove the drive.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Name <String>
        The Name of the PSDrive object to be removed.  Specify either this or 
        the Controller or Drive parameters, or specify none of them to remove 
        all PSDrives associated with the value in the global variable 
        CurrentNaController.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        Specify the Controller parameter to remove all PSDrives associated with 
        that controller.  If neither Controller nor Name nor Drive is 
        specified, this cmdlet will remove all PSDrives associated with the 
        value in the global variable CurrentNaController.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Drive <NaDriveInfo>
        Specify Drive to remove a specific PSDrive instance.  This parameter is 
        mutually exclusive with the Name and Controller parameters.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: toolkit
    
    --------------  Example 1 --------------
    
    C:\PS>Dismount-NaController dunn
    
    
    Dismount the PSDrive object named 'dunn'.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-PSDrive -PSProvider DataONTAP | Dismount-NaController
    
    
    Remove all PSDrive objects created by the DataONTAP provider.
    
    
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Dismount-NaController
    
    
    Remove all PSDrive objects associated with the NaController value in the 
    global variable CurrentNaController.
