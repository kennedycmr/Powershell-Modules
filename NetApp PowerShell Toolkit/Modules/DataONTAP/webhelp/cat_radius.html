Cmdlet category: Radius

Add-NaRadiusServer
Add a radius server to the radius client service.
Disable-NaRadius
Stop radius client service. Service will no longer be available once the call returns with success.
Enable-NaRadius
Start radius client service. Service will be available once the call returns with success.
Get-NaRadiusServer
Get information about radius servers known to the radius client service.
Get-NaRadiusStatistics
Get radius client statistics.
Remove-NaRadiusServer
Remove a radius server from the radius client service.
Reset-NaRadiusStatistics
Reset radius client counters.
Test-NaRadius
Get status of the radius client service, whether or not it is running.
