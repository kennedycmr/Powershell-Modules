Cmdlet: Get-NaSnapmirror


NAME
    Get-NaSnapmirror
    
SYNOPSIS
    Return the SnapMirror status.
    
SYNTAX
    Get-NaSnapmirror [[-Location] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Return the SnapMirror status.  This API can be issued on either the source 
    or destination filer.
    

PARAMETERS
    -Location <String>
        The source location or destination location of the SnapMirror pair. 
        Possible types are volume or qtree only. If this input is provided, 
        only the SnapMirror relationships with a matching source or destination 
        will be reported. The argument is invalid if the named location doesn't 
        exist.  The argument can also be invalid if it is a flexclone name. (Be 
        aware that the "Get-NaSnapmirrorDestination" API can return flexclone 
        names.) Then this API will return an output value with a "state" of 
        "unknown". If the argument is not specified, all source, destination 
        SnapMirror pairs are returned.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snapmirror.SnapmirrorStatusInfo
        
        
    
     
    
NOTES
    
    
        Category: snapmirror
        API: snapmirror-get-status
