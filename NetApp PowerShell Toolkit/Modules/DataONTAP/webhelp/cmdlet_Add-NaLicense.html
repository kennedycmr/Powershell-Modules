Cmdlet: Add-NaLicense


NAME
    Add-NaLicense
    
SYNOPSIS
    Enable license for a Data ONTAP service.
    
SYNTAX
    Add-NaLicense [-Codes] <String[]> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Enable license for a Data ONTAP service.
    
    Prior to Data ONTAP 8.2, the added license will be returned if the license 
    is successfully enabled.  In Data ONTAP 8.2 and up, if no error is written, 
    the license has been added successfully.
    

PARAMETERS
    -Codes <String[]>
        One or more license codes of a Data ONTAP service to be enabled.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.License.LicenseInfo
