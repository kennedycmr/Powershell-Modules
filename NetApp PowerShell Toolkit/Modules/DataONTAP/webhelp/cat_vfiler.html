Cmdlet category: Vfiler

Complete-NaVfilerMigrate
Complete the migration by stopping the remote vfiler and activating the vfiler on the local machine. This can be executed only from the pfiler. The source vfiler unit will be destroyed after migration is complete.
Get-NaVfiler
Retrieve information about all vfilers.
Get-NaVfilerDr
Get the status of the disaster recovery relationship with the given remote vfiler and filer.
Get-NaVfilerMigrate
Get the status of the vfiler migrate relationship with the given remote vfiler and filer. Once the status is "snapmirrored", the Complete-NaVfilerMigrate cmdlet can be run to complete the migration.
Get-NaVfilerProtocol
Get the protocols (allowed and disallowed) for a vfiler.
Get-NaVfilerStatus
Get the status of a vfiler.
Invoke-NaVfilerDr
Stop the remote vfiler and activate the disaster recovery vfiler on the current controller.
Invoke-NaVfilerDrResync
Move a vfiler which has been migrated by vfiler disaster recovery back to its original controller.
New-NaVfiler
Create a new vfiler.
New-NaVfilerDr
Create a new vfiler disaster recovery relationship. The inputs specify the information for the remote vfiler/filer to connect to and configuration information for the vfiler to use upon activation.
Remove-NaVfiler
Destroy a vfiler.
Remove-NaVfilerDr
Delete the vfiler disaster recovery relationship on this controller.
Set-NaVfilerAddress
Manage vfiler addresses.
Set-NaVfilerAdminHost
Set the administrative host for a vfiler.
Set-NaVfilerDns
Set DNS info for a vfiler.
Set-NaVfilerNis
Set NIS info for a vfiler.
Set-NaVfilerPassword
Set the root password for a vfiler.
Set-NaVfilerProtocol
Allow / disallow protocols on a vfiler.
Set-NaVfilerStorage
Add or remove storage units from a vfiler.
Start-NaVfiler
Start a stopped vfiler.
Start-NaVfilerMigrate
Starts vfiler unit migration from remote filer to this filer. The data and remote vfiler configuration are migrated as a vfiler unit.
Stop-NaVfiler
Stop a started vfiler.
Stop-NaVfilerMigrate
Stop the vfiler migration.
