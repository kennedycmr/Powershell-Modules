Cmdlet: Clear-NaDiskOwner


NAME
    Clear-NaDiskOwner
    
SYNOPSIS
    Removes ownership information on a disk.
    
SYNTAX
    Clear-NaDiskOwner -Name <String[]> [-OwnerID <UInt32>] [-All] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Removes ownership information on a disk. Default usage is to remove 
    ownership information for all disks owned by the local node, in maintenance 
    mode.
    

PARAMETERS
    -Name <String[]>
        
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -OwnerID <UInt32>
        Remove ownership of all disks owned by this owner-id (maintenance mode 
        only). Range [0..2^32-1
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -All
        If 'true' remove ownership information from all disks. (maintenance 
        mode only)
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Disk.DiskSanownDetailInfo

        API: disk-sanown-remove-ownership
