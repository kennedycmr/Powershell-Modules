Cmdlet: Set-NaQuota


NAME
    Set-NaQuota
    
SYNOPSIS
    Sets a quota entry.
    
SYNTAX
    Set-NaQuota [-Type] <String> [-Target] <String[]> [-Volume] <String> 
    [[-Qtree] <String>] [-DiskLimit <String>] [-FileLimit <String>] [-Threshold 
    <String>] [-SoftDiskLimit <String>] [-SoftFileLimit <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    Set-NaQuota [-Volume] <String> [[-Qtree] <String>] -UserName <String> 
    [-DiskLimit <String>] [-FileLimit <String>] [-Threshold <String>] 
    [-SoftDiskLimit <String>] [-SoftFileLimit <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    Set-NaQuota [-Volume] <String> [[-Qtree] <String>] -GroupId <String> 
    [-DiskLimit <String>] [-FileLimit <String>] [-Threshold <String>] 
    [-SoftDiskLimit <String>] [-SoftFileLimit <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    Set-NaQuota -Path <String> [-DiskLimit <String>] [-FileLimit <String>] 
    [-Threshold <String>] [-SoftDiskLimit <String>] [-SoftFileLimit <String>] 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Sets a quota entry.  If the type, target, volume, and tree do not exist, a 
    new entry is created.  If the type, target, volume, and tree exist, then 
    the entry is modified.
    

PARAMETERS
    -Type <String>
        This is the quota type: "user", "group", or "tree".
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Target <String[]>
        This is the quota target of the type specified.  The target can be of 
        the form: <name>, <number>, or <path name>.  Multiple targets can be 
        specified by a comma-separated list.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Volume <String>
        This is the volume name that the quota resides on.
        
        Required?                    true
        Position?                    3
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Qtree <String>
        This is the qtree name that the quota resides on.  It can be the qtree 
        name or omitted if no qtree.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -DiskLimit <String>
        This is the amount of disk space that is reserved for the the target.  
        The value is expressed in bytes.  Omit or set the value to "-" if the 
        limit is to be unlimited.  Default is unlimited.
        
        The format to use is: 
        
        <number>[k|m|g|t] 
        
        where "k" means kilobytes, "m" means megabytes, "g" means gigabytes, 
        and "t" means terabytes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -FileLimit <String>
        This is the number of files that the target can have.  Omit or set the 
        value to "-" if the limit is to be unlimited.  Default is unlimited.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Threshold <String>
        This is the amount of disk space the target would have to exceed before 
        a message is logged.  The value is expressed in bytes.  Omit or set the 
        value to "-" if the limit is to be unlimited.  Default is unlimited.
        
        The format to use is: 
        
        <number>[k|m|g|t] 
        
        where "k" means kilobytes, "m" means megabytes, "g" means gigabytes, 
        and "t" means terabytes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -SoftDiskLimit <String>
        This is the amount of disk space the target would have to exceed before 
        a message is logged and an SNMP trap is generated.  The value is 
        expressed in bytes.  Omit or set the value to "-" if the limit is to be 
        unlimited.  Default is unlimited.
        
        The format to use is: 
        
        <number>[k|m|g|t] 
        
        where "k" means kilobytes, "m" means megabytes, "g" means gigabytes, 
        and "t" means terabytes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -SoftFileLimit <String>
        This is the number of files the target would have to exceed before a 
        message is logged and an SNMP trap is generated.  Omit or set the value 
        to "-" if the limit is to be unlimited.  Default is unlimited.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -UserName <String>
        A user can be specfied as a: unix user name, numerical user ID, windows 
        account name, Windows SID or a comma separated list of multiple users.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -GroupId <String>
        May be one of the following: Unix group name, numerical group id (If 
        the group id is 0, no limits set will be enforced, but usage will still 
        be tracked), or the pathname of a file owned by that group (specifying 
        a file or directory will not affect the quota on that file or 
        directory, only the GID).
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Path <String>
        A complete path to an existing qtree.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Quota.QuotaEntryInfo
        
        
    
     
    
NOTES

    C:\PS>Set-NaQuota tree * vol0 -FileLimit 50000 -DiskLimit 20g

    FileLimit          : 50000
    PerformUserMapping : False

    SoftDiskLimit      : -
    SoftFileLimit      : -
    Threshold          : -

    FileLimit          : -
