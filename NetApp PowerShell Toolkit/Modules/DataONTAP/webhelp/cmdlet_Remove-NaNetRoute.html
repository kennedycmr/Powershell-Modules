Cmdlet: Remove-NaNetRoute


NAME
    Remove-NaNetRoute
    
SYNOPSIS
    Delete a kernel route.
    
SYNTAX
    Remove-NaNetRoute [-Destination] <String> [-Host] [-Ipspace <String>] 
    [-Creator <String>] [-ActiveOnly] [-Controller <NaController>] 
    [<CommonParameters>]
    
    Remove-NaNetRoute [-Destination] <String> -IPv6 [-Host] [-PrefixLength 
    <UInt16>] [-Ipspace <String>] [-Creator <String>] [-ActiveOnly] 
    [-Controller <NaController>] [<CommonParameters>]
    
    Remove-NaNetRoute [-Host] [-Ipspace <String>] [-Creator <String>] -Route 
    <RouteInfo> [-ActiveOnly] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Delete a kernel route.  Modifies active and persistent config.
    

PARAMETERS
    -Destination <String>
        Destination of route.  Possible values: An IP address, hostname, or 
        "default".
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Host
        Specify if deleting a route to a specific host.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Ipspace <String>
        The ipspace of the route entry.  If not specified, the default value is 
        "default-ipspace".
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Creator <String>
        The vfiler that owns the route to be deleted.  If not specified, the 
        default is vfiler0 (i.e. the physical controller).
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -ActiveOnly
        If specified, only the active config is modified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IPv6
        Specify if deleting an IPv6 route.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -PrefixLength <UInt16>
        Prefix length (netmask) for destination.  Specify only for IPv6 routes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Route <RouteInfo>
        A route object as returned by Get-NaNetRoute.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: net
        API: net-route-delete
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaNetRoute 10.61.165.80 | Remove-NaNetRoute
    
    
    Remove a specific route.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaNetRoute | Remove-NaNetRoute
    
    
    Remove all routes.
