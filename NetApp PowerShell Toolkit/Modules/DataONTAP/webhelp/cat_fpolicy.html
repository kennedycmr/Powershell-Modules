Cmdlet category: Fpolicy

Add-NaFpolicyExtension
Adds to the list of extensions in exclude or include set.
Disable-NaFpolicy
Sets options fpolicy enable to off. If a name is given, disables a specific named policy.
Enable-NaFpolicy
Sets options fpolicy enable to on. If a name is given, enables a specific named policy. The operation will fail if the policy doesn't exist.
Get-NaFpolicy
Returns a list of existing policies.
Get-NaFpolicyExtensionList
Returns information on existing extension sets.
Get-NaFpolicyOption
Shows value of policy options.
Get-NaFpolicySecondaryServer
Gets a list of secondary servers for this fpolicy.
Get-NaFpolicyServer
Shows a list of primary servers serving the policy.
Get-NaFpolicyVolumeList
Returns a volume-regular-expression list for an exclude or include set.
New-NaFpolicy
Creates a new policy.
Remove-NaFpolicy
Destroys existing policy.
Remove-NaFpolicyExtension
Removes a list of extensions from the exclude or include set.
Reset-NaFpolicyExtensionList
Resets the list of extensions in the exclude or include set.
Set-NaFpolicy
Manipulate a list of operations and network protocols for a policy.
Set-NaFpolicyExtensionList
Sets the list of extensions in the exclude or include set.
Set-NaFpolicyOption
Sets policy's options.
Set-NaFpolicySecondaryServer
Sets secondary servers information in a form of a list of ip addresses.
Set-NaFpolicyVolumeList
Manipulate a list of volumes in an exclude or include set.
Stop-NaFpolicyServer
Stops specific primary server serving the policy.
Test-NaFpolicy
Return the status of fpolicy. Returns true if fpolicy is enabled, false if it is not.
