Cmdlet: Set-NaVfilerStorage


NAME
    Set-NaVfilerStorage
    
SYNOPSIS
    Add or remove storage units from a vfiler.
    
SYNTAX
    Set-NaVfilerStorage [-Name] <String> [-AddStorage <String[]>] 
    [-RemoveStorage <String[]>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Add or remove storage units from a vfiler.
    

PARAMETERS
    -Name <String>
        The name of the vfiler.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -AddStorage <String[]>
        List of volume or qtree storage paths to be added, in the form 
        "/vol/<volume>/...".  The "/vol/" prefix is added automatically if not 
        specified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RemoveStorage <String[]>
        List of volume or qtree storage paths to be removed, in the form 
        "/vol/<volume>/...".  The "/vol/" prefix is added automatically if not 
        specified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Vfiler.VfilerInfo
        
        
    
     
    
NOTES
    
    
        Category: vfiler
        API: vfiler-add-storage, vfiler-remove-storage
    
    --------------  Example 1 --------------
    
    C:\PS>Set-NaVfilerStorage vfiler2 -AddStorage testvol3
    
    
    Add volume 'testvol3' to vfiler 'vfiler2'.
