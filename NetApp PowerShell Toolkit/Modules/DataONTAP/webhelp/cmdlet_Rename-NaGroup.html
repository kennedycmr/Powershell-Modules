Cmdlet: Rename-NaGroup


NAME
    Rename-NaGroup
    
SYNOPSIS
    Renames a group.
    
SYNTAX
    Rename-NaGroup [-Group] <String> [-NewName] <String> [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Renames a group.
    

PARAMETERS
    -Group <String>
        The group to rename.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -NewName <String>
        The new group name.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Useradmin.UseradminGroupInfo
        
        
    
     
    
NOTES
    
    
        Category: useradmin
        API: useradmin-group-modify
    
    --------------  Example 1 --------------
    
    C:\PS>Rename-NaGroup HyperVAdmins VirtAdmins
    
    
    Change name of group 'HyperVAdmins' to 'VirtAdmins'.

    Comment             : HyperV VM administrators
    UseradminRoles      : {power, audit}
    AllowedCapabilities :
    Rid                 : 131085
