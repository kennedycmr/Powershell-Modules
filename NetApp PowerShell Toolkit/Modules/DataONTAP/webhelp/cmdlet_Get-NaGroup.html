Cmdlet: Get-NaGroup


NAME
    Get-NaGroup
    
SYNOPSIS
    Lists full information for all groups on the system.
    
SYNTAX
    Get-NaGroup [[-Group] <String>] [-GetCapabilities] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Lists full information for all groups on the system.
    

PARAMETERS
    -Group <String>
        List only the information associated with this group.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -GetCapabilities
        Default is false. If set to true, then the allowed capabilities are 
        placed into the group-info structure. Depending on number of groups and 
        roles, this operation may take a long time.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Useradmin.UseradminGroupInfo

        API: useradmin-group-list
