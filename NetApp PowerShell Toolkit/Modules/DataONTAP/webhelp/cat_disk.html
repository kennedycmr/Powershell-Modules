Cmdlet category: Disk

Clear-NaDiskOwner
Removes ownership information on a disk.
Get-NaDisk
Get disk status information from RAID.
Get-NaDiskOwner
Get sanown disk information.
Get-NaDiskOwnerFiler
Get sanown filer information.
Remove-NaDisk
Remove a spare disk.
Set-NaDiskFail
Fail a file system disk.
Set-NaDiskOwner
Changes ownership on a disk.
Set-NaDiskUnfail
Unfail a disk in the broken pool.
Start-NaDiskReplace
Initiate replacing a file system disk with an appropriate spare disk.
Start-NaDiskUpdate
Start disk firmware download process to update firmware on disks.
Start-NaDiskZeroSpare
Set up all non-zeroed spares owned by the filer to start zeroing.
Stop-NaDiskReplace
Abort disk replace.
Switch-NaDiskOwner
Changes ownership on disks already belonging to an owner.
