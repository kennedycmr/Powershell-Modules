Cmdlet: Restore-NaHostFile


NAME
    Restore-NaHostFile
    
SYNOPSIS
    Restore a single file from a snapshot.
    
SYNTAX
    Restore-NaHostFile [-Path] <String> [-Snapshot] <String> [-MountPoint 
    <String>] [-Overwrite] [-Controller <AbstractController>] [-SkipUnmap] 
    [-UnmapSize <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Restore a single file from a snapshot.  The file is cloned from a temporary 
    LUN backed by the given snapshot.
    
    This cmdlet requires Data ONTAP 8.1 or later.
    

PARAMETERS
    -Path <String>
        Full path to the file to restore.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Snapshot <String>
        Name of the snapshot to use to restore the file.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -MountPoint <String>
        Mount point to give to the temporary LUN.  If the LUN is not assigned a 
        drive letter by the host, this mount point is used.  If not provided 
        and a drive letter is not automatically assigned by the host, the 
        cmdlet will fail.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Overwrite
        If provided, the original file will be overwritten.  If not provided, a 
        backup of the original file will be kept with the extension ".bkup".
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <AbstractController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -SkipUnmap
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -UnmapSize <String>
        
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.IO.FileInfo
        
        
    
     
    
NOTES
    
    
        Category: host

    C:\PS>Restore-NaHostFile K:\vm1.vhd snapshot1
