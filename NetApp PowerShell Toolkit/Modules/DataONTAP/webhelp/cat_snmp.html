Cmdlet category: Snmp

Add-NaSnmpCommunity
Adds a community to the list of communities.
Add-NaSnmpTrapHost
Adds a host to the list of trap hosts.
Disable-NaSnmp
Disables SNMP protocol.
Disable-NaSnmpTrap
Disables SNMP traps.
Enable-NaSnmp
Enables SNMP protocol.
Enable-NaSnmpTrap
Enables SNMP traps.
Get-NaSnmp
Returns configuration information of the SNMP agent daemon.
Invoke-NaSnmpGet
Retrieves the value of a snmp object.
Invoke-NaSnmpGetNext
This is used to retrieve the next OID in the mib tree of data.
Invoke-NaSnmpWalk
This cmdlet walks the SNMP MIB starting from the given OID.
Remove-NaSnmpCommunity
Deletes a community from the list of communities.
Remove-NaSnmpTrapHost
Deletes a host from the list of trap hosts.
Set-NaSnmpContact
Sets the SNMP contact name.
Set-NaSnmpLocation
Sets the SNMP location.
