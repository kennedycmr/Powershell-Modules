Cmdlet: Remove-NaHostFile


NAME
    Remove-NaHostFile
    
SYNOPSIS
    Delete a file, completely reclaiming all of the space it consumed on the 
    storage controller.
    
SYNTAX
    Remove-NaHostFile [-FullName] <String> [-SkipUnmap] [-UnmapSize <String>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Delete a file, completely reclaiming all of the space it consumed on the 
    storage controller.
    
    The file to be deleted must be specified with a fully-qualified path that 
    resides on a Data ONTAP LUN or Data ONTAP CIFS share.  Cluster-shared 
    volumes (CSVs) are supported.  Administrator privileges may be required to 
    use this cmdlet.
