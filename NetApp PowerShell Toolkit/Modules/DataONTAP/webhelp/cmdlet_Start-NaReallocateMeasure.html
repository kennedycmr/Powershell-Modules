Cmdlet: Start-NaReallocateMeasure


NAME
    Start-NaReallocateMeasure
    
SYNOPSIS
    Starts a new measurement-only reallocate job.
    
SYNTAX
    Start-NaReallocateMeasure [-Path] <String> [-Threshold <Int32>] [-Interval 
    <String>] [-RunOnce] [-LogFile <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Starts a new measurement-only reallocate job.
    

PARAMETERS
    -Path <String>
        Path to large file, LUN or volume to measure optimization on, starting 
        with /vol/.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Threshold <Int32>
        The allocation threshold before path is regarded as needing 
        optimization.  A larger value indicates the file or LUN can be more 
        fragmented.  Range: [3..10]
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Interval <String>
        The interval between repeated scans.  The format is numX, where X is m, 
        h or d, indicating minutes, hours or days.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -RunOnce
        If true, run the scan once then exit.  Interval and RunOnce are 
        mutually exclusive
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -LogFile <String>
        Path to a full name on the appliance where information about the most 
        recent measurement made will be logged, such as 
        "/vol/logvol/measurelog.txt".
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Reallocate.ReallocateJobInfo
        
        
    
     
    
NOTES
    
    
        Category: reallocate
        API: reallocate-measure
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaLun /vol/opalis/w2k3OpalisVM | Start-NaReallocateMeasure 
    -Threshold 3 -RunOnce -LogFile /vol/vol0/reallocLog.txt
    
    
    Begin the measure-only reallocate job on LUN 'w2k3OpalisVM'.

    Interval           : n/a
    IntervalTS         :

    MeasureOnly        : True
    Path               : /vol/opalis/w2k3OpalisVM
    ScanDetail         :
    Schedule           : n/a
    State              : Checking
    Threshold          : 1
    ThresholdSpecified : True
