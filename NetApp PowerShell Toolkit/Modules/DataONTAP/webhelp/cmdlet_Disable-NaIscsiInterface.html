Cmdlet: Disable-NaIscsiInterface


NAME
    Disable-NaIscsiInterface
    
SYNOPSIS
    Disables one or more interfaces for use by iSCSI.
    
SYNTAX
    Disable-NaIscsiInterface -Interfaces <String[]> [-Controller 
    <NaController>] [<CommonParameters>]
    
    Disable-NaIscsiInterface -Name <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Disables one or more interfaces for use by iSCSI.
    

PARAMETERS
    -Interfaces <String[]>
        A list of interfaces disable for iSCSI.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Name <String>
        
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Iscsi.IscsiInterfaceListEntryInfo
        
        
    
     
    
NOTES
    
    
        Category: iscsi
        API: iscsi-interface-disable
    
    --------------  Example 1 --------------
    
    C:\PS>Disable-NaIscsiInterface e0b
    
    
    Disallow iSCSI traffic on interface e0b.
    
    
    
    Interface            Enabled TpgroupTag TpgroupName
    ---------            ------- ---------- -----------
    e0b                   False        1001 e0b_default
