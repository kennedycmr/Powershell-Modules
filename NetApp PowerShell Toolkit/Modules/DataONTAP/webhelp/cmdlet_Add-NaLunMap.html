Cmdlet: Add-NaLunMap


NAME
    Add-NaLunMap
    
SYNOPSIS
    Maps the LUN to all the initiators in the specified initiator group.
    
SYNTAX
    Add-NaLunMap [-Path] <String> [-InitiatorGroup] <String> [-ID <Int32>] 
    [-Force] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Maps the LUN to all the initiators in the specified initiator group.
    

PARAMETERS
    -Path <String>
        Path of the LUN.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -InitiatorGroup <String>
        Initiator group to map to the given LUN.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -ID <Int32>
        If the lun ID is not specified, the smallest number that can be used 
        for the various initiators in the group is automatically picked.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Force
        Forcibly map the lun, disabling mapping conflict checks with the 
        cluster partner.  If not specified all conflict checks are performed.
        
        Required?                    false
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
