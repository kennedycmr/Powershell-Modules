Cmdlet: Get-NaLunTargetDeviceId


NAME
    Get-NaLunTargetDeviceId
    
SYNOPSIS
    Returns the SCSI Target Device Id.
    
SYNTAX
    Get-NaLunTargetDeviceId [-SerialNumber] <String> [-Protocol] <String> 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns the SCSI Target Device Id.
    

PARAMETERS
    -SerialNumber <String>
        Serial number of the lun.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Protocol <String>
        Transport protocol through which the lun will be accessed. Possible 
        values are "fcp" or "iscsi".
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.String
        
        
    
     
    
NOTES
    
    
        Category: lun
        API: lun-get-target-device-id
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaLunTargetDeviceId -SerialNumber P4DahZb7hUCv -Protocol iscsi
    
    
    Get the iscsi LUN target device id for the LUN with serial number 
    P4DahZb7hUCv.
