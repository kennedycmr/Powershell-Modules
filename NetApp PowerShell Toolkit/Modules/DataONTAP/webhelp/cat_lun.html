Cmdlet category: Lun

Add-NaLunMap
Maps the LUN to all the initiators in the specified initiator group.
Clear-NaLunPersistentReservation
Clear the persistent reservation information for a given LUN.
Clear-NaLunStatistics
Resets (zeroes) block-protocol access statistics for LUN(s).
Confirm-NaLunHasScsiReservation
Queries for all types of scsi reservations covering both iSCSI and FCP.
Confirm-NaLunInitiatorLoggedIn
Determine if an initiator is logged in via fcp or iSCSI
Confirm-NaLunRestore
Get state of LUN restore.
Get-NaLun
Get the status of the given LUN, or all LUNs.
Get-NaLunAttribute
Get a named attribute for a given LUN.
Get-NaLunClone
Get all LUN clones contained in a given snapshot.
Get-NaLunCloneSplit
Get the cloning status of LUN(s).
Get-NaLunComment
Get the optional descriptive comment for a LUN.
Get-NaLunGeometry
Get SCSI disk geometry for a given LUN.
Get-NaLunInquiry
Get the SCSI INQUIRY response data for vendor id (vid), product id (pid), and firmware revision (rev) based on the igroup that the lun in question is mapped to.
Get-NaLunMap
Returns a list of initiator groups and their members (the initiators) mapped to the given LUN.
Get-NaLunMapByInitiator
List all the luns mapped to an initiator
Get-NaLunMaxSize
Returns the maximum possible size in bytes of a lun on a given volume or qtree.
Get-NaLunMinSize
Returns the minimum possible size in bytes.
Get-NaLunOccupiedSize
Get the size occupied by the LUN in the active FS.
Get-NaLunPersistentReservation
Get the persistent reservation information for a given LUN.
Get-NaLunSelect
Get the select attribute for the specified LUN.
Get-NaLunSerialNumber
Get the serial number for the specified LUN.
Get-NaLunSignature
Get the MBR or GPT signature of a partitioned LUN.
Get-NaLunSnapUsage
Lists all LUNs backed by data in the specified snapshot.
Get-NaLunSpaceReserved
Queries the space reservation settings for the named LUN.
Get-NaLunStatistics
Get block-protocol access statistics (in bytes) for LUN(s).
Get-NaLunTargetDeviceId
Returns the SCSI Target Device Id.
New-NaLun
Create a lun by one of three methods: by size, from file, or from snapshot.
New-NaLunClone
Create a LUN clone.
Remove-NaLun
Destroy the specified LUN.
Remove-NaLunMap
Reverses the effect of Add-NaLunMap on the specified LUN for the specified group.
Rename-NaLun
Move (rename) a LUN.
Set-NaLun
Sets the specified LUN online or offline.
Set-NaLunAttribute
Set a named attribute for a given LUN.
Set-NaLunComment
Set the optional descriptive comment for a LUN.
Set-NaLunDeviceId
Set or clear the SCSI Device Identifier for the LUN.
Set-NaLunSelect
Set the select attribute for the specified LUN.
Set-NaLunSerialNumber
Set the serial number for the specified LUN.
Set-NaLunShare
Enables file system protocol-based access to a lun.
Set-NaLunSignature
Set the MBR or GPT signature of a partitioned LUN.
Set-NaLunSize
Changes the size of the lun.
Set-NaLunSpaceReserved
Sets the space reservation settings for the named LUN.
Start-NaLunCloneSplit
Start the cloning of the given LUN.
Stop-NaLunCloneSplit
Stop the cloning of the given LUN.
Test-NaLunConfig
Returns a list of configuration warnings for various use cases.
