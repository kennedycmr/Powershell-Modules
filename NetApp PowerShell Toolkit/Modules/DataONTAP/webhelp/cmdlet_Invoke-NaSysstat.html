Cmdlet: Invoke-NaSysstat


NAME
    Invoke-NaSysstat
    
SYNOPSIS
    PowerShell analogue of Data ONTAP's sysstat CLI command.
    
SYNTAX
    Invoke-NaSysstat [-SampleIntervalSeconds <Double>] [-Count <Int32>] 
    [-Hashtable] [-Controller <NaController>] [<CommonParameters>]
    
    Invoke-NaSysstat -Cifs [-SampleIntervalSeconds <Double>] [-Count <Int32>] 
    [-Hashtable] [-Controller <NaController>] [<CommonParameters>]
    
    Invoke-NaSysstat -Disk <String[]> [-SampleIntervalSeconds <Double>] [-Count 
    <Int32>] [-Hashtable] [-Controller <NaController>] [<CommonParameters>]
    
    Invoke-NaSysstat -Fcp [-SampleIntervalSeconds <Double>] [-Count <Int32>]

    [-Controller <NaController>] [<CommonParameters>]

DESCRIPTION
    PowerShell analogue of Data ONTAP's sysstat CLI command.
    
    This cmdlet uses the performance APIs to deliver a running tabular display 
    of real-time performance metrics.  The values presented are as similar as 
    possible to those provided by sysstat within the bounds of the performance 
    APIs.
    
    Unlike sysstat, Invoke-NaSysstat supports not only top-level controller 
    metrics but also performance values for other object classes such as 
    volumes, vfilers, LUNs, network interfaces, storage adapters, individual 
    data protocols, and WAFL.
    
    In some cases, there are more data fields than fit in the tabular views; 
    use "Format-List *" to see everything.
    
    In the unformatted output, data sizes are presented in bytes.  Data rates 
    are presented in bytes per second.  Latency values are presented in 
    milliseconds.
