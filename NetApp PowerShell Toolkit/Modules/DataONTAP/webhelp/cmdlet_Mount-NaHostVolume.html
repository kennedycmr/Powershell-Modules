Cmdlet: Mount-NaHostVolume


NAME
    Mount-NaHostVolume
    
SYNOPSIS
    Add a drive letter or empty folder mount point to an existing Windows 
    volume.
    
SYNTAX
    Mount-NaHostVolume [[-Path] <String>] [-MountPoint] <String> 
    [<CommonParameters>]
    
    Mount-NaHostVolume [-VdsId] <Guid> [-MountPoint] <String> 
    [<CommonParameters>]
    
    
DESCRIPTION
    Add a drive letter or empty folder mount point to an existing Windows 
    volume.
    
    This cmdlet must be run with administrative privileges.
    

PARAMETERS
    -Path <String>
        The mount point (i.e. G:, 
        \\?\Volume{12345678-abcd-1234-abcd-1234567890ab}\) or a path to any 
        file on the volume to process
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -MountPoint <String>
        A drive letter (i.e. 'G') or the path to an empty folder to mount the 
        volume
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -VdsId <Guid>
        The Guid object which uniquely identifies the volume in VDS.  Note that 
        this is not the same as the volume guid path.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.PowerShell.SDK.Cmdlets.Toolkit.Windows.HostVolume
        
        
    
     
    
NOTES
    
    
        Category: host
    
    --------------  Example 1 --------------
    
    C:\PS>Mount-NaHostVolume E:\ -MountPoint C:\NewDir
    
    
    Add a folder mount point to the volume currently mounted at E:\.
    
    
    
    MountPoints                    Disk       Size GuidPaths
    -----------                    ----       ---- ---------
    {E:\, C:\NewDir\}                 4    10.0 GB 
    {\\?\Volume{20a7fedf-f8e4-11e0-949b-00155da1ce00}\}
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Dismount-NaHostVolume Q:\ | Mount-NaHostVolume -MountPoint E:\
    
    
    Change the drive letter of a volume.
    
    
    
    MountPoints                    Disk       Size GuidPaths
    -----------                    ----       ---- ---------
    {E:\}                             4    10.0 GB 
    {\\?\Volume{20a7fedf-f8e4-11e0-949b-00155da1ce00}\}
