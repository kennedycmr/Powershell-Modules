Cmdlet: Remove-NaNetVlan


NAME
    Remove-NaNetVlan
    
SYNOPSIS
    Delete a VLAN interface.
    
SYNTAX
    Remove-NaNetVlan -VlanInfo <VlanInfo> [-ActiveOnly] [-Controller 
    <NaController>] [<CommonParameters>]
    
    Remove-NaNetVlan [-Interface] <String> -Vlans <UInt16[]> [-ActiveOnly] 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Delete a VLAN interface.  Modifies active and persistent config.
    

PARAMETERS
    -VlanInfo <VlanInfo>
        A VlanInfo object, as returned by Get-NaNetVlan.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -ActiveOnly
        If specified, only the active config is modified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Interface <String>
        The interface that is hosting the VLAN interfaces to be deleted.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Vlans <UInt16[]>
        The VLAN IDs to delete.  Range: [1..4094]
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

NOTES
    
    
        Category: net
        API: net-vlan-delete
    
    --------------  Example 1 --------------
    
    C:\PS>Remove-NaNetVlan e0c 20 30 40
    
    
    Remove VLAN tags 20, 30 and 40 from interface 'e0c'.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaNetVlan e0c-* | Remove-NaNetVlan
    
    
    Remove all VLANs from interface 'e0c'.
