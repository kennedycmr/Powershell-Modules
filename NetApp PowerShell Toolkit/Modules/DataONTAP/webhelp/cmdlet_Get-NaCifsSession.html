Cmdlet: Get-NaCifsSession


NAME
    Get-NaCifsSession
    
SYNOPSIS
    Gives information on current CIFS activity.
    
SYNTAX
    Get-NaCifsSession [-User <String>] [-Host <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Gives information on current CIFS activity. Without arguments, it returns a 
    summary of information about the filer and lists the users who are 
    connected to the filer.
    

PARAMETERS
    -User <String>
        User name. If user is specified , this api returns information about 
        the specified user, along with the names and access level of files that 
        the user has opened. This api returns information about all the users, 
        if the user is specified as '*' or is not specified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Host <String>
        IP address/machine name of the user's client machine.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Cifs.CifsSessionInfo

        Category: cifs
