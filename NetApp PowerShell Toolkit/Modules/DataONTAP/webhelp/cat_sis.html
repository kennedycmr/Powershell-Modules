Cmdlet category: Sis

Disable-NaSis
Disables SIS operation on a volume.
Enable-NaSis
Enables SIS on a volume.
Get-NaSis
Get the dedupe status of an SIS volume.
Set-NaSis
Setup or modify the SIS operation schedule or options for a SIS volume.
Start-NaSis
Start SIS operation on a SIS volume.
Stop-NaSis
Abort currently active SIS operation on the volume.
