Cmdlet: New-NaVfilerDr


NAME
    New-NaVfilerDr
    
SYNOPSIS
    Create a new vfiler disaster recovery relationship.  The inputs specify the 
    information for the remote vfiler/filer to connect to and configuration 
    information for the vfiler to use upon activation.
    
SYNTAX
    New-NaVfilerDr [-RemoteVfiler] <String> [-RemoteFiler] <String> [-NisServer 
    <String[]>] [-DnsServer <String[]>] -IpAddressInfo <IpaddrInfo[]> 
    [-Credential <PSCredential>] [-Synchronous] [-Secure] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Create a new vfiler disaster recovery relationship.  The inputs specify the 
    information for the remote vfiler/filer to connect to and configuration 
    information for the vfiler to use upon activation.
    

PARAMETERS
    -RemoteVfiler <String>
        Name of the remote vfiler.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RemoteFiler <String>
        Name of the remote filer.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -NisServer <String[]>
        Ip addresses of the NIS servers the vfiler should use.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DnsServer <String[]>
        Ip addresses of the DNS servers the vfiler should use.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -IpAddressInfo <IpaddrInfo[]>
        A list of all the IP addresses and associated information which should 
        be bound to the vfiler at the destination.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Credential <PSCredential>
        Login name and password to use on the remote filer.  If not specified, 
        the credential cache will be consulted.  If credentials and not 
        specified and not found in the credentials cache, the cmdlet will fail.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Synchronous
        When specified, synchronous snapmirror is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Secure
        When specified, a secure command channel is used while communicating 
        with the remote filer.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false

NOTES
    
    
        Category: vfiler
        API: vfiler-dr-configure
    
    --------------  Example 1 --------------
    
    C:\PS>$ips = @()
    $ips += New-Object DataONTAP.Types.Vfiler.IpaddrInfo
    $ips[0].Interface = "e0a"
    $ips[0].IpAddress = "10.61.167.9"
    $ips[0].Netmask = "255.255.255.0"
    New-NaVfilerDr -RemoteVfiler vfiler3 -RemoteFiler dunn -IpAddressInfo $ips 
    -Credential root
    
    
    Create a new vfiler disaster recovery relationship with vfiler3 on dunn.  A 
    new IpaddrInfo object is first created and filled out to be used with 
    New-NaVfilerDr.

    VfstoreCount AdminHost

            1
