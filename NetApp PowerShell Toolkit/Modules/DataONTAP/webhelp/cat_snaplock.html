Cmdlet category: Snaplock

Get-NaSnaplockComplianceClock
Get the SnapLock System Compliance Clock date and time.
Get-NaSnaplockLog
Get the status of the active WORM log files.
Get-NaSnaplockLogVolume
Get the active WORM log volume.
Get-NaSnaplockOption
Get the value of a given SnapLock option on a volume. Currently the only supported option is "privdel".
Get-NaSnaplockRetentionTime
Get the SnapLock retention attributes for a file or list of files.
New-NaSnaplockLog
Open a new WORM log file. This will archive the active WORM log file.
Remove-NaSnaplockFile
Execute a privileged delete on a SnapLock file.
Set-NaSnaplockComplianceClock
Set the Compliance Clock of the system equal to the current system clock. This can only be done once.
Set-NaSnaplockLogVolume
Set the active WORM log volume.
Set-NaSnaplockOption
Set a SnapLock option on a volume. Currently, the only supported option is "privdel".
Set-NaSnaplockRetentionTime
Set the SnapLock retention attributes of a file. The retention time is set to the given DateTime.
