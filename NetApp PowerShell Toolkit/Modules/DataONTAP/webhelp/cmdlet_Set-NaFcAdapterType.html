Cmdlet: Set-NaFcAdapterType


NAME
    Set-NaFcAdapterType
    
SYNOPSIS
    Change the adapter driver and/or configuration state.
    
SYNTAX
    Set-NaFcAdapterType [-Adapter] <String> [-Type] <String> [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Change the adapter driver and/or configuration state.  Each configurable 
    adapter has an adapter-type and adapter-state.  The adapter-type indicates 
    which driver is attached to the adapter, the adapter-state indicates what 
    the configuration state of the adapter is. The fc-type is used to modify 
    both the adapter-type and the adapter-state. 
    
    After setting the adapter fc-type a filer reboot is sometimes needed to 
    make the change effective.  Use Get-NaFcAdapter to determine if a filer 
    reboot is needed.
    

PARAMETERS
    -Adapter <String>
        FC adapter name (e.g. 0a)
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Type <String>
        Sets the type and state of the adapter.  Possible inputs
        
        "unconfigured" - set adapter-type to "initiator" and adapter-state to 
        UNCONFIGURED
        "initiator"    - set adapter-type to "initiator" and adapter-state to 
        CONFIGURED
        "target"       - set adapter-type to "target" and adapter-state to 
        CONFIGURED
        "vi"           - set adapter-type to "vi" and adapter-state to 
        CONFIGURED
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).

NOTES
    
    
        Category: fc
        API: fc-config-set-adapter-fc-type

    C:\PS>Set-NaFcAdapterType 0d target
    
    
    Set FC adapter 0d to be an FCP target adapter.
    
    
    
    AdapterName   : 0d
    AdapterState  : CONFIGURED
    AdapterStatus : offline
    AdapterType   : target
    PendingFcType :
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Set-NaFcAdapterType 0d unconfigured
    
    
    Set FC adapter 0d to unconfigured.
    
    
    
    AdapterName   : 0d
    AdapterState  : PENDING
    AdapterStatus : offline
    AdapterType   : target
    PendingFcType : unconfigured
