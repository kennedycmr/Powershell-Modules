Cmdlet: Invoke-NaSnmpGet


NAME
    Invoke-NaSnmpGet
    
SYNOPSIS
    Retrieves the value of a snmp object.
    
SYNTAX
    Invoke-NaSnmpGet [-OID] <String> [-Raw] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Retrieves the value of a snmp object.  This is a convenience method, not a 
    high-performance SNMP query tool.
    

PARAMETERS
    -OID <String>
        Fully qualified object identifier of a snmp object. Only numeric OID's 
        (ex: .1.3.6.1.4.1.789.1.1.1.0) are allowed.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Raw
        Specify Raw to get a string value.  Otherwise, a hashtable is written 
        to the output.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.Object
        
        
    
     
    
NOTES
    
    
        Category: snmp
        API: snmp-get
    
    --------------  Example 1 --------------
    
    C:\PS>Invoke-NaSnmpGet 1.3.6.1.4.1.789.1.1.2.0
    
    
    Get the value of a single OID.
    
    If 'IsValueHexadecimal' is true, the string returned in 'Value' is the 
    hexadecimal representation of the octet string returned by a snmp call to 
    the OID. This output is not present if 'Value' is a normal string.

    IsValueHexadecimal
