Cmdlet: Get-NaRadiusServer


NAME
    Get-NaRadiusServer
    
SYNOPSIS
    Get information about radius servers known to the radius client service.
    
SYNTAX
    Get-NaRadiusServer [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Get information about radius servers known to the radius client service.
    

PARAMETERS
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Radius.RadiusServerListEntryInfo
        
        
    
     
    
NOTES
    
    
        Category: radius
        API: radius-show-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaRadiusServer
    
    
    Get the radius servers known to the radius client service.
    
    
    
    IpAddress          Port       RadiusServerInfo
    ---------          ----       ----------------
    10.10.10.8         1812       Default RADIUS server
    10.10.10.9         1812       Backup RADIUS server 0
