Cmdlet: Confirm-NaLunHasScsiReservation


NAME
    Confirm-NaLunHasScsiReservation
    
SYNOPSIS
    Queries for all types of scsi reservations covering both iSCSI and FCP.
    
SYNTAX
    Confirm-NaLunHasScsiReservation [-Path] <String> [-Controller 
    <NaController>] [<CommonParameters>]
    
    Confirm-NaLunHasScsiReservation [-Path] <String> [-Port] <String> 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Queries for all types of scsi reservations covering both iSCSI and FCP.
    

PARAMETERS
    -Path <String>
        Path of the lun.  The path should start with '/vol/'.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Port <String>
        Name of the initiator port.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.Boolean
        
        
    
     
    
NOTES
    
    
        Category: lun
        API: lun-has-scsi-reservations, lun-port-has-scsi-reservations
    
    --------------  Example 1 --------------
    
    C:\PS>Confirm-NaLunHasScsiReservation /vol/vol2/lun2
    
    
    Check if LUN 'lun2' has any SCSI reservations.
    
    
    
    False
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Confirm-NaLunHasScsiReservation /vol/vol2/lun2 
    iqn.1991-05.com.microsoft:x3550rre8.rtprre.testdomain
    
    
    Check if LUN 'lun2' has any SCSI reservations for a given initiator on 
    server 'x3550rre8'.
    
    
    
    False
