Cmdlet: Set-NaVolCharmap


NAME
    Set-NaVolCharmap
    
SYNOPSIS
    Associate a charmap description with a specified volume.
    
SYNTAX
    Set-NaVolCharmap [-Name] <String> [-Controller <NaController>] [[-Charmap] 
    <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Associate a charmap description with a specified volume.
    

PARAMETERS
    -Name <String>
        Name of the volume on which to list the charmap information.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Charmap <String>
        Description of the character mapping to be done for this volume. This 
        mapping is to allow CIFS clients to use NFS file names that would 
        otherwise result in invalid CIFS names. The values are comma-separated 
        pairs of hex character mappings. The A-F hex values can be in upper or 
        lower case, and the values do not have to be padded. Example: 
        "5c:f2e1,3c:b6,3e:ae,7C:394". If a value is not passed, any existing 
        charmap will be removed.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Volume.VolumeInfo
