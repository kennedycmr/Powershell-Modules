Cmdlet: Get-NaNdmpCopy


NAME
    Get-NaNdmpCopy
    
SYNOPSIS
    Get the status of a running NDMP copy operation.
    
SYNTAX
    Get-NaNdmpCopy [[-Id] <Int32>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the status of a running NDMP copy operation.
    

PARAMETERS
    -Id <Int32>
        The ID of the NDMP copy operation to get.  If not specified, all 
        currently running NDMP copy operations will be returned.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.PowerShell.SDK.Cmdlets.Ndmp.NaNdmpCopyOp
