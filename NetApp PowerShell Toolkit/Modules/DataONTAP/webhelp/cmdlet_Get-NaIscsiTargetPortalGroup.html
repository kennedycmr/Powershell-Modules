Cmdlet: Get-NaIscsiTargetPortalGroup


NAME
    Get-NaIscsiTargetPortalGroup
    
SYNOPSIS
    List information about iSCSI target portal groups.
    
SYNTAX
    Get-NaIscsiTargetPortalGroup [[-Tag] <Int32>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    List information about iSCSI target portal groups.
    

PARAMETERS
    -Tag <Int32>
        Portal group being queried; if not supplied, information on all portal 
        groups is returned
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Iscsi.IscsiTpgroupListEntryInfo
        
        
    
     
    
NOTES
    
    
        Category: iscsi
        API: iscsi-tpgroup-list-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaIscsiTargetPortalGroup
    
    
    List all target portal groups.
    
    
    
    TpgroupName          : e0a_default
    InterfaceListEntries : {e0a}
    TpgroupTag           : 1000
    TpgroupAluaState     : optimized
    TpgroupAluaPreferred : True
    
    TpgroupName          : e0b_default
    InterfaceListEntries : {e0b}
    TpgroupTag           : 1001
    TpgroupAluaState     : optimized
    TpgroupAluaPreferred : True
    
    TpgroupName          : testvif_default
    InterfaceListEntries : {testvif}
    TpgroupTag           : 2000
    TpgroupAluaState     : optimized
    TpgroupAluaPreferred : False
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaIscsiTargetPortalGroup 1000
    
    
    Get iSCSI target portal group with group tag '1000'.

    InterfaceListEntries : {e0a}
    TpgroupTag           : 1000
    TpgroupAluaState     : optimized
    TpgroupAluaPreferred : True
