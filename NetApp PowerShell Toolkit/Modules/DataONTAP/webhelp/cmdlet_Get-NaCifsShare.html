Cmdlet: Get-NaCifsShare


NAME
    Get-NaCifsShare
    
SYNOPSIS
    Gives information about one or more shares.
    
SYNTAX
    Get-NaCifsShare [[-Share] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Gives information about one or more shares.
    

PARAMETERS
    -Share <String>
        Cifs share name. If Share is specified, only information about that 
        share is returned. If Share is not specified, then information about 
        all the shares is returned.  If the name contains the wildcard 
        characters * or ?, then all the shares matching the specified name are 
        listed.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Cifs.CifsShareInfo
        
        
    
     
    
NOTES
    
    
        Category: cifs
        API: cifs-share-list-iter-start, cifs-share-list-iter-next, 
        cifs-share-list-iter-end

    C:\PS>Get-NaCifsShare *Files
