Cmdlet: Get-NaVssSnapshot


NAME
    Get-NaVssSnapshot
    
SYNOPSIS
    Get a list of completed shadow copies in the system that reside in the 
    given context.
    
SYNTAX
    Get-NaVssSnapshot [[-Context] <VSS_SNAPSHOT_CONTEXT>] [<CommonParameters>]
    
    
DESCRIPTION
    Get a list of completed shadow copies in the system that reside in the 
    given context.
    

PARAMETERS
    -Context <VSS_SNAPSHOT_CONTEXT>
        The VSS_SNAPSHOT_CONTEXT to query snapshots.  Only snapshots with this 
        context will be returned.  Default value is BACKUP. Possible values: 
        ALL, BACKUP, FILE_SHARE_BACKUP, NAS_ROLLBACK, APP_ROLLBACK, 
        CLIENT_ACCESSIBLE, CLIENT_ACCESSIBLE_WRITERS.
        
        Required?                    false
        Position?                    1
        Default value                BACKUP
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.HostUtilities.Vss.VssSnapshotProperties
