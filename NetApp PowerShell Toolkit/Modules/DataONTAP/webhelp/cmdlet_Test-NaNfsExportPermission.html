Cmdlet: Test-NaNfsExportPermission


NAME
    Test-NaNfsExportPermission
    
SYNOPSIS
    Returns true if the host IP has mount permissions for a specified path.
    
SYNTAX
    Test-NaNfsExportPermission [-Host <String>] [-Path <String>] [-Permission 
    <String>] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Returns true if the host IP has mount permissions for a specified path.
    

PARAMETERS
    -Host <String>
        IP address of the host to check in appropriate format:
        
        IPv4: AAA.BBB.CCC.DDD
        IPv6: [AAAA:BBBB:...:GGGG]
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Path <String>
        Returns the permissions for this path.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Permission <String>
        Possible values: read-only, read-write, and root.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.Boolean
        
        
    
     
    
NOTES
    
    
        Category: nfs
        API: nfs-exportfs-check-permission
    
    --------------  Example 1 --------------
    
    C:\PS>Test-NaNfsExportPermission -Path /vol/vol2 -Host 10.61.160.6 
    -Permission read-write
    
    
    Check whether host 10.61.160.6 has read/write permission to /vol/vol2.
    
    
    
    True
