Cmdlet: Start-NaReallocate


NAME
    Start-NaReallocate
    
SYNOPSIS
    Starts a new reallocation job.
    
SYNTAX
    Start-NaReallocate [-Path] <String> [-Threshold <Int32>] [-Interval 
    <String>] [-RunOnce] [-NoCheck] [-Full] [-PreserveLogicalBlockNumbers] 
    [-Aggregate] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Starts a new reallocation job.
    

PARAMETERS
    -Path <String>
        Path to large file or LUN to reallocate, starting with /vol/.  If the 
        full option is true then an entire volume may also be specified, as 
        /vol/VOLNAME.  If an aggregate, just the aggregate name must be 
        specified.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Threshold <Int32>
        The allocation threshold before reallocation occurs.  The value must be 
        in the range 3-10, where a larger value indicates the file or LUN can 
        be more fragmented.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Interval <String>
        The interval between repeated scans.  The format is numX, where X is m, 
        h or d, indicating minutes, hours or days.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -RunOnce
        If specified, run the scan once then exit.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -NoCheck
        If specified, do not check if the current layout optimization is above 
        the threshold before reallocating.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Full
        If specified, perform a once-only full reallocate of the indicated 
        file, LUN or volume (specified as /vol/VOLNAME).  This option implies 
        RunOnce and NoCheck are specified.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -PreserveLogicalBlockNumbers
        If specified, reallocate the physical block location in the aggregate, 
        but do not change the logical block number in the flexible volume.  
        This option may only be selected for flexible volumes or files/LUNs 
        within flexible volumes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Aggregate
        If specified, reallocate the aggregate indicated by the path.  This 
        performs block reallocation, not file reallocation.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Reallocate.ReallocateJobInfo
        
        
    
     
    
NOTES
    
    
        Category: reallocate
        API: reallocate-start

    C:\PS>Start-NaReallocate /vol/vol0 -Threshold 3 -Interval 8h
