Cmdlet: Remove-NaQuota


NAME
    Remove-NaQuota
    
SYNOPSIS
    Deletes a quota entry specified by type, target, volume, and tree.
    
SYNTAX
    Remove-NaQuota -UserName <String[]> -Volume <String> [-Qtree <String>] 
    [-Controller <NaController>] [<CommonParameters>]
    
    Remove-NaQuota -GroupId <String> -Volume <String> [-Qtree <String>] 
    [-Controller <NaController>] [<CommonParameters>]
    
    Remove-NaQuota -Path <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    Remove-NaQuota [-Type] <String> [-Target] <String[]> -Volume <String> 
    [-Qtree <String>] [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Deletes a quota entry specified by type, target, volume, and tree.
    

PARAMETERS
    -UserName <String[]>
        A user can be specfied as a: unix user name, numerical user ID, windows 
        account name, Windows SID or a comma separated list of multiple users.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Volume <String>
        Name of the volume for the quota.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Qtree <String>
        This is the qtree name that the quota resides on.  It can be the qtree 
        name or omitted if no qtree.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -GroupId <String>
        May be one of the following: Unix group name, numerical group id (If 
        the group id is 0, no limits set will be enforced, but usage will still 
        be tracked), or the pathname of a file owned by that group (specifying 
        a file or directory will not affect the quota on that file or 
        directory, only the GID).
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Path <String>
        A complete path to an existing qtree.
        
        Required?                    true
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Type <String>
        Quota type: "user", "group", or "tree".
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Target <String[]>
        This is the quota target of the type specified.  The target can be of 
        the form: <name>, <number>, or <path name>.  Multiple targets can be 
        specified by a comma-separated list.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: quota
        API: quota-delete-entry
    
    --------------  Example 1 --------------
    
    C:\PS>Remove-NaQuota tree /vol/testvol/testq testvol testq
    
    
    Remove a quota.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaQuotaEntry | Remove-NaQuota
    
    
    Remove all quotas.
    
    
    
    
    
    --------------  Example 3 --------------
    
    C:\PS>Remove-NaQuota -GroupId Users -Volume testvol -Qtree qtree2
    
    
    Remove a group quota.
    
    
    
    
    
    --------------  Example 4 --------------
    
    C:\PS>Remove-NaQuota -Path /vol/testvol/qtree2
    
    
    Remove the quota for qtree2 on testvol.
    
    
    
    
    
    --------------  Example 5 --------------
    
    C:\PS>Remove-NaQuota -UserName vf1user -Volume testvol -Qtree qtree2
    
    
    Removes the quota for vf1user.
