Cmdlet: Get-NaStorageAdapterInfo


NAME
    Get-NaStorageAdapterInfo
    
SYNOPSIS
    Display the information about a specified host adapter.
    
SYNTAX
    Get-NaStorageAdapterInfo [-Name] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Display the information about a specified host adapter.  The information is 
    displayed base on the controller interface type: ATA, Parallel SCSI, SAS, 
    FC.
    

PARAMETERS
    -Name <String>
        The adapter name is either a slot number, or, if a port letter is also 
        presented, a slot number and port letter concatenated into a single 
        name, for example '8a' or '11b'.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.StorageAdapter.AdapterDetails
