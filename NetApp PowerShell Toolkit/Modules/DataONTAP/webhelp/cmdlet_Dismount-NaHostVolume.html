Cmdlet: Dismount-NaHostVolume


NAME
    Dismount-NaHostVolume
    
SYNOPSIS
    Remove one or all mount points from a Windows volume.
    
SYNTAX
    Dismount-NaHostVolume [-MountPoint] <String> [-Force] [<CommonParameters>]
    
    Dismount-NaHostVolume [-VdsId] <Guid> [-Force] [<CommonParameters>]
    
    
DESCRIPTION
    Remove one or all mount points from a Windows volume.
    
    This cmdlet must be run with administrative privileges.
    

PARAMETERS
    -MountPoint <String>
        An active drive letter (i.e. 'G') or folder mount point to remove from 
        the volume
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Force
        Continue even if volume is in use. This might cause unexpected errors 
        in any application using this volume.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -VdsId <Guid>
        The Guid object which uniquely identifies the volume in VDS.  Note that 
        this is not the same as the volume guid path.  Specify this parameter 
        directly or via the pipeline to remove all mount points from a single 
        volume.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.PowerShell.SDK.Cmdlets.Toolkit.Windows.HostVolume
        
        
    
     
    
NOTES
    
    
        Category: host
    
    --------------  Example 1 --------------
    
    C:\PS>Dismount-NaHostVolume J:\
    
    
    Remove the drive letter J:\ from the volume.
    
    
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaHostVolume -DiskIndex 4 | Dismount-NaHostVolume
    
    
    Pass the volume object on the pipeline to remove all mount points (drive 
    letter & folders) from a volume.
