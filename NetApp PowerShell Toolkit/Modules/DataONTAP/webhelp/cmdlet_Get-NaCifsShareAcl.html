Cmdlet: Get-NaCifsShareAcl


NAME
    Get-NaCifsShareAcl
    
SYNOPSIS
    Gives information about one or more CIFS shares' access control list (ACL).
    
SYNTAX
    Get-NaCifsShareAcl [[-Share] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Gives information about one or more CIFS shares' access control list (ACL).
    

PARAMETERS
    -Share <String>
        Cifs share name.  If specified, only information about that share is 
        returned.  If not specified, then information about all shares is 
        returned.  If the name contains the wildcard characters * or ?, then 
        all the shares matching the specified pattern are listed.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Cifs.CifsShareAclInfo
        
        
    
     
    
NOTES
    
    
        Category: cifs
        API: cifs-share-acl-list-iter-start, cifs-share-acl-list-iter-next, 
        cifs-share-acl-list-iter-end
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaCifsShareAcl
    
    
    List ACL info for all CIFS shares.
    
    
    
    ShareName      UserAclInfo
    ---------      -----------
    ETC$           {BUILTIN\Administrators}
    HOME           {everyone, root}
    C$             {BUILTIN\Administrators}
    test           {everyone, DUNN\krusty}
    clinton_backup {DUNN\cknight}
    alfredo_backup {everyone}
