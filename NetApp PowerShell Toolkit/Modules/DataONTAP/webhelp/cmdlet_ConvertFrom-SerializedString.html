Cmdlet: ConvertFrom-SerializedString


NAME
    ConvertFrom-SerializedString
    
SYNOPSIS
    Convert a serialized .NET object back into a native one.
    
SYNTAX
    ConvertFrom-SerializedString [-Data] <String> [<CommonParameters>]
    
    
DESCRIPTION
    Convert a serialized .NET object back into a native one.
    
    This cmdlet is the companion of ConvertTo-SerializedString.
    

PARAMETERS
    -Data <String>
        The string that was generated by ConvertTo-SerializedString.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    System.Object
        
        
    
     
    
NOTES
    
    
        Category: toolkit
    
    --------------  Example 1 --------------
    
    C:\PS>$str = Connect-NaController dunn | ConvertTo-SerializedString
    Get-NaSystemVersion -Controller (ConvertFrom-SerializedString $str)
    
    
    Reconstitute a controller instance from a serialized string and use it 
    subsequent Toolkit cmdlet invocations.
