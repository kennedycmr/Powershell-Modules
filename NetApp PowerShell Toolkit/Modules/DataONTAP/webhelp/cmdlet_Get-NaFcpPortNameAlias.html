Cmdlet: Get-NaFcpPortNameAlias


NAME
    Get-NaFcpPortNameAlias
    
SYNOPSIS
    Get the list of WWPN for a given alias or an alias for for a given WWPN or 
    complete list of all current alias-WWPN mappings.
    
SYNTAX
    Get-NaFcpPortNameAlias [-PortName <String>] [-Alias <String>] [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the list of WWPN for a given alias or an alias for for a given WWPN or 
    complete list of all current alias-WWPN mappings.
    

PARAMETERS
    -PortName <String>
        WWPN for which all aliases will be returned. When supplied all aliases 
        for that WWPN will be returned. Otherwise, all aliases with their WWPNs 
        present in the system will be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Alias <String>
        Alias for a WWPN is the 32-character alias that may contain A-Z, a-z, 
        0-9, _,-,.,{,} and no spaces. When supplied, the alias with the 
        corresponding WWPN will be returned. Otherwise, all aliases with their 
        WWPNs present in the system will be returned.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Fcp.AliasesInfo
        
        
    
     
    
NOTES
    
    
        Category: fcp
        API: fcp-wwpnalias-get-alias-info
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaFcpPortNameAlias -Alias port1
    
    
    Get the WWPN having the alias 'port1'.
    
    
    
    AliasesAlias                            AliasesWwpn
    ------------                            -----------
    port1                                   50:0a:09:81:97:09:3b:1f
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Get-NaFcpPortNameAlias -PortName 50:0a:09:81:97:09:3b:1f
    
    
    Get the alias for the specified WWPN.
    
    
    
    AliasesAlias                            AliasesWwpn
    ------------                            -----------
    port1                                   50:0a:09:81:97:09:3b:1f
