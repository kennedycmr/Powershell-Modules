Cmdlet: Remove-NaConsistencyGroup


NAME
    Remove-NaConsistencyGroup
    
SYNOPSIS
    Deletes the snaps associated with a consistency group checkpoint in this 
    controller.
    
SYNTAX
    Remove-NaConsistencyGroup [-Snapshot] <String> [-Volumes] <String[]> 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Deletes the snaps associated with a consistency group checkpoint in this 
    controller.
    

PARAMETERS
    -Snapshot <String>
        The name of the snapshot that is deleted in each volume.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Volumes <String[]>
        A list of volumes in this controller that is part of this consistency 
        group operation.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    
        
        
    
     
    
NOTES
    
    
        Category: consistency group
        API: cg-delete
    
    --------------  Example 1 --------------
    
    C:\PS>Remove-NaConsistencyGroup cg_snapshot vol1,vol2
    
    
    Remove the consistency group snapshots named 'cg_snapshot' from volumes 
    'vol1' and 'vol2'.
