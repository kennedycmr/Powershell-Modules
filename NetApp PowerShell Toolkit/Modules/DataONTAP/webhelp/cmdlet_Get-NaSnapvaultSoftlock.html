Cmdlet: Get-NaSnapvaultSoftlock


NAME
    Get-NaSnapvaultSoftlock
    
SYNOPSIS
    List all snapvault softlocks on the given snapshot which are locked by 
    external means.
    
SYNTAX
    Get-NaSnapvaultSoftlock [-Volume] <String> [-Snapshot] <String> 
    [-Controller <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    List all snapvault softlocks on the given snapshot which are locked by 
    external means.
    

PARAMETERS
    -Volume <String>
        Name of the volume where the snapshot exists.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Snapshot <String>
        Name of the snapshot for which softlocks are to be listed.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snapvault.SnapvaultSoftlockInfo
        
        
    
     
    
NOTES
    
    
        Category: snapvault
        API: snapvault-get-softlocks
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaSnapvaultSoftlock vol0 vol0_sv_sched.1
    
    
    Get the softlocks for snapshot 'vol0_sv_sched.1' on volume 'vol0'.
    
    
    
    SoftlockName                            Type
    ------------                            ----
    lock1                                   api
