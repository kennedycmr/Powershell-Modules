Cmdlet: Remove-NaSnmpCommunity


NAME
    Remove-NaSnmpCommunity
    
SYNOPSIS
    Deletes a community from the list of communities.
    
SYNTAX
    Remove-NaSnmpCommunity [-Community] <String> [-Controller <NaController>] 
    [<CommonParameters>]
    
    Remove-NaSnmpCommunity -All [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Deletes a community from the list of communities.
    

PARAMETERS
    -Community <String>
        Community name to be deleted.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -All
        Specify -All to delete all communities.
        
        Required?                    true
        Position?                    named
        Default value                false
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snmp.SnmpStatusInfo
        
        
    
     
    
NOTES
    
    
        Category: snmp
        API: snmp-community-delete, snmp-community-delete-all
