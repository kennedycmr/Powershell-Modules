Cmdlet category: Snapshot

Get-NaSnapshot
Return snapshot information for a specified volume or aggregate.
Get-NaSnapshotAutodelete
Returns the current snapshot autodelete settings.
Get-NaSnapshotDelta
Returns the amount of space consumed between two snapshots or a snapshot and active filesystem.
Get-NaSnapshotReclaimable
Returns the amount of space that would be freed when a set of snapshots are deleted from a specified volume.
Get-NaSnapshotReserve
Gets the percentage of disk space that is reserved for snapshots in the indicated volume or aggregate.
Get-NaSnapshotRestoreStatus
Get information about snapshot file restores on a given controller. Returns maximum snapshot file restores limit and snapshot file restores in progress numbers.
Get-NaSnapshotSchedule
Obtain the current snapshot schedule on a specified volume.
Get-NaSnapshotVolumeSpace
Returns snapshot related volume information.
New-NaSnapshot
Create a new snapshot on a specified volume or aggregate.
New-NaSnapshotMulti
Create a snapshot with a specific name on each of the specified volumes.
Remove-NaSnapshot
Delete a snapshot on a specified volume or aggregate.
Rename-NaSnapshot
Rename a specified snapshot to a new name on a specified volume or aggregate.
Restore-NaSnapshotFile
Reverts a single file to a revision from a specified snapshot.
Restore-NaSnapshotVolume
Reverts a volume to a specified snapshot.
Set-NaSnapshotAutodelete
Set the option named to the value specified by in the autodelete settings of the specified volume.
Set-NaSnapshotReserve
Sets the size of the snapshot reserve to the specified percentage on the indicated volume or aggregate.
Set-NaSnapshotSchedule
Set the snapshot schedule on a specified volume.
Test-NaSnapshotMulti
Validates the multi-snapshot creation operation on the specified volumes.
