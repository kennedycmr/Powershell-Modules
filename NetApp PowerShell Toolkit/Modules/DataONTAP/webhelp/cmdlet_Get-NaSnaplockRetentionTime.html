Cmdlet: Get-NaSnaplockRetentionTime


NAME
    Get-NaSnaplockRetentionTime
    
SYNOPSIS
    Get the SnapLock retention attributes for a file or list of files.
    
SYNTAX
    Get-NaSnaplockRetentionTime [-Pathnames] <String[]> [-Controller 
    <NaController>] [<CommonParameters>]
    
    
DESCRIPTION
    Get the SnapLock retention attributes for a file or list of files.
    

PARAMETERS
    -Pathnames <String[]>
        List of List of pathnames to query.  Pathname must be in the form 
        /vol/<vol-name>/<path>.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Snaplock.FileRetentionInfo
        
        
    
     
    
NOTES
    
    
        API: file-snaplock-retention-time-list-info
        Category: snaplock
    
    --------------  Example 1 --------------
    
    C:\PS>Get-NaSnaplockRetentionTime /vol/snaplock_vol0/file0.txt
    
    
    Get the retention time attributes for the file /vol/snaplock_vol0/file0.txt
    
    
    
    Pathname                                               SnaplockType     
    RetentionTimeDT
    --------                                               ------------     
    ---------------
    /vol/snaplock_vol0/file0.txt                           SLC              
    6/11/2011 4:14:32 AM
    
    
    
    --------------  Example 2 --------------
    
    C:\PS>Read-NaDirectory /vol/snaplock_vol0 | %{ if($_.Name -ne "." -and
