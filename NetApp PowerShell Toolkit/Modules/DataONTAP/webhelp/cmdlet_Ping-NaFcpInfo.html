Cmdlet: Ping-NaFcpInfo


NAME
    Ping-NaFcpInfo
    
SYNOPSIS
    Send an ELS ECHO frame to a Fibre Channel address or WWPN. The cmdlet pings 
    the address or WWPN count times and returns the number of successful pings 
    and times.
    
SYNTAX
    Ping-NaFcpInfo [-FcpAdapter] <String> [-FcpPortId] <UInt32> [[-PingCount] 
    <Int32>] [[-Payload] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    Ping-NaFcpInfo [-FcpAdapter] <String> [-FcpWwpn] <String> [[-PingCount] 
    <Int32>] [[-Payload] <String>] [-Controller <NaController>] 
    [<CommonParameters>]
    
    
DESCRIPTION
    Send an ELS ECHO frame to a Fibre Channel address or WWPN. The cmdlet pings 
    the address or WWPN count times and returns the number of successful pings 
    and times.
    

PARAMETERS
    -FcpAdapter <String>
        Adapter to send the ping request from.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue, ByPropertyName)
        Accept wildcard characters?  false
        
    -FcpPortId <UInt32>
        The Fibre Channel address to ping.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -PingCount <Int32>
        The number of pings.  Default is 3. Max is 16.
        
        Required?                    false
        Position?                    3
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Payload <String>
        Additional data to send in the payload of the ELS ECHO frame. The 
        payload can be up to 255 characters long.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Controller <NaController>
        The Data ONTAP controller object, embodied by an NaController object.  
        This parameter is returned by the Connect-NaController cmdlet.  If not 
        specified, the value in the global variable CurrentNaController is used.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -FcpWwpn <String>
        The Fibre Channel WWPN to ping. The format of a WWPN is 
        XX:XX:XX:XX:XX:XX:XX:XX where X is a hexadecimal digit.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer and OutVariable. For more information, see 
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
        
        
    
     
    
OUTPUTS
    DataONTAP.Types.Fcp.PingFcpInfo
